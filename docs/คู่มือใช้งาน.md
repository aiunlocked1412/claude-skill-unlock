> เอกสารนี้ช่วยให้คุณเริ่มต้นใช้ skills ทั้ง **150 ตัว** ได้ภายใน 5 นาที

## 1. ก่อนเริ่ม: สิ่งที่ต้องมี

| สิ่งที่ต้องมี | วิธีตรวจสอบ |
|-------------|-------------|
| macOS หรือ Linux | `uname -a` |
| Claude Code ติดตั้งแล้ว | `claude --version` |
| Bash 4+ | `bash --version` |

ยังไม่มี Claude Code? → ติดตั้งที่ https://claude.com/claude-code

---

## 2. ติดตั้ง Skills

### แบบปกติ (ข้าม skill ที่มีชื่อซ้ำ)
```bash
cd /path/to/claude-skill-unlock
./install.sh
```

### แบบเขียนทับ (ถ้าเคยติดตั้งแล้วอยากอัปเดต)
```bash
./install.sh --force
```

Skills จะถูกติดตั้งไปที่ `~/.claude/skills/<skill-name>/`

---

## 3. เริ่มใช้งาน Skills

### 3.1 เปิด Claude Code
```bash
claude
```

### 3.2 เรียก skill ด้วย slash command

**รูปแบบ 1:** เรียกโดยไม่มี argument → skill จะแสดงเมนู
```
/youtuber-pro
```

**รูปแบบ 2:** เรียกพร้อม argument → skill ทำงานทันที
```
/youtuber-pro สคริปต์ 10 นาที เรื่องลงทุนหุ้น
/copywriter-pro headline สำหรับโฆษณาครีมกันแดด
/teacher-pro แผนการสอนคณิตศาสตร์ ม.2
/personal-finance วางแผนเก็บล้านแรกใน 5 ปี
/notion-builder workspace สำหรับทีม 5 คน
```

### 3.3 ตัวอย่างใช้งานจริง

#### 🎬 สร้างสคริปต์ YouTube
```
/youtuber-pro
คุณ: รีวิวกล้องฟิล์มสำหรับมือใหม่ ยาว 8 นาที
Claude: [สร้างสคริปต์ + hook + thumbnail prompt + title]
```

#### 💻 Review โค้ด
```
/programmer-pro review @src/auth.ts
```

#### 💰 วางแผนการเงินส่วนตัว
```
/personal-finance อายุ 28 รายได้ 50K/เดือน อยากเก็บเงินซื้อบ้าน 5 ปี
```

#### ⚙️ จัด Notion Workspace
```
/notion-builder workspace สำหรับฟรีแลนซ์ designer ที่มี 10 ลูกค้า
```

## 4. Tips & Tricks

### 💡 Tip 1: ใช้ @ เพื่อแนบไฟล์
```
/programmer-pro review @src/components/Button.tsx
/blogger-pro ปรับปรุงบทความนี้ @draft.md
/data-analyst-pro วิเคราะห์ @sales.csv
```

### 💡 Tip 2: Skill หลายตัวในงานเดียว
```
/digital-marketer วางแผน campaign โปรโมทคอร์สออนไลน์
↓
/copywriter-pro headline สำหรับ campaign
↓
/social-media-admin ปฏิทินโพส 30 วัน
↓
/email-marketer email sequence 5 emails
```

### 💡 Tip 3: บันทึก output ไว้ใช้ต่อ
Skills ส่วนใหญ่จะถาม "บันทึกเป็นไฟล์หรือไม่" → ตอบ "ใช่" เพื่อเก็บไว้อ้างอิง

### 💡 Tip 4: ปรับ prompt ให้เฉพาะเจาะจง
- ❌ `/copywriter-pro เขียน headline`
- ✅ `/copywriter-pro headline ครีมกันแดด SPF50+ กลุ่มเป้าหมายสาวออฟฟิศ 25-35 ปี`

## 6. แก้ปัญหา

| ปัญหา | วิธีแก้ |
|-------|---------|
| `/<skill>` ไม่ขึ้น | restart Claude Code, ตรวจว่า `~/.claude/skills/<skill>/SKILL.md` มีอยู่ |
| ติดตั้งแล้วแต่ไม่มีผล | รัน `./install.sh --force` |
| `Permission denied: ./install.sh` | `chmod +x install.sh uninstall.sh` |
| อยากแก้ skill | แก้ที่ `~/.claude/skills/<skill>/SKILL.md` โดยตรง |

ปัญหาอื่นๆ → ดู [`FAQ.md`](./FAQ.md)
