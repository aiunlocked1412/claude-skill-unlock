# หมวด 9: Productivity & Modern Work (7 skills)

Skills สำหรับการทำงานยุคใหม่ — meeting, email, Notion, project, async, EA

| Skill | คำสั่ง | ใช้เมื่อไหร่ |
|-------|--------|-------------|
| Meeting Note-taker | `/meeting-notetaker` | สรุป meeting + action items |
| Email Manager | `/email-manager` | inbox zero, template, follow-up |
| Notion Builder | `/notion-builder` | workspace, database, automation |
| Project Manager | `/project-manager-pro` | Gantt, sprint, retrospective |
| Async Communicator | `/async-communicator` | RFC, doc-first, async culture |
| Executive Assistant | `/executive-assistant` | EA tasks, scheduling, support |
| Presentation Designer | `/presentation-designer` | slide design, pitch deck |

## รูปแบบ output ทั่วไป

- Structured note (markdown)
- Template (email, RFC, retrospective)
- Workspace schema (Notion/Airtable)
- Slide outline + visual notes

## Workflow ตัวอย่าง

```
1. /meeting-notetaker สรุป kickoff meeting
   ↓
2. /project-manager-pro ตั้ง project + sprint plan
   ↓
3. /notion-builder workspace สำหรับ team
   ↓
4. /async-communicator RFC สำหรับ key decision
```
