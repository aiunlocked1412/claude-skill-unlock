# Email Manager

> AI ผู้จัดการ inbox — triage, reply, follow-up, scheduling, template bank, signature

## ⚡ ใช้ยังไง

```
/email-manager
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Inbox Zero ด้วย 4D method (Delete / Delegate / Defer / Do)
- ✅ ร่าง reply ภาษาไทย/อังกฤษ ปรับระดับความเป็นทางการ
- ✅ Follow-up sequence 3-step (ถ้าไม่ตอบ)
- ✅ Scheduling email + propose 3 slots
- ✅ Template bank (sales, decline, intro, cold outreach, thank you)
- ✅ Email signature design
- ✅ Best time to send recommendation

## 💡 Use cases

- ผู้บริหารที่ inbox ทะลัก 200+ emails
- Sales ที่ต้องส่ง outreach + follow-up หลายเคส
- Founder ที่ต้องตอบทั้งลูกค้า + investor + partner
- HR ที่ต้องตอบคำถามซ้ำ ๆ จากผู้สมัคร
- Knowledge worker ที่อยากปิด inbox ทุกเย็น

## 🎓 Tips

- ใช้ **Schedule send** เลื่อนเวลาส่ง → land ที่ best time
- **Snooze** email ที่ defer ออกไป → inbox สะอาด
- ใช้คู่กับ `/meeting-notetaker` → ส่ง follow-up หลัง meeting
- ใช้คู่กับ `/executive-assistant` → จัด calendar + email พร้อมกัน
