# Prompt Formula — Email Manager

## Template เก็บ context (Reply)

```
จาก: <ชื่อ + ตำแหน่ง + บริษัท>
ความสัมพันธ์: <ลูกค้า / vendor / partner / boss / peer / cold>
ระดับทางการ: <formal / semi / casual>
หัวข้อ: <subject>
เนื้อหา original:
<วาง email มาเลย>

ผู้ใช้อยาก reply: <ตอบรับ / ปฏิเสธ / ขอเวลา / negotiate / ถามต่อ>
ข้อมูลเพิ่มเติม: <context ที่ต้องใส่ใน reply>
```

## 4D Triage Recipe

```
สำหรับทุก email ใน inbox — จัดเข้า 1 ตะกร้า:

🗑️ DELETE — newsletter ไม่อ่าน, notification, spam
👥 DELEGATE — ส่งให้คนเหมาะกว่า + 1 บรรทัด context
📅 DEFER — ต้องคิดเกิน 2 นาที → snooze + เวลาที่จะกลับมา
⚡ DO — ตอบได้ใน 2 นาที → ทำเลย

Goal: inbox = 0 ภายใน session
```

## Reply Recipe (4 ส่วน)

```
1. Greeting (1 บรรทัด) — ปรับระดับ
2. Acknowledge (1 บรรทัด) — รับรู้สิ่งที่เขาส่ง
3. Body (2-5 บรรทัด) — ตอบเรื่องหลัก
4. CTA + Closing — ขออะไร เมื่อไหร่
```

## Template Bank

### Sales Outreach (Cold)
```
Subject: <ปัญหา>ที่<บริษัทเขา>อาจกำลังเจอ?

สวัสดีครับ คุณ X

ผมเห็นว่า <บริษัท> เพิ่ง <event ล่าสุด> — ขอแสดงความยินดีครับ

ผมคุยกับ <บริษัทคล้ายกัน> และพบว่าเขามักเจอ <ปัญหา A>
เราที่ <บริษัทเรา> ช่วย <บริษัทอื่น> แก้ปัญหานี้ — ผลคือ <metric>

อยากนัดคุย 15 นาที ดูว่าเหมาะกับ <บริษัท> ไหม?
สะดวก [3 slots] ไหมครับ

ขอบคุณครับ
[ชื่อ]
```

### Decline ขอ Discount
```
สวัสดีครับ คุณ X

ขอบคุณที่สนใจ <product> ครับ

ผมเข้าใจว่า budget สำคัญ — แต่ราคาที่เสนอเป็นราคาสุทธิแล้ว
เพราะ <เหตุผล: คุณภาพ / scope / partnership tier>

ทางเลือกที่ผมเสนอได้:
- Option A: ลด scope ให้ตรง budget
- Option B: ผ่อนชำระ 3 งวด
- Option C: เริ่มจาก pilot scope เล็กก่อน

อยากคุยเพิ่มเติมไหมครับ?
ขอบคุณครับ
```

### Intro 2 คน
```
Subject: Intro — <ชื่อ A> meet <ชื่อ B>

สวัสดีคุณ A และ B ครับ

ขออนุญาต intro 2 ท่านครับ:

🔹 <ชื่อ A> — <ตำแหน่ง> ที่ <บริษัท>
   ผู้เชี่ยวชาญ <field> + กำลัง <project>

🔹 <ชื่อ B> — <ตำแหน่ง> ที่ <บริษัท>
   เคย <achievement> + อยาก <goal>

ผมว่า 2 ท่านน่าจะคุยกันได้ดีเรื่อง <topic>

ขออนุญาต hand off — สนุกกับการสนทนาครับ!
[ชื่อผู้แนะนำ]
```

### Thank You หลัง Meeting
```
สวัสดีครับ คุณ X

ขอบคุณมากที่สละเวลาคุยวันนี้ — ได้ insight ดีมาก

สรุปสิ่งที่เราคุย:
- ...
- ...

Action items:
- ผม: <action> — ส่งภายใน <วัน>
- คุณ: <action> — ภายใน <วัน>

หวังว่าจะได้ทำงานด้วยกันเร็ว ๆ นี้ครับ
[ชื่อ]
```

### Polite Decline (ปฏิเสธ pitch)
```
สวัสดีครับ คุณ X

ขอบคุณที่นึกถึง <บริษัท> สำหรับ <opportunity>

ตอนนี้เรา <focus priority อื่น> — ยังไม่ใช่จังหวะที่เหมาะ
ขอเก็บ contact ไว้ — ถ้าจังหวะเหมาะเมื่อไหร่จะรีบติดต่อกลับ

ขอให้ <project> ของคุณประสบความสำเร็จครับ
[ชื่อ]
```

## Tips

- **Subject ก่อน body** — เขียน subject สุดท้าย เพราะรู้ผลลัพธ์แล้ว
- **5-second test** — เปิดมาภายใน 5 วิ ต้องรู้ "ต้องทำอะไร"
- **Cut 30%** — เขียนเสร็จ ตัดทิ้ง 30% ของคำ
- **One ask per email** — ขอ 2 อย่าง = ได้ 0
- **Read aloud** — อ่านออกเสียงก่อน send → tone จะถูก
