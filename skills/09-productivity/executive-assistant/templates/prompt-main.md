# Prompt Formula — Executive Assistant

## Template เก็บ context (Executive Profile)

```
Executive: <ชื่อ + ตำแหน่ง + บริษัท>
Working hours: <09:00-18:00 + flexibility>
Time zone: <BKK / UTC+7>
Energy peak: <morning / afternoon>
Sacred time: <lunch 12-13, family 18-20, weekend>
Allergies: <food / hotel preference>
Loyalty programs: <airline / hotel>
Languages: <TH, EN, JP>
Spouse + kids info: <ระบุชื่อ — โดน mention ใน email/event>
Dietary: <vegetarian / halal / กินเผ็ด>
```

## Calendar Planning Recipe

```
Color code:
🔴 External (board, investor, key client)
🟢 Internal (1:1, team)
🔵 Focus (deep work, no interrupt)
🟡 Personal (lunch, workout, family)
🟣 Travel
⚪ Buffer

Constraints:
- Focus 🔵 ≥ 10 ชม./สัปดาห์
- 1:1 ทุก direct report ≥ 30 นาที/สัปดาห์
- Lunch + workout = sacred
- Buffer 15 นาทีรอบ external meeting
- No meeting > 30 นาที ก่อน 9:00 / หลัง 17:00
```

## Travel Itinerary Recipe (7 sections)

```
1. Trip Overview (destination, purpose, dates, budget)
2. Flight Details (airline, #, time, seat, lounge, jet lag plan)
3. Accommodation (hotel, address, check-in/out, loyalty, preferences)
4. Ground Transport (driver, daily plan, Uber backup)
5. Daily Schedule (timeline + buffer + meal)
6. Briefing Pack (1-page brief ทุก meeting)
7. Logistics & Emergency (SIM, currency, visa, embassy, insurance)
```

## Pre-meeting Briefing Recipe

```
1. Meeting basics (date, time, location, format)
2. Attendees (background, focus, last interaction)
3. Objective (1 sentence — outcome wanted)
4. Background (company, news, previous meetings)
5. Talking points (3-5 max)
6. Ask (concrete — what executive wants)
7. Avoid / sensitive topics
8. Gift (if applicable)
9. Post-meeting actions
```

## Email Triage Tiers

```
Tier 1: ส่งให้ executive
- Board / investor
- C-level peer
- Top 10 customers
- Family
- Whitelist personal

Tier 2: EA reply + CC executive
- Scheduling request
- Vendor inquiry
- Speaking invitation
- Press inquiry

Tier 3: EA reply (no CC)
- Newsletter / promotion (unsubscribe)
- Auto-reply
- Spam / cold
```

## Polite Decline Templates

### Meeting decline (no time)
```
ขอบคุณที่อยากคุยกับคุณ X
ตารางเต็มจนถึง <date>

ขออนุญาตเสนอ:
- Option A: นัดเดือนหน้า
- Option B: คุยกับท่านอื่นในองค์กร
- Option C: ส่ง question เป็น email
```

### Speaking decline
```
ขอบคุณที่นึกถึงคุณ X สำหรับ <event>
มี commitment อื่นวันนั้น

ขออนุญาตเสนอ:
- ปีหน้า: ติดต่อล่วงหน้า 3 เดือน
- ผู้บริหารอื่นที่อาจสนใจ: คุณ Y
```

### Request decline (favor)
```
ขอบคุณที่นึกถึง

น่าเสียดายช่วงนี้ commitment เต็ม
หาก urgent ผมแนะนำ: <alternative resource>

หากในอนาคตมีโอกาสจะรีบติดต่อกลับ
```

## Weekly Summary Format

```
🎯 WINS (3-5)
📊 KEY METRICS (vs target)
⚠️ ATTENTION NEEDED
📅 NEXT WEEK PRIORITIES (by day)
✈️ UPCOMING TRAVEL
📋 DECISIONS WAITING (need by date)
💡 RECOMMENDATIONS
```

## Tips

- **Always confirm × 2** — booking, time zone, address
- **Anticipate need** — passport scan ready 6 mo before expire
- **Backup plan** — alternative flight/hotel ready
- **Quiet hours** — no work email Sat 18:00 - Sun 18:00 (default)
- **EA notebook** — เก็บ preference ทุกอย่าง (Boss likes window seat, no peanut)
- **Cross-time-zone scheduling** — show 2 time zones always
- **Currency rule** — exchange before trip + cash backup 5,000 บาท equivalent
