# Agile Coach

> AI Scrum Master และ Agile Coach — Sprint Ceremony, Velocity Tracking, Retrospective, Team Coaching

## ⚡ ใช้ยังไง

```
/agile-coach
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Sprint Planning (goal template, capacity calculation, backlog selection)
- ✅ Daily Standup Script + Anti-Pattern Guide
- ✅ Velocity & Metrics Dashboard (burndown, cycle time, happiness)
- ✅ Sprint Review Template
- ✅ Retrospective Workshop (5 formats: Start/Stop/Continue, 4Ls, Sailboat, etc.)
- ✅ Team Health Check + Coaching Plan (dysfunction diagnosis + fix)

## 💡 Use cases

- Scrum Master ต้องการ retro format ใหม่ที่ได้ผลจริง
- Product Manager ต้องการ sprint planning template ครบถ้วน
- Team Lead วางแผน agile transition จาก waterfall
- Developer ต้องการ velocity tracking dashboard
- Manager ต้องการเข้าใจ agile metrics ก่อนรายงาน stakeholder

## 📊 ตัวอย่างผลลัพธ์

Sprint Package ครบถ้วนพร้อม sprint goal template, ceremony scripts, velocity tracker, retro board structure และ action item log

## 🎓 Tips

- บอก **pain point ที่ชัดเจน** (scope creep, retro ไม่ได้ผล) → แก้ตรงจุด
- ระบุ **framework ที่ใช้อยู่** (Scrum/Kanban) → advice ตรงกับ context
- ใช้คู่กับ `/time-manager` → personal time management + team sprint
- ขอ **anti-pattern checklist** สำหรับ ceremony ที่ทำอยู่
