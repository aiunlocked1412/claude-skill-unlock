# Knowledge Manager

> AI Personal Knowledge Manager — Zettelkasten, Obsidian, PARA, Note-Taking System, Second Brain

## ⚡ ใช้ยังไง

```
/knowledge-manager
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Vault / Workspace Structure Design (Obsidian, Notion)
- ✅ Zettelkasten Setup (fleeting, literature, permanent notes + linking)
- ✅ PARA Method Setup (Projects, Areas, Resources, Archive)
- ✅ Note-Taking System (capture workflow + templates)
- ✅ Review System (daily/weekly process + spaced repetition)
- ✅ Knowledge to Output (notes → MOC → draft → publish pipeline)

## 💡 Use cases

- นักศึกษาต้องการ PKM สำหรับ thesis และ research
- Freelancer จัดการ notes จากหลาย project พร้อมกัน
- นักเขียน/blogger ต้องการ system เปลี่ยน notes เป็น content
- Knowledge worker อยากจำสิ่งที่อ่านได้นานขึ้น
- ผู้เรียนรู้ตลอดชีวิตอยากสร้าง second brain ที่ใช้งานจริง

## 📊 ตัวอย่างผลลัพธ์

## 🎓 Tips

- บอก **goal ของ PKM** (เขียน blog, งานวิจัย, จำดีขึ้น) → structure ตรงจุด
- เริ่มจาก **PARA ก่อน Zettelkasten** ถ้าเพิ่งเริ่ม → ง่ายกว่า
- ใช้คู่กับ `/time-manager` → GTD reference system + PKM
- ขอ **note templates** เพื่อเริ่มได้ทันที ไม่ต้อง design เอง
