# 📊 Project Plan — เปิดร้าน Omakase "Aji" สาขาแรก ทองหล่อ

## 📋 Project Charter

| Field | Value |
|-------|-------|
| Project Name | Open "Aji" Omakase Restaurant — Thonglor Branch 1 |
| Sponsor | คุณวิน (Owner / Investor หลัก) |
| Project Manager | คุณเจน (Operations Director) |
| Team | 8 คน (PM, Chef, Sous Chef, Designer, Marketing, Legal, Construction PM, F&B Consultant) |
| Budget | THB 5,200,000 |
| Start Date | 22/04/26 |
| End Date | 18/10/26 (Soft launch 11/10, Grand opening 18/10) |
| Methodology | Hybrid — Waterfall (construction + legal), Agile (marketing + menu) |

### 🎯 Goal
ภายใน 26 สัปดาห์ เปิดร้าน Omakase ระดับ premium (ราคา 3,500-5,500 บาท/หัว) ที่ทองหล่อ — รับลูกค้า 8-12 ที่นั่ง/รอบ × 2 รอบ/วัน × 6 วัน/สัปดาห์ — ภายใน Q1 หลังเปิด ทำ booking rate ≥ 75%

### ✅ Success Criteria
1. เปิดทันวันที่ 18/10/26 (grand opening)
2. ค่าก่อสร้าง + setup ≤ 3.5 ลบ. (เหลือ working capital 1.7 ลบ.)
3. License ครบ (ทะเบียน, F&B, ขายแอลกอฮอล์, แรงงาน)
4. Soft launch 1 สัปดาห์ — feedback ลูกค้า 50 คน, NPS ≥ 60
5. Pre-booking grand opening เต็ม 90% (จาก 96 ที่นั่ง × 7 วัน = 672 booking)

### 🚫 Out of Scope
- สาขา 2 (จะวางแผนหลังเปิด 6 เดือน)
- Delivery / takeaway (omakase ไม่เหมาะ)
- Catering / event service (phase 2)
- ระบบ membership (จะเพิ่มหลังเปิด 3 เดือน)

### 🔒 Constraints
- Chef หลัก (เชฟทาเคชิ) start ได้แค่ 1 มิ.ย. (มี contract ที่ Tokyo จบ)
- Lease สัญญาเริ่ม 1 พ.ค. (ค่าเช่าเริ่มจ่าย)
- Investor ขอ ROI projection ภายใน 2 ปี
- Building inspection ต้องผ่าน BMA (กทม.) ก่อนเปิด
