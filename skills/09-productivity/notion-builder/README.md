# Notion Builder

> AI สถาปนิก Notion workspace — PARA method, database schema, relation, automation, template

## ⚡ ใช้ยังไง

```
/notion-builder
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Personal workspace ด้วย PARA method (Projects/Areas/Resources/Archives)
- ✅ Team workspace 7 databases (Projects, Tasks, Sprints, OKRs, Meetings, People, Wiki)
- ✅ Database schema design (property + view + filter)
- ✅ Relation + Rollup pattern (Project↔Task↔OKR)
- ✅ Template gallery (CRM, Content calendar, Sprint board, OKR tracker)
- ✅ Automation (Notion native + Zapier/Make)
- ✅ Full workspace blueprint

## 💡 Use cases

- Knowledge worker ที่ข้อมูลกระจาย → รวมที่เดียว
- Startup founder ตั้ง workspace ทีมตั้งแต่วันแรก
- Agency ที่ manage 20+ clients พร้อมกัน
- Content creator ที่ต้องการ content calendar + idea backlog
- HR ที่ต้องการ employee directory + onboarding

## 🎓 Tips

- เริ่ม **simple** ก่อน — 3-5 databases พอ ขยายทีหลัง
- ใช้ **Templates** ของ Notion — ไม่ต้องสร้างจาก 0
- ใช้คู่กับ `/project-manager-pro` → ออกแบบ Sprint + RAID log
- ใช้คู่กับ `/meeting-notetaker` → meetings database
