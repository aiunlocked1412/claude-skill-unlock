## 🔗 Relation + Rollup Map

```
OKRs ──┐
       └──→ Projects ──┐
                       └──→ Tasks ──→ Sprints
                            ↑
                       Meetings (action items)
```

**Key rollups:**
- Project.Completed % = Tasks done / total
- OKR.Progress = avg(Project.Completed %)
- Sprint.Velocity = sum(Tasks.Hours where Status=Done)
