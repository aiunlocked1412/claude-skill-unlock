- Freelancer อยากส่ง invoice + follow-up อัตโนมัติ
- Startup ต้องการ CRM + notification pipeline ที่ไม่ต้อง code

## 🎓 Tips

- อธิบาย **manual steps ปัจจุบัน** ให้ชัด → workflow design ตรง
- บอก **apps ที่ใช้อยู่** ทั้งหมด → หา integration ที่มีอยู่แล้ว
- เริ่มจาก **automation ง่ายๆ ก่อน** — 1-2 step ก่อนขยาย
- ใช้คู่กับ `/knowledge-manager` → automate capture notes ไป vault
