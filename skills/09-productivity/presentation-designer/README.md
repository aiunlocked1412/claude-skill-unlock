# Presentation Designer

> AI ออกแบบ slide ระดับมืออาชีพ — pitch deck, board presentation, exec summary, sales deck

## ⚡ ใช้ยังไง

```
/presentation-designer
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Pitch Deck Guy Kawasaki 10-slide (fundraising)
- ✅ Board / Executive presentation (15-20 slides)
- ✅ Sales presentation (problem → solution → ask)
- ✅ Executive Summary 1-page (pure doc, no slide)
- ✅ Demo Day / Conference talk (15-min storytelling)
- ✅ Training / Workshop deck
- ✅ Design system (color, font, layout)
- ✅ Speaker notes ทุก slide + Q&A prep

## 💡 Use cases

- Founder ระดมทุน Series A/B/C
- CEO ทำ board presentation รายไตรมาส
- Sales team เสนอ enterprise customer
- Conference speaker ที่อยาก slide สวย
- Internal: training, all-hands, town hall
- Consultant ที่ทำ deck ให้ลูกค้า

## 🎓 Tips

- **1 idea per slide** — ห้ามยัด 3 ไอเดียใน slide เดียว
- **10/20/30 rule** — 10 slides, 20 นาที, 30pt font (Guy Kawasaki)
- **Practice 3 ครั้ง** — ลด stutter + time check
- ใช้คู่กับ `/business-consultant` → strategic foundation
- ใช้คู่กับ `/copywriter-pro` → headline ที่ทรงพลัง
