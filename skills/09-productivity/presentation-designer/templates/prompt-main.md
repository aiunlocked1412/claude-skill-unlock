# Prompt Formula — Presentation Designer

## Template เก็บ context

```
ประเภท: <pitch / board / sales / exec summary / talk / training>
Audience: <VC / board / customer / internal team / general>
จำนวน: <ผู้ฟัง X คน>
Goal: <decision / commit / follow-up / educate / inspire>
Time slot: <X นาที>
Format: <in-person / Zoom / hybrid / async>
Brand: <มี style guide ไหม / สี / font>
Number of slides: <แล้วแต่ goal — pitch ~10, board ~15-20>
```

## Pitch Deck (Guy Kawasaki 10-slide) Recipe

```
Slide 1: Cover (logo + tagline)
Slide 2: Problem (1 stat + persona)
Slide 3: Solution (what we do — 1 sentence)
Slide 4: Market Size (TAM-SAM-SOM)
Slide 5: Product (screenshot / 3 features)
Slide 6: Business Model (how we make money)
Slide 7: Traction (3 hockey-stick metrics + logos)
Slide 8: Competition (2x2 matrix — we are top-right)
Slide 9: Team (photos + relevant experience)
Slide 10: Ask (amount + use of funds + milestone)

10/20/30 rule: 10 slides, 20 min, 30pt font
```

## Board Presentation Recipe

```
1. Cover + agenda
2. Executive summary (TL;DR)
3-5. Performance review (revenue, key metrics, growth)
6-8. Strategic update (product, market, team)
9-10. Risk + mitigation
11-12. Financial outlook (next quarter, next year)
13-14. Asks / decisions needed
15. Q&A / appendix
```

## Sales Deck Recipe

```
1. Cover (their logo + your value prop)
2. Their problem (specific to them — research!)
3. Cost of inaction
4. Our solution
5. How it works (3 steps)
6. Case study (similar customer)
7. ROI calculation
8. Implementation plan
9. Pricing + options
10. Next steps + close
```

## Executive Summary 1-page Recipe

```
1. Bottom line (3 sentences)
2. Key metrics (5-7 numbers)
3. Strategic rationale (3-5 bullets)
4. Risks & mitigations (top 3)
5. Investment required
6. Timeline
7. Ask (specific decision)
```

## Visual Hierarchy Rules

```
1. 1 idea per slide
2. Headline = takeaway (full sentence, not topic)
3. Font ≥ 24pt body, ≥ 36pt headline
4. Contrast ratio ≥ 4.5:1
5. 6×6 rule (≤ 6 bullets × ≤ 6 words)
6. White space ≥ 40%
7. Image > text when possible
8. Chart > table when possible
9. 1 chart per slide
10. Use color sparingly (3 max)
```

## Color Palette Recipe

```
Primary: brand color (logo)
Secondary: accent for CTA / highlight
Neutral 1: text dark (#333 not #000)
Neutral 2: subtitle / caption (#666)
Background: white / #FAFAFA

Avoid:
- Pure black on white (too harsh)
- Red + green together (color blind)
- More than 3 colors
- Gradient backgrounds (distracting)
```

## Typography Recipe

```
Headline: Sans-serif bold
- English: Inter, Helvetica, Arial, SF Pro
- Thai: Sarabun, Noto Sans Thai, Prompt

Body: Sans-serif regular (same family or pair)
- size: 24pt minimum
- line height: 1.4-1.6

Avoid:
- Comic Sans, Papyrus, Curlz
- Italic for body (hard to read)
- Underline (looks like link)
- More than 2 fonts in deck
```

## Speaker Notes Recipe

```
[SPEAKER NOTES — ทุก slide]

🎯 KEY MESSAGE (1 sentence)
สิ่งที่ผู้ฟังจำหลังจบ slide

📝 SCRIPT (30-60 sec — เขียนแบบพูด)
"..."

💡 TIPS
- Pause 3 sec ให้ดู visual ก่อนพูด
- Anticipate Q&A: ถ้าถาม X → ตอบ Y
- Optional anecdote: ...

⏰ TIMING: XX seconds
```

## Pre-presentation Checklist

```
1 day before:
- [ ] Practice 3 รอบ (timed)
- [ ] PDF backup
- [ ] Q&A prep (top 10 questions)
- [ ] Send pre-read (ถ้ามี)

Day of:
- [ ] Test screen / mic 30 min ก่อน
- [ ] Water + tissue + clicker
- [ ] Backup laptop + adapter
- [ ] Energy — eat light, no caffeine overdose
- [ ] Closing CTA prepared

After:
- [ ] Send PDF + thank-you ภายใน 24 ชม.
- [ ] Schedule follow-up
- [ ] Track outcome
```

## Tips

- **Practice in front of mirror** — see your gestures
- **Record yourself** — hear "um", "like", filler words
- **End on time** — going over = lose trust
- **Ask before tell** — "show of hands, ใครเคย ___?" engages
- **Pause after asking** — silence = audience thinks
- **Acknowledge tough questions** — "great question, here's how I think about it"
- **Last slide is most remembered** — make it strong + clear ask
