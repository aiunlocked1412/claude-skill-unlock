# Time Manager

> AI ผู้จัดการเวลา — Pomodoro, Time Block, GTD, Deep Work, Calendar Design รองรับทุก role และ tool

## ⚡ ใช้ยังไง

```
/time-manager
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Pomodoro System (session design ตาม work type + energy)
- ✅ Time Block Weekly Plan (template + color coding)
- ✅ GTD Setup (5 steps + Notion database structure)
- ✅ Deep Work Schedule (focus window + distraction defense)
- ✅ Calendar Design (Google Cal, Notion, Todoist integration)
- ✅ Time Audit (วิเคราะห์ว่าเวลาหายไปไหน)

## 💡 Use cases

- Freelancer ต้องการ structure วันทำงานที่ไม่มี boss กำหนด
- Office worker อยากสร้าง deep work slot ท่ามกลาง meeting
- Student ต้องการ study schedule + Pomodoro สำหรับสอบ
- Entrepreneur ต้องการ time block จัดลำดับความสำคัญ
- Remote worker ต้องการ work-life boundary ที่ชัดเจน

## 🎓 Tips

- บอก **energy pattern** (morning/night person) → schedule ที่ยั่งยืนขึ้น
- ระบุ **tool ที่ใช้อยู่แล้ว** → ไม่ต้องเริ่มจากศูนย์
- ใช้คู่กับ `/agile-coach` → sprint planning + time management
- ใช้คู่กับ `/knowledge-manager` → ทำ GTD reference system ด้วย Obsidian
