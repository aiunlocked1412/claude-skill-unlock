# Blockchain Developer

> AI นักพัฒนา Blockchain/Web3 — Smart Contract, DeFi Protocol, Security Audit เบื้องต้น, Deploy Testnet/Mainnet

## ⚡ ใช้ยังไง

```
/blockchain-dev
```

## 🎯 ทำอะไรได้บ้าง

- ✅ เขียน Smart Contract (ERC-20 / ERC-721 / ERC-1155 / custom Solidity/Rust)
- ✅ Security Audit เบื้องต้น — ตรวจ reentrancy, access control, oracle, rug pull
- ✅ DeFi Protocol Design (DEX, Lending, Staking, Yield Farming)
- ✅ Deploy Script สำหรับ Hardhat / Foundry / Anchor
- ✅ Test Suite (unit + integration) ด้วย Hardhat/Foundry
- ✅ Tokenomics & Architecture Review
- ✅ Full Stack Web3 Boilerplate (contract + frontend + wallet connect)

## 💡 Use cases

- Startup อยากออก token / NFT collection บน Ethereum หรือ Polygon
- Dev เขียน DeFi protocol แต่ต้องการตรวจ security ก่อน deploy
- ทีมต้องการ boilerplate staking contract พร้อม test
- Founder ต้องการ review tokenomics ก่อนเปิด IDO
- นักพัฒนามือใหม่ Web3 อยากเรียนรู้ pattern ที่ถูกต้อง

## 📊 ตัวอย่างผลลัพธ์

Solidity contract พร้อม NatSpec comment + deploy script + test file + gas report

## 🎓 Tips

- ระบุ **chain** ที่ต้องการ deploy (Ethereum/Polygon/BSC/Solana) → ได้ code ที่ตรง toolchain
- บอก **feature** ที่ต้องการ (mint role, pause, upgrade) → ได้ contract ที่สมบูรณ์
- วาง **Solidity code** มาพร้อม audit request → ได้รายงาน vulnerability ละเอียด
