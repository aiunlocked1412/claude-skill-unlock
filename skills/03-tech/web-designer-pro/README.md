# Web Designer Pro

> AI Web/UX Designer — wireframe + component + style guide พร้อม handoff ให้ dev

## ใช้ยังไง

```
/web-designer-pro
```

## ทำอะไรได้บ้าง

- Wireframe ASCII/markdown ของทั้งหน้า
- Component spec (Button / Card / Form / Modal / Nav)
- Design System (color, typography, spacing, shadow token)
- Responsive breakpoint plan (mobile → tablet → desktop)
- Developer handoff doc พร้อม CSS variables
- Design review / critique หน้าที่มีอยู่

## Output พร้อมใช้

- Copy ไป Figma / Sketch ได้
- CSS tokens copy ไป Tailwind config ได้
- HTML/CSS example สำหรับ component สำคัญ
- Accessibility checklist (WCAG AA)

## Use cases

- Landing page SaaS / startup
- E-commerce (homepage, product page)
- Dashboard admin
- Portfolio / agency site
- Blog / magazine
- App marketing page

## Tips

- บอก brand tone ให้ชัด — corporate / playful / luxury / minimalist
- แปะ reference link 2-3 เว็บที่ชอบ (จะได้จับ aesthetic)
- บอก tech stack — Next.js/Tailwind? Webflow? WordPress?
