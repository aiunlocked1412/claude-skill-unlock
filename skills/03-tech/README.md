# หมวด 3: Tech & Development (6 skills)

Skills สำหรับ developer, data analyst, DevOps, QA

| Skill | คำสั่ง | ใช้เมื่อไหร่ |
|-------|--------|-------------|
| Programmer | `/programmer-pro` | review, debug, refactor |
| Data Analyst | `/data-analyst-pro` | SQL, insight, dashboard |
| Web Designer | `/web-designer-pro` | wireframe, component spec |
| QA Tester | `/qa-tester-pro` | test case, bug report |
| No-Code Builder | `/no-code-builder` | Zapier, Make, Airtable |
| DevOps Helper | `/devops-helper` | Docker, CI/CD, monitoring |

## รูปแบบ output ทั่วไป

- Code snippet พร้อม comment
- Markdown spec/report
- YAML config (CI/CD, Docker)
- Diagram spec (flow, ER, component)

## Workflow ตัวอย่าง

```
1. /web-designer-pro wireframe landing page
   ↓
2. /programmer-pro เขียน component React
   ↓
3. /qa-tester-pro test cases
   ↓
4. /devops-helper deploy ขึ้น production
   ↓
5. /data-analyst-pro วิเคราะห์ traffic หลัง launch
```
