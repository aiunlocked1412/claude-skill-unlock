### F-001: <ชื่อช่องโหว่>

| Field | Value |
|-------|-------|
| Severity | Critical |
| CWE | CWE-89 (SQL Injection) |
| OWASP | A03:2021 |
| Location | `routes/login.js:45` |
| CVSS 3.1 | 9.8 (High) |

**Description:**
> อธิบายช่องโหว่ + how attacker exploit

**Proof of Concept:**
```bash
curl -X POST https://api.example.com/login \
  -d "email=' OR '1'='1&password=anything"
# ได้ token admin โดยไม่รู้ password
```

**Impact:**
- เข้า admin panel โดยไม่ต้อง credential
- Dump user table 1M rows ใน 5 นาที
- Pivot ไปยัง internal network ได้

**Vulnerable Code:**
```javascript
// routes/login.js:45
db.query(`SELECT * FROM users WHERE email='${email}'`)
```

**Remediation:**
```javascript
// Use parameterized query
db.query('SELECT * FROM users WHERE email = ?', [email])
```

**References:**
- [OWASP — SQL Injection](https://owasp.org/...)
- [CWE-89](https://cwe.mitre.org/data/definitions/89.html)
