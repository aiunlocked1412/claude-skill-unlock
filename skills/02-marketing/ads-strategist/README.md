# Ads Strategist

> AI กลยุทธ์ Ads — Google/Meta/TikTok/Line Ads — Campaign Structure, Audience, Creative Brief, Budget Allocation, Bid Strategy สำหรับตลาดไทย

## ⚡ ใช้ยังไง

```
/ads-strategist
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Campaign Structure (account → campaign → ad set → ad สำหรับทุก platform)
- ✅ Audience Strategy (interest, lookalike, retargeting, custom audience)
- ✅ Creative Brief (hook, body, CTA, format spec ต่อแพลตฟอร์ม)
- ✅ Budget Allocation (จัด budget ระหว่าง channel + funnel stage)
- ✅ Bid Strategy Guide (CPC, CPM, CPA target, ROAS bidding)
- ✅ Performance Audit (วิเคราะห์ campaign + หาจุดปรับ)
- ✅ Full Ads Strategy + Launch Timeline

## 💡 Use cases

- SME ไทยอยากเริ่ม Meta Ads แต่ไม่รู้จะ structure ยังไง
- E-commerce อยากจัด budget ระหว่าง Shopee Ads + Meta + TikTok
- Agency ต้องการ creative brief มาตรฐานสำหรับทีม production
- Brand manager อยากทำ audit campaign ที่ ROAS ต่ำกว่าเป้า
- Performance marketer อยากเข้าใจ audience strategy ตลาดไทย

## 📊 ตัวอย่างผลลัพธ์

เมื่อ run `/ads-strategist full ads strategy` จะได้:
- Campaign structure พร้อม naming convention
- Audience segment ทุก funnel stage
- Creative brief ทุก ad format
- Budget split table
- Launch timeline + optimization schedule

## 🎓 Tips

- บอก **platform หลัก** ก่อน — strategy Meta ≠ TikTok ≠ Google
- บอก **งบต่อวัน/เดือน** — ส่งผลต่อ recommendation อย่างมาก
- แนบ **ผลลัพธ์ปัจจุบัน** ถ้ามี → audit ได้ตรงจุดกว่า
- ใช้คู่กับ `/conversion-optimizer` → ads นำ traffic ดี แต่ landing page ต้อง convert
