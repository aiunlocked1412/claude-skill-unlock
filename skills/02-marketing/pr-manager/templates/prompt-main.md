
## Newsworthiness Test (5/3 Rule)

ทุก story ต้องผ่านอย่างน้อย 3 จาก 5:

| Element | Question | Score |
|---------|----------|-------|
| First | "เป็นครั้งแรกของ <industry>?" | 0-1 |
| Biggest/Only | "ใหญ่ที่สุด หรือมีคนเดียว?" | 0-1 |
| Timely | "ผูกกับ event ตอนนี้?" | 0-1 |
| Impact | "กระทบคนกี่คน?" | 0-1 |
| Human interest | "มีคน + story?" | 0-1 |

ผ่าน 3+ → คุ้ม pitch
ผ่าน 1-2 → ทำเป็น blog แทน
ผ่าน 0 → ไม่ใช่ news

## Headline Formulas (Press Release)

1. **[Brand] [Action] [Specific Number] [Outcome]**
   "Skooldio ระดมทุน 200 ล้าน เร่งขยาย skill ดิจิทัลในไทย"

2. **[Brand] เปิดตัว [Product] — [USP เด่น]**
   "Plotters Cloud เปิดตัว AI plotter ตัวแรกที่อ่าน Chinese script"

3. **[Number] of [Industry] [Trend Insight]**
   "ผลสำรวจ: 73% ของพนักงานไทย พร้อมลาออกใน 6 เดือน"

4. **[Brand] ผนึก [Partner] [Outcome]**
   "Bitkub ผนึก SCB เปิดตัว Crypto savings account แรกในไทย"

5. **[Brand] รับรางวัล [Award] [Authority]**
   "Lalamove ได้ Best E-Logistics 2026 จาก ASEAN Awards"

## Quote Writing Rules

### ❌ ห้าม (marketing speak)
> "เราภูมิใจเป็นอย่างยิ่งที่ได้นำเสนอนวัตกรรมที่จะปฏิวัติวงการ"

### ✅ ดี (specific + insightful)
> "เราเริ่มสังเกตว่า 60% ของลูกค้าทิ้งตะกร้าตอน checkout เพราะ payment ไม่รองรับ — Series A นี้จะแก้ปัญหานี้ใน 12 เดือน"
>
> — สมชาย ใจดี, CEO และ Co-founder, Skooldio

**Quote ที่ดีต้อง:**
- มี data / specific number
- บอก "why this matters now"
- ใช้คำพูดที่ CEO พูดจริงได้
- หลีกเลี่ยง "exciting", "thrilled", "revolutionary"

## Crisis Communication Framework

### Severity Levels

| Level | Example | Response time |
|-------|---------|--------------|
| L1 (Minor) | Negative review on social | 4 hours |
| L2 (Moderate) | Viral complaint, 1k+ engagement | 1 hour holding |
| L3 (Major) | Product recall, lawsuit | Immediate holding |
| L4 (Critical) | Death/injury, data breach | Immediate + legal |

### Holding Statement Template (1 ชม. แรก)

```
เราได้รับทราบเรื่อง [issue] แล้วและกำลังตรวจสอบอย่างละเอียด

ลูกค้า/พนักงานคือสิ่งสำคัญที่สุดของเรา และเรารับเรื่องนี้อย่างจริงจัง

ทีมงานกำลังประสานงานกับ [authority/expert] เพื่อหาคำตอบที่ถูกต้อง
เราจะแถลงรายละเอียดภายใน [time frame] ครับ

— <ชื่อ บริษัท>
```

### Official Statement (6-24 ชม.)

3 paragraph:
1. **Acknowledge** — ยอมรับสิ่งที่เกิด ไม่ปฏิเสธความรู้สึก
2. **Action** — สิ่งที่กำลังทำ (specific + verifiable)
3. **Commitment** — สิ่งที่จะทำต่อ + ป้องกันในอนาคต

### Don'ts
- "No comment"
- ตอบ technical jargon (ลูกค้าไม่เข้าใจ)
- โทษคนอื่น / 3rd party
- Defensive tone
- ส่ง low-rank spokesperson ใน crisis ใหญ่

## Spokesperson Training — Bridge Phrases

ใช้ตอบ tough question + redirect ไป key message:

- "ประเด็นที่สำคัญกว่านั้นคือ..."
- "ขอ context ก่อนนะคะ..."
- "ตอนนี้สิ่งที่เราโฟกัสคือ..."
- "ขออนุญาตตอบในมุม..."
- "เป็นคำถามดี — ขอแยกเป็น 2 ส่วน..."

## Earned Media Metrics

| Metric | Definition | Benchmark |
|--------|-----------|-----------|
| Reach | Estimated audience exposed | Tier 1 = 100k-1M |
| Coverage count | จำนวน article | 5-15 ต่อ launch |
| Sentiment | Pos / Neutral / Neg % | >70% pos = good |
| Share of Voice | % vs competitor | >25% = leader |
| Message pull-through | Key message ปรากฏ % | >60% = on-msg |
| AVE | Ad Value Equivalent | PR ROI estimate |

## Common Mistakes

1. Press release ที่ไม่ใช่ news — กลายเป็น spam
2. ไม่ personalize pitch — auto-delete
3. ส่ง mass แทน exclusive — Tier 1 ไม่สนใจ
4. Quote marketing speak — ไม่ถูก quote
5. Crisis silence — issue amplify
6. ไม่มี media monitoring — พลาด narrative
7. PR แต่ไม่ amplify ใน social ของตัวเอง — เสีย opportunity
