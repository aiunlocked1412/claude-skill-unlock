# PR Manager

> AI PR Manager — press release, media pitch, crisis communication สำหรับธุรกิจไทย

## ⚡ ใช้ยังไง

```
/pr-manager
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Press release AP style (ไทย + อังกฤษ)
- ✅ Media pitch email ที่ journalist ตอบ
- ✅ Media list ไทย matched กับ industry (Matichon, Prachachat, The Standard, etc.)
- ✅ Crisis communication playbook 24-48h response
- ✅ Spokesperson training brief (Q&A + bridge phrases)
- ✅ Story angle workshop (5 angle จาก 1 story)
- ✅ Earned media report (reach + sentiment + AVE)
- ✅ Full PR campaign tracking

## 💡 Use cases

- Funding announcement (Series A/B/C)
- Product launch ที่อยากขึ้นข่าว
- Partnership / M&A announcement
- Crisis: viral negative review, data breach, recall
- Founder story / award announcement
- Industry report release

## 🎓 Tips

- **Newsworthiness** ต้องผ่าน 3/5 ข้อ ก่อน pitch (First/Biggest/Only, Timely, Impact, Conflict, Human)
- ส่ง embargo + exclusive ให้ Tier 1 ก่อน 24-48 ชม. ก่อน mass send
- Pitch ต้อง personalize — อ้าง article ของ reporter จริง
- Crisis: 1 ชม. แรก = holding statement, 6 ชม. = official, 24 ชม. = full action plan
