### Anticipated Q&A

**Q: 200M ใหญ่ขนาดนี้ในตลาดที่ยังเล็ก — overpriced ไหม?**
> A: ขอ context ก่อนนะครับ — ตลาด HR tech SEA จะถึง 2.4B USD ใน 2028 (Gartner) ตอนนี้เรา dominate ไทย 500 บริษัท + 200,000 พนักงาน Series A นี้สำหรับขยาย 3 ประเทศ — ตัวเลข fundraise สอดคล้อง market opportunity

**Q: คู่แข่งใหญ่อย่าง Workday, BambooHR เข้ามา — สู้ยังไง?**
> A: ประเด็นที่สำคัญกว่านั้นคือ — global player ออกแบบสำหรับองค์กร US/EU แต่ HR ไทย/ASEAN มี context ต่าง: payroll, leave, ภาษี, วัฒนธรรม PeopleHQ build จาก ground up สำหรับ SEA — นั่นคือ moat ของเรา

**Q: AI Agent นี้จะแทนที่ HR ไหม?**
> A: ตอนนี้สิ่งที่เราโฟกัสคือ — เอางาน admin 60% ออกจาก HR เพื่อให้พวกเขาได้ทำสิ่งที่ AI ทำไม่ได้: คุยกับคนจริง coach พนักงาน นั่นคือ "เพิ่มอำนาจ HR" ไม่ใช่ "แทนที่"

**Q: ลูกค้าเอ่ยชื่อ CP All / Krungsri / Lazada — เขายินดีให้พูดถึงไหม?**
> A: ใช่ครับ ทั้ง 3 ราย sign release ให้ใช้ชื่อใน announcement นี้แล้ว มี case study และ quote จาก HR Director ของ CP All ให้ด้วยถ้าสนใจ
