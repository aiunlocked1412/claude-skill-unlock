# Community Manager

> AI ผู้ดูแลคอมมูนิตี้ — Discord/FB Group/Line OA — Onboarding, Engagement Calendar, Moderation, Crisis, Member Retention

## ⚡ ใช้ยังไง

```
/community-manager
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Onboarding Flow (welcome journey 7 วันแรก + message template)
- ✅ Engagement Calendar (กิจกรรมรายสัปดาห์/เดือน + theme)
- ✅ Moderation System (roles, warning 3 ขั้น, ban policy)
- ✅ Crisis Handling (CALM protocol + response script)
- ✅ Retention Campaign (re-engage inactive + loyalty tier)
- ✅ Community Health Metrics (DAU, engagement rate, churn)
- ✅ Full Community Playbook (ทุกอย่างรวมกัน)

## 💡 Use cases

- แบรนด์ไทยเปิด Line OA / FB Group ใหม่ อยากได้ระบบที่ทำงานได้จริง
- Creator เปิด Discord server fan club แต่ไม่รู้จะดูแลยังไง
- Community ขนาดกลาง (1,000-10,000 คน) engagement ตก อยากกลับมา
- ทีม mod ที่ต้องการ SOP ชัดเจนสำหรับจัดการ crisis
- Brand manager อยากวัด community health เป็นตัวเลข

## 📊 ตัวอย่างผลลัพธ์

เมื่อ run `/community-manager full playbook` จะได้:
- Welcome message template (Line OA / Discord / FB Group)
- Engagement calendar 4 สัปดาห์
- Moderation policy + role structure
- Crisis response script
- KPI dashboard template

## 🎓 Tips

- บอก **ขนาด community** (เล็ก/กลาง/ใหญ่) → strategy ต่างกัน
- บอก **goal หลัก** (retention vs engagement vs monetize) → focus ต่างกัน
- **Line OA** มีข้อจำกัด broadcast — วางแผน frequency ก่อน
- ใช้คู่กับ `/growth-hacker` → ดึงสมาชิกใหม่เข้า community
