# หมวด 2: Marketing & Sales (14 skills)

Skills สำหรับ digital marketer, brand strategist, sales pro, PR, event, CRM, research

| Skill | คำสั่ง | ใช้เมื่อไหร่ |
|-------|--------|-------------|
| Digital Marketer | `/digital-marketer` | วางแผน campaign, budget, KPI |
| Social Media Admin | `/social-media-admin` | ปฏิทินโพส, caption, reply |
| SEO Specialist | `/seo-specialist` | keyword, on-page audit |
| Copywriter | `/copywriter-pro` | headline, ad copy, email |
| Online Seller | `/online-seller` | live script, product caption |
| Influencer Marketing | `/influencer-marketing` | brief, negotiation, review |
| **Email Marketer** | `/email-marketer` | sequence, automation, deliverability, segmentation |
| **Affiliate Marketer** | `/affiliate-marketer` | commission, partner outreach, tracking |
| **Brand Strategist** | `/brand-strategist` | archetype, positioning, voice, brand book |
| **PR Manager** | `/pr-manager` | press release, media pitch, crisis communication |
| **Event Marketer** | `/event-marketer` | event planning, promotion, sponsor, post-event |
| **CRM Strategist** | `/crm-strategist` | RFM, lifecycle, churn prediction, loyalty |
| **Sales Pro** | `/sales-pro` | B2B sales, cold outreach, objection, pipeline |
| **Market Researcher** | `/market-researcher` | survey, focus group, persona, JTBD, competitive |

## รูปแบบ output ทั่วไป

- Campaign brief (ตารางรวม)
- Copy variants 5-10 แบบ
- Posting calendar (30 วัน)
- KPI dashboard spec
- Persona + JTBD synthesis
- Email sequence + automation flow
- Sales playbook + script
- PR campaign + press kit
- Brand book + voice matrix
- CRM segmentation + lifecycle map

## Workflow ตัวอย่าง — Launch แบรนด์ใหม่

```
1. /market-researcher วิจัยตลาด + persona + competitive
   ↓
2. /brand-strategist ออกแบบ archetype + positioning + brand book
   ↓
3. /digital-marketer วางแผน launch campaign
   ↓
4. /copywriter-pro เขียน ad copy + email + landing
   ↓
5. /pr-manager ออก press release + media pitch
   ↓
6. /event-marketer จัด launch event
   ↓
7. /social-media-admin ปฏิทินโพส 30 วัน
   ↓
8. /influencer-marketing หา KOL 5 คน
   ↓
9. /affiliate-marketer ตั้ง affiliate program
   ↓
10. /email-marketer welcome series + abandoned cart
    ↓
11. /seo-specialist ทำ SEO ให้ landing
    ↓
12. /online-seller live script + product page
```
