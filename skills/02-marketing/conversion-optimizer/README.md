## 🎓 Tips

- บอก **current CVR** ถ้ามี — ช่วย benchmark ว่าห่างจาก good เท่าไหร่
- บอก **traffic source** (organic vs paid ads) → user intent ต่างกัน → strategy ต่าง
- **Mobile-first เสมอ** — ลูกค้าไทย > 70% ใช้ mobile
- ใช้คู่กับ `/ads-strategist` → ads นำ traffic ที่ quality แล้ว page ต้อง convert
