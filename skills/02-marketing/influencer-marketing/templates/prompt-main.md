## Tier & Rate Benchmark (ไทย 2026)

### Nano (1-10k follower)
- **ER benchmark:** 5-8%
- **IG Reel/Post rate:** 1,500-5,000
- **TikTok video:** 2,000-6,000
- **Best for:** Niche product, UGC, micro test

### Micro (10-100k)
- **ER benchmark:** 3-5%
- **IG Reel/Post:** 5,000-25,000
- **IG Story (set 3):** 3,000-10,000
- **TikTok video:** 8,000-35,000
- **YouTube integrated (5-min):** 15,000-60,000
- **Best for:** Trust-building, conversion

### Mid (100k-500k)
- **ER benchmark:** 2-3%
- **IG Reel/Post:** 25,000-100,000
- **TikTok video:** 35,000-150,000
- **YouTube full video:** 80,000-250,000
- **Best for:** Reach + credibility + brand lift

### Macro (500k+)
- **ER benchmark:** 1-2% (ต่ำกว่านี้คือ bot)
- **IG Reel:** 100,000-500,000
- **TikTok:** 150,000-600,000
- **YouTube:** 250,000-1,500,000
- **Best for:** Awareness แบรนด์ใหญ่ / mass launch

### Celeb/Mega (2M+)
- **Rate:** Custom (500k-5M+ / โพส)
- **Best for:** PR stunt, global awareness

## Fair Rate Formula

```
Base fee = Avg view × CPM benchmark
         + Production cost (8-20% of base)
         + Usage right (30-100% if repurpose)
         + Exclusivity (20-40% if restrict)
         + Rush (25% if < 7 days)
```

### CPM Benchmark (ไทย 2026)

| Platform | Low | Mid | High |
|----------|-----|-----|------|
| TikTok video | 120 | 160 | 200 |
| IG Reel | 100 | 140 | 180 |
| IG Static | 80 | 115 | 150 |
| IG Story | 50 | 75 | 100 |
| YouTube integrated | 200 | 300 | 400 |
| YouTube dedicated | 400 | 600 | 800 |
| Facebook video | 60 | 90 | 120 |

### Example Calculation

KOL: IG Reel 85k follower, avg 30k view/reel, ER 3.2%

```
Base: 30,000 view × 140 บาท/1k = 4,200
+ Production: 4,200 × 15% = 630
+ Usage right (ad boost 30 days): 4,200 × 50% = 2,100
+ Exclusivity 30 days: 4,200 × 25% = 1,050

Total fair rate: 7,980 ≈ 8,000 บาท
```

ถ้า KOL quote > 12,000 = แพงเกิน 50% → ต่อรอง
ถ้า KOL quote < 5,000 = อาจเป็น bot / desperate → vet เพิ่ม

## Vetting Red Flags

### 🚨 Fake Follower
- Follower jump > 20% / เดือน โดยไม่มี viral post
- Engagement rate < tier benchmark / 2
- Comment ส่วนใหญ่ = emoji + 1 word
- Follower country ไทย < 50% (ถ้า target ไทย)

### 🚨 Engagement Pod
- คนเดิม comment ทุกโพสหลัก
- Comment timing = ภายใน 5 นาทีทุกครั้ง
- ใช้ comment template เดียวกัน (copy pattern)

### 🚨 Brand Risk
- โพสที่ลบใน 6 เดือนล่าสุด (screenshot web archive)
- Controversy ใน Twitter/X
- Past brand mention ที่ไม่ดี

### 🚨 Over-sponsored
- 5 โพสหลังคือ sponsored content ทั้งหมด = audience blind
- แบรนด์ที่ผ่านมาคือคู่แข่งโดยตรง 2 เดือน

## Pitch Email Template

```
Subject: <KOL name> — <specific content> ที่ทำให้อยากร่วมงาน

สวัสดีค่ะ/ครับ คุณ <ชื่อ>

ดูคลิป "<ชื่อคลิปจริง>" เมื่อสัปดาห์ที่แล้ว
ชอบตรงที่ <specific — พิสูจน์ว่าดูจริง>

เราเป็น <1 ประโยคบอกแบรนด์>
สังเกตเห็นว่า content ของคุณ
มี <audience/tone> ที่ตรงกับกลุ่มเป้าหมายเรา

อยากชวนร่วมงาน:
📦 Product: <product + ส่ง free sample>
📝 Deliverable: <X posts + Y stories>
💰 Budget: <range — เช่น 15,000-25,000>
📅 Timeline: <กรอบเวลา>

ถ้าสนใจ — reply กลับได้เลยค่ะ
ไม่สะดวกก็บอกตรงๆ ได้ค่ะ จะไม่ follow up หลายรอบ 🙏

ขอบคุณล่วงหน้าค่ะ
<ชื่อ + บริษัท + link>
```

### Best Practices
- **Subject:** ใช้ชื่อ KOL + specific detail
- **Body length:** ≤ 200 คำ
- **Budget:** บอก range ตั้งแต่แรก
- **Out clause:** "บอกตรงๆ ได้ถ้าไม่สะดวก" — reply rate ดีขึ้น
- **Send time:** จันทร์-พุธ 10-11 โมง

## Disclosure Requirement (ไทย)

ต้องมี:
- `#โฆษณา` หรือ `#ได้รับการสนับสนุน`
- `#ads`, `#paid partnership`, `#sponsored` (บวกไม่เสีย)
- บอกชัดในคลิป/caption ว่าเป็น sponsored
- IG paid partnership tag
- YouTube: "Includes paid promotion" checkbox
