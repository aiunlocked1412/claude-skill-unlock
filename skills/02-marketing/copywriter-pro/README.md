# Copywriter Pro

> AI Copywriter — headline, ad copy, email sequence, landing copy ที่ขายได้จริง

## ⚡ ใช้ยังไง

```
/copywriter-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Headline 10 variants (5 angle × 2 variant — A/B test ready)
- ✅ Ad copy FB / IG / Google (3-5 angle: pain / benefit / story / proof / compare)
- ✅ Email sequence 5 ตัว (welcome / story / teach / sell / last-call)
- ✅ Landing page copy ครบ (headline → FAQ → final CTA)
- ✅ Sales / VSL script (long-form)
- ✅ Product description e-commerce (short + long)

## 💡 Use cases

- Launch สินค้าใหม่
- โปรโมชั่นตามฤดูกาล
- Email list nurture
- Re-engagement customer เก่า
- Landing page คอร์ส / บริการ / product
