## Live 5-Act Structure (30 นาที)

```
0:00 ├─ Act 1: Warm-up (3 min)
     │   ทักทาย + tease + ขอกดแชร์
3:00 ├─ Act 2: Demo (12 min)
     │   โชว์จริง + pain-solution + story
15:00├─ Act 3: Close #1 (5 min)
     │   Promo live + scarcity + CTA
20:00├─ Act 4: Social proof + Q&A (6 min)
     │   Review + handle objection
26:00└─ Act 5: Final close (4 min)
         Countdown + bump offer + หลังไลฟ์ทำไง
```

---

## Live Hook Templates (Act 1)

- "ใครมาดูคนแรก — จำชื่อไว้ ท้ายไลฟ์มีของแจก"
- "วันนี้ 3 เรื่องพิเศษ — ของเด็ด + ราคาลับ + แถมที่ไม่เคยแถม"
- "ถ้ากำลังหา [product type] — คลิปนี้ 30 นาทีต้องดูให้ครบ"
- "<จำนวน> ชิ้นสุดท้าย — วันนี้ขายหมดปิด restock 2 สัปดาห์"

## Demo Framework (Act 2)

1. **Context** — "ใครเคย <pain>?"
2. **Problem** — "ปกติเราแก้ยังไง... แต่ยัง <remaining pain>"
3. **Reveal** — "<product> ต่างตรงนี้..."
4. **Live demo** — ทำให้ดู 30-60 วิ
5. **Result** — โชว์ผลลัพธ์
6. **Bridge to next** — "ทีนี้ข้อต่อไปสำคัญ..."

## Close Framework (Act 3 & 5)

### Close ตัว 1 (mid-live)
1. Price reveal (anchor ราคา → วันนี้เท่าไหร่)
2. Promo live — "เฉพาะคนในไลฟ์วันนี้"
3. Trigger keyword — "คอมเม้นต์ 'จอง' แอดมินจะทัก"
4. Urgency — "เหลือ X ชิ้น"

### Close ตัว 2 (final)
1. Recap มูลค่ารวม
2. Bump offer — "ถ้าจองใน 3 นาที แถมเพิ่ม..."
3. Countdown 3-2-1
4. Direction for late — "พลาดไลฟ์ ทัก LINE รับ price เดิม 24 ชม."

## Pricing Psychology Toolbox

### 1. Anchor Pricing
```

Now: 690 บาท
Save: 200 บาท (22%)
```

### 2. Bundle
```

```

## Platform-specific Tips

### Shopee
- Title ≤ 60 char (SEO-friendly + keyword)
- Description: benefits top 3 บรรทัดแรก
- Brand registered → unlock ad + badge
- ห้าม link ภายนอก

### Lazada
- ใช้ LazMall เพื่อ trust
- Flash deal ขึ้น 23:00, 1:00
- Free shipping label สำคัญ

### TikTok Shop
- Short video 15-60 วิ ขายได้ดีสุด
- Affiliate program เปิดให้ creator
- Hook 3 วิแรกต้องโชว์สินค้า

### FB Live
- Boost ad 1 ชั่วโมงก่อนไลฟ์
- ตั้ง keyword auto-reply "จอง", "สนใจ"
- Replay upload เป็น video ต่อ

### LINE OA
- OA 60k friends ขึ้น = invest Rich Menu
- Broadcast best time 11:30, 17:30, 20:30
- สตอรี่ + Card Message > plain text
