# Online Seller

> AI สำหรับพ่อค้าแม่ค้าออนไลน์ — live script, product caption, FAQ, pricing

## ⚡ ใช้ยังไง

```
/online-seller
```

## 🎯 ทำอะไรได้บ้าง

- ✅ สคริปต์ Live 15/30/60 นาที (5 act structure — warmup → demo → close)
- ✅ Product caption 3 variants (story / benefit / deal)
- ✅ FAQ bank 20 คำถาม (product / price / shipping / return)
- ✅ Pricing psychology (anchor, bundle, decoy, payment plan)
- ✅ Shopee/Lazada listing (title + desc + spec)
- ✅ TikTok Shop short-form 15-60 วิ

## 💡 Use cases

- FB Live / IG Live / TikTok Live
- Shopee Live / Lazada Live
- ร้านเครื่องสำอาง / อาหาร / แฟชั่น / บ้านและสวน / pet
- Pre-order แบรนด์ใหม่
