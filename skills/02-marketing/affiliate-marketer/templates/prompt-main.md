## Commission Economics Formula

```
Max sustainable commission = AOV × Margin% - Operating cost - Target profit

ตัวอย่าง: AOV 1,000 / Margin 60% / Ops 100 / Target profit 200
Max commission = 1,000 × 0.60 - 100 - 200 = 300 (30%)

ถ้า LTV > AOV × 3 → จ่ายได้เยอะกว่า เพราะ recoup จาก repeat
```

## Tracking Architecture

```
[Affiliate Link]
    ↓ click
[Landing Page] — drop cookie + UTM
    ↓ browse / leave
[Return visit X days later] — read cookie
    ↓ purchase
[Checkout] — fire conversion pixel
    ↓ postback
[Affiliate Platform] — credit partner
    ↓ wait refund window 7-14 day
[Approve commission] → next payout
```

## Fraud Patterns + Detection

| Pattern | Detection |
|---------|-----------|
| Self-affiliate | IP/email match between partner + buyer |
| Cookie stuffing | Impression:click ratio > 1000:1 |
| Brand bidding | Search "<brand>" PPC ad with affiliate link |
| Refund abuse | Partner refund rate > 30% |
| Velocity spike | Conversion 10x normal in 24h |
| Bot traffic | High traffic + 0% engagement |

---

## Partner Lifecycle

```
Apply → Approve → Onboard → First sale (target: 14 day)
  → Active (1+ sale/month) → Top tier (10+/mo)
  → Champion (referral source)
```

Drop-off zones:
- Apply → Approve: ~50% drop
- Approve → First sale: ~70% drop ← KEY
- First sale → Repeat: ~40% drop

**Solution:** First-week onboarding email sequence + 30-day check-in call

## Common Mistakes

1. ตั้ง commission ต่ำกว่า benchmark → partner ไม่สนใจ
2. Cookie 24 ชม. → partner ไม่ได้เครดิต
3. ไม่มี onboarding → 70% drop หลัง approve
4. จ่ายช้า → partner เลิก promote
5. ไม่ดูแล top performer → คู่แข่งดึงตัว
6. Fraud detection อ่อน → จ่าย commission ผิด → margin หาย
7. Brand bidding ไม่บล็อก → คู่แข่งฟ้องได้
