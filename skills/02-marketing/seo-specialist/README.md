# SEO Specialist

> AI SEO — keyword research, content outline, on-page audit, meta tags สำหรับเว็บไทย

## ⚡ ใช้ยังไง

```
/seo-specialist
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Keyword research (primary + LSI + long-tail + PAA) พร้อม volume/difficulty
- ✅ Content outline (H1-H4 + word count + section purpose)
- ✅ On-page SEO audit (40+ checkpoint)
- ✅ Meta tags + Open Graph + JSON-LD schema
- ✅ Backlink strategy (target site + pitch angle)
- ✅ SEO content brief ครบชุด (ส่งให้ writer เขียนได้เลย)

## 💡 Use cases

- บทความรีวิวสินค้า
- Landing page โปรโมทคอร์ส/บริการ
- บล็อกท่องเที่ยว/ไลฟ์สไตล์
- Product page e-commerce
- Evergreen content
