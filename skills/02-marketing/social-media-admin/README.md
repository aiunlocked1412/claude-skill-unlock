# Social Media Admin

> AI แอดมินโซเชียล — ปฏิทินโพส, caption, reply template, hashtag สำหรับแบรนด์ไทย

## ⚡ ใช้ยังไง

```
/social-media-admin
```

## 🎯 ทำอะไรได้บ้าง

- ✅ ปฏิทินโพส 30 วัน (3-5 โพส/สัปดาห์) พร้อม hook + caption + visual idea
- ✅ Caption 3 variants ต่อโพส (A/B testable)
- ✅ Reply template (DM, comment, negative review)
- ✅ Hashtag bank จัดกลุ่ม (broad / niche / branded)
- ✅ Content pillar แบ่ง 4E (Educate/Entertain/Engage/Evangelize)
- ✅ Crisis playbook (complaint, refund, public complaint)

## 💡 Use cases

- ร้านอาหาร / คาเฟ่
- Skincare / beauty brand
- แฟชั่น / เสื้อผ้า
- คอร์สออนไลน์ / edutainment
- ขนม / ของฝาก / SME
- Pet product

## 🎓 Tips

- บอก **brand voice** ให้ชัด (fun / warm / luxury) → caption ตรงจริต
- Pair กับ `/copywriter-pro` เพื่อต่อยอดเป็น ad copy
- Pair กับ `/graphic-designer-pro` เพื่อสร้าง visual ตาม caption
