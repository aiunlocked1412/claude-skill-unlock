# Event Marketer

> AI Event Marketer — concept, funnel, pricing, sponsor, run-of-show, post-event analysis

## ⚡ ใช้ยังไง

```
/event-marketer
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Event concept + positioning (theme + audience + outcome)
- ✅ Event plan ครบ (run-of-show + budget + team RACI)
- ✅ Pricing tier strategy (Early-bird / Standard / VIP / Group / Sponsor)
- ✅ Promotion funnel 90 วัน (awareness → reg → attend → post)
- ✅ Sponsor pitch deck (10 slide + tier package)
- ✅ Email sequence 8 emails (save-the-date → recap)
- ✅ Run-of-show template (timing + owner + notes)
- ✅ Post-event analysis (ROI + NPS + content multiplier)

## 💡 Use cases

- Conference / Summit (200-2,000 คน)
- Workshop / Masterclass (20-100 คน)
- Product launch event (100-500 คน)
- Meetup / Community gathering (50-200 คน)
- Virtual summit / Webinar series
- Hybrid events

## 🎓 Tips

- บอก **goal + capacity + budget total** ครบ → pricing + tier แม่น
- Set 1 KGI ชัดก่อน (revenue / lead / brand)
- Speaker reveal ทยอยทุก 1-2 สัปดาห์ — sustain awareness
- Email reminder D-3, D-1, D-day → attendance boost 15-25%
- Post-event content ใน 7 วัน — repurpose สูงสุด
