# หมวด 4: Design & Creative (6 skills)

Skills สำหรับ graphic designer, photographer, video editor, architect

| Skill | คำสั่ง | ใช้เมื่อไหร่ |
|-------|--------|-------------|
| Graphic Designer | `/graphic-designer-pro` | poster, logo, infographic |
| Photographer | `/photographer-pro` | retouch, shot list, mood board |
| Video Editor | `/video-editor-pro` | storyboard, edit plan |
| Architect | `/architect-pro` | concept, render, mood board |
| Fashion Designer | `/fashion-designer-pro` | collection, pattern, lookbook |
| Product Designer | `/product-designer-pro` | packaging, mockup, spec |

## รูปแบบ output ทั่วไป

- Midjourney/DALL-E prompt พร้อม parameter
- Design brief (โจทย์, target, mood)
- Mood board (แนะนำ keyword ค้นภาพ reference)
- Spec sheet (ขนาด, สี, material)

## Workflow ตัวอย่าง

```
1. /product-designer-pro brief ผลิตภัณฑ์ใหม่
   ↓
2. /graphic-designer-pro logo + packaging prompt
   ↓
3. /photographer-pro shot list ถ่ายผลิตภัณฑ์
   ↓
4. /video-editor-pro วิดีโอโปรโมท 15 วิ
   ↓
5. /online-seller caption ขายใน marketplace
```
