# Jewelry Designer

> ออกแบบเครื่องประดับ — ring, necklace, earring + gem 4Cs + metal selection + render prompt

## ⚡ ใช้ยังไง

```
/jewelry-designer
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Concept + symbolism statement
- ✅ Gem spec (4Cs + treatment + cert lab)
- ✅ Metal selection (Au karat / Pt / Ag) + color
- ✅ Setting type recommendation (prong/bezel/halo/pavé)
- ✅ Sizing chart (ring/necklace/earring)
- ✅ AI render prompt 3 variants (hero, on-hand, macro)
- ✅ Cost estimate (gem + metal + labor)
- ✅ Care + maintenance guide

## 💡 Use cases

- Engagement / Wedding ring
- Anniversary gift
- Cocktail / Statement jewelry
- Signet ring (men)
- Pendant / Necklace
- Earring (stud / drop / chandelier)
- Custom set (matching collection)

## 🎓 Tips

- ระบุ **wearer + occasion + budget** ตั้งแต่ต้น
- บอก skin tone → match metal color
- ใช้ `/illustrator-pro` ก่อนเพื่อ sketch concept 2D
- ใช้ `/3d-modeler` ต่อยอด CAD model

## 🔗 Skills ที่ใช้คู่กัน

- `/illustrator-pro` — concept sketch
- `/3d-modeler` — 3D CAD for casting
- `/photographer-pro` — final product shoot
- `/graphic-designer-pro` — brand for jewelry brand
