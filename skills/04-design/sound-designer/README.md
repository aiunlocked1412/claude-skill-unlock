# Sound Designer

> AI นักออกแบบเสียง — SFX, Foley, Game Audio, Ambience, Audio Brand Identity (non-musical sound)

## ⚡ ใช้ยังไง

```
/sound-designer
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Game Audio Design (SFX matrix ตาม event, adaptive music, UI sound)
- ✅ Film/Video SFX Brief (foley, atmospheric, impact, transition)
- ✅ Audio Brand Identity (sonic logo, brand sound palette, earworm)
- ✅ Ambience / Soundscape Design (co-working space, retail, film)
- ✅ Foley Recording Session Brief (props, mic setup, room)
- ✅ DAW Workflow & Plugin Recommendations (Reaper, Pro Tools, FMOD, Wwise)
- ✅ Full Sound Design Spec สำหรับ project ทั้งหมด

## 💡 Use cases

- Game dev อยากได้ SFX list ครบสำหรับ indie game
- แบรนด์ต้องการ sonic logo สำหรับ commercial + app
- Film maker เขียน foley brief ก่อน recording session
- Interior designer วางแผน soundscape สำหรับ space
- Podcast producer อยากได้ sound identity ที่จำได้

## 📊 ตัวอย่างผลลัพธ์

Sound concept brief + SFX list พร้อม description + recording props list + implementation guide + reference playlist

## 🎓 Tips

- ระบุ **medium** (game/film/brand) + **mood** → ได้ concept ที่ตรง emotional target
- แชร์ **reference เสียงที่ชอบ** (ชื่อ game หรือ brand) → ได้ direction ชัดขึ้น
- บอก **platform** (stereo/5.1/spatial) → ได้ spec ที่ถูกต้อง
