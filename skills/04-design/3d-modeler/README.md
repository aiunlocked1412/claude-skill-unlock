# 3D Modeler

> Blender workflow + topology + UV + texturing + render brief สำหรับ game/product/character

## ⚡ ใช้ยังไง

```
/3d-modeler
```

## 🎯 ทำอะไรได้บ้าง

- ✅ 3D production plan (modeling → UV → texture → render)
- ✅ Topology spec (quads, edge flow, polycount budget)
- ✅ UV unwrap strategy (seam, texel density, UDIM)
- ✅ PBR texturing brief (BC/M/R/N/AO maps)
- ✅ Bake workflow (high-poly to low-poly)
- ✅ Render setup (lighting, camera, engine config)
- ✅ AI prompt for concept (Midjourney + Tripo3D / Meshy)

## 💡 Use cases

- Game asset (mobile / console / next-gen)
- Product viz (e-commerce, packaging)
- Character model (game / animation)
- Environment / Scene
- Hard surface (mech, vehicle)
- Stylized art (Disney/Arcane)
- Architectural viz

## 🎓 Tips

- ระบุ **target engine + polycount** ตั้งแต่ต้น
- ส่ง concept art / reference image ได้
- ใช้ `/illustrator-pro` ก่อนเพื่อ concept 2D
- ใช้ `/motion-designer` ต่อยอด 3D animation
