# UX Designer

> End-to-End UX Design — discovery, IA, user flow, wireframe, prototype, usability test ครบในที่เดียว

## ⚡ ใช้ยังไง

```
/ux-designer
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Persona + Jobs-To-Be-Done statement
- ✅ Information Architecture (sitemap + nav pattern)
- ✅ User flow (Mermaid diagram)
- ✅ Wireframe ASCII + Figma spec (8-pt grid, spacing tokens)
- ✅ Prototype interaction spec (animation, microinteraction)
- ✅ Usability test plan + script (5-task method)

## 💡 Use cases

- ออกแบบ feature ใหม่ในแอป (onboarding, checkout)
- Redesign legacy product
- IA สำหรับเว็บ e-commerce / SaaS
- Wireframe ก่อน hand-off นัก dev
- เขียน test plan ก่อน launch

## 🎓 Tips

- บอก **business goal** + **user goal** ชัดเจน → output ตรงเป้า
- ส่ง competitor reference (URL Figma community) ได้
- ใช้ `/graphic-designer-pro` ต่อยอด visual design
- ใช้ `/copywriter-pro` ต่อยอด UX writing
