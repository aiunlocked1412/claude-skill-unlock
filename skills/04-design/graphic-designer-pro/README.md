# Graphic Designer Pro

> AI Prompt + Design Brief Generator สำหรับงานกราฟิกทุกประเภท — โลโก้ โปสเตอร์ อินโฟกราฟิก โซเชียล

## ⚡ ใช้ยังไง

```
/graphic-designer-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Midjourney v6 / DALL-E 3 / Flux prompts (3-5 variants ต่องาน)
- ✅ Design brief ภาษาไทยครบชุด (brand, tone, reference)
- ✅ Color palette พร้อม HEX + rationale
- ✅ Typography pairing recommendation
- ✅ Mood board keywords (ใช้ค้น Pinterest ได้เลย)
- ✅ Composition & layout guide

## 💡 Use cases

- โลโก้ (wordmark / lettermark / combination)
- โปสเตอร์ / Key visual
- อินโฟกราฟิก
- Social media post (FB/IG/TikTok)
- Web hero / Cover
- Design system ครบชุด

## 🎓 Tips

- บอก **brand personality** 3 คำ → brief แม่นขึ้น
- ส่ง reference image/URL ได้ (Pinterest, Behance) → match สไตล์ได้ตรง
- ใช้ `/photographer-pro` ต่อยอดถ่าย product
- ใช้ `/copywriter-pro` ต่อยอด tagline
