# Packaging Designer

> AI ผู้เชี่ยวชาญออกแบบบรรจุภัณฑ์ — Box/Bottle/Label, Dieline, Material, Print Spec (CMYK/Pantone), Shelf Impact

## ⚡ ใช้ยังไง

```
/packaging-designer
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Dieline & Structure Design (box, pouch, label, sleeve, rigid box)
- ✅ Graphic Design Direction (visual concept, hierarchy, color system)
- ✅ Print Specification (CMYK, Pantone PMS, finish: foil/emboss/spot UV)
- ✅ Material Selection (paperboard, eco/FSC, plastic alternative)
- ✅ Shelf Impact Analysis (visibility, differentiation, consumer read)
- ✅ Unboxing Experience Design (premium, e-commerce, subscription)
- ✅ Full Packaging Brief สำหรับ product launch

## 💡 Use cases

- Brand owner อยากออกแบบกล่อง skincare ให้ดู premium
- Startup อาหารต้องการ label ที่มี อย. ครบ + ดูดีบนชั้นวาง
- Designer ต้องการ dieline spec + print file checklist ก่อนส่งโรงพิมพ์
- ทีม product อยาก unboxing experience สำหรับ subscription box
- เจ้าของแบรนด์ต้องการเปลี่ยน plastic packaging เป็น eco material

## 📊 ตัวอย่างผลลัพธ์

Packaging brief + dieline dimension + color spec (CMYK+Pantone) + material recommendation + print file checklist

## 🎓 Tips

- ระบุ **product type + channel** (retail/e-commerce/gift) → ได้ structure ที่เหมาะสม
- บอก **budget per unit** → ได้ material + finish ที่เป็นไปได้จริง
- ระบุ **market** (ไทย/export) → ได้ regulatory requirement ที่ถูกต้อง
