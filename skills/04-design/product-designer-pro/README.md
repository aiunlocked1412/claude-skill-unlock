# Product Designer Pro

> Product Brief + Packaging Design + Mockup Prompt + Design Spec สำหรับสินค้า + บรรจุภัณฑ์

## ⚡ ใช้ยังไง

```
/product-designer-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Product brief (functional + ergonomic)
- ✅ Packaging structure (pouch / box / bottle / tube)
- ✅ Design spec ละเอียด (dimensions, material, print method)
- ✅ Material recommendation (eco + conventional)
- ✅ Mockup prompt (hero, lifestyle, flat-lay, in-hand)
- ✅ Shelf appeal analysis (3-sec test, billboard test)
- ✅ Unboxing experience design (e-commerce)
- ✅ Regulatory checklist (barcode, อย., expiry)

## 💡 Use cases

- ถุงกาแฟ / tea bag / food packaging
- Cosmetic bottle / tube / jar
- Gift set / subscription box
- Electronics packaging
- E-commerce mailer
- Retail shelf product
- Rebrand existing product

## 🎓 Tips

- บอก **price tier + competitors** → ได้ positioning ที่แม่น
- ส่ง **product photo เดิม** (ถ้า rebrand) → analyze weakness ได้
- ใช้ `/graphic-designer-pro` สำหรับ logo + branding
- ใช้ `/photographer-pro` สำหรับ shoot final product
