## 📦 Packaging Structure Library

### Stand-Up Pouch (Food / Coffee)

```
STRUCTURE:
- Gusseted bottom (stands upright)
- Side-seal or 3-side seal
- Height × width: typical 180×120mm for 250g coffee
- Film gauge: 80-120 microns total
- Laminate: PET 12µ / AL 7µ / LDPE 80µ (barrier)

CLOSURES:
- Heat-seal top (one-time use)
- Zip seal (resealable) — +15% cost
- Slider zip — premium
- Degassing valve (coffee) — essential for fresh beans

PRINT:
- Flexo printing (cost-effective)
- Rotogravure (premium quality) — higher MOQ
- Digital print (small batch)

FINISH:
- Matte lamination (+) hides fingerprints
- Gloss lamination (+) vibrant colors
- Soft-touch coating (premium feel) +30% cost
```

### Folding Box (FBB / SBS)

```
STRUCTURE:
- Tuck top + tuck bottom (standard)
- Auto-bottom / crash-lock (quick assembly)
- Two-piece rigid box (luxury)
- Drawer box (premium tech products)

MATERIAL:
- FBB 300-350 gsm (folding box board) — standard
- SBS 280-320 gsm (solid bleached sulfate) — food-safe
- Recycled kraft 280-350 gsm — sustainable look

PRINT:
- Offset 4-color CMYK + 1-2 spot colors
- Varnish: matte / gloss / spot UV
- Foil: gold / silver / holographic
- Emboss / deboss (adds dimension)

DIELINE:
- Must include bleed (3mm)
- Safety zone for die-cut (5mm from trim)
- Glue tabs marked
```

### Rigid Box (Luxury Gift)

```
STRUCTURE:
- 2-piece: base + lid (shoulder neck)
- Magnetic close (premium experience)
- Book-style (opens like book)
- Drawer (slides out)

MATERIAL:
- Greyboard 2-3mm (strong base)
- Paper wrap 120-160 gsm (colored / printed)
- Ribbon insert (for drawer style)
- EVA foam insert (custom cut for product fit)

PRINT (on wrap paper):
- Offset print on wrap before wrapping
- Or plain colored paper + foil stamp / emboss
- Soft-touch lamination for premium tactile
```

### Tube Packaging (Cosmetic / Cream)

```
STRUCTURE:
- PE / PP tube 25-50mm diameter
- Wall thickness 0.3-0.5mm
- Lengths 80-150mm (varies)
- Capacity 20-100ml typical

CLOSURES:
- Flip-top cap
- Screw cap
- Pump head (lotion)
- Nozzle tip (precise application)

PRINT:
- Screen print (opaque, layered)
- Offset (full graphic)
- Hot stamp (foil accents)

ECO OPTIONS:
- Sugarcane PE (bio-based) +15% cost
- Refillable cartridge system (premium)
```

### Glass Bottle / Jar

```
TYPES:
- Amber glass (UV protection) — beer, essential oil
- Clear flint glass — luxury product visibility
- Frosted glass — premium look
- Boston round / Woozy / custom mold

CLOSURES:
- Screw top (plastic or metal)
- Cork (premium, for oils / spirits)
- Dropper (serum)
- Pump (foundation, lotion)

LABELING:
- Pressure-sensitive label (PSL) — most common
- Shrink sleeve (full wrap)
- Direct screen print (permanent)
- Embossed glass (luxury, high MOQ)
```

### Mailer / E-commerce Box

```
STRUCTURE:
- E-flute corrugated 3mm (standard)
- B-flute 2.5mm (compact)
- Kraft (brown) or bleached white

DESIGN:
- Outside: branded or minimal
- Inside: printed tissue / custom print interior
- Tape: branded kraft tape or custom printed

SIZE STANDARDS:
- Small: 200×150×50mm (apparel fold)
- Medium: 250×200×80mm
- Large: 350×250×100mm (multiple items)
```

## ♻️ Sustainable Alternatives

| Conventional | Sustainable Option | Trade-off |
|--------------|-------------------|-----------|
| Plastic pouch | Compostable PLA / PHA | +20-30% cost |
| Plastic bottle | Glass or aluminum | heavier, fragile |
| Plastic bubble wrap | Cornstarch air pillow / paper honeycomb | +15% |
| Virgin paperboard | Recycled paperboard / FSC | Comparable cost |
| Polybag | Compostable bag | +25% |
| Plastic tape | Kraft paper tape | Comparable |
| Foam insert | Molded pulp / recycled cardboard | -10% to +10% |
| Standard ink | Soy-based / vegetable ink | +5-10% |

**Certifications to mention:**
- FSC (Forest Stewardship Council) — paper
- PEFC — paper alternative
- BRC / ISO 22000 — food safety
- Compostable per ASTM D6400 / EN 13432
- Recycled content % (PCR — post-consumer recycled)

## 🎯 Shelf Appeal Test Framework

### 3-Second Test
- Look at packaging for 3 seconds → can you identify?
  - Brand name: yes/no
  - Product category: yes/no
  - Key benefit: yes/no
- Pass: 2/3 yes minimum

### Billboard Test
- Squint eyes looking at mockup
- Can you see hierarchy?
  1. Strongest element (logo or hero visual)
  2. Product name
  3. Key claim
- If all 3 not clear → redesign hierarchy

### Competitor Wall Test
- Place design among 10 competitors (mockup wall)
- Stand out? Gets picked up?
- If it blends in → add differentiator (color, shape, material)

### Thumbnail Test (e-commerce)
- Shrink mockup to 200×200px (listing size)
- Still readable / recognizable?
- If not → simplify primary visual

## ⚠️ Common Mistakes

❌ Designing without dieline (can't print)
❌ Forgetting bleed (3mm) and safety zone (5mm)
❌ Using RGB colors (must convert to CMYK for print)
❌ Forgetting to check reg. requirements
❌ Over-packaging (costly + anti-eco)
❌ Forgetting fill-line / functional zones
❌ Ignoring shelf orientation (top-down vs facing consumer)
❌ Not considering palletization (shipping efficiency)
