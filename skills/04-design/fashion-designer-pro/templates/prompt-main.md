## 👕 Silhouette Description Framework

สำหรับ piece แต่ละชิ้น ต้องตอบคำถามเหล่านี้:

### Top / Outerwear
- Fit: fitted / relaxed / oversized / cocoon?
- Shoulder: natural / dropped / exaggerated?
- Sleeve: fitted / puff / kimono / raglan?
- Length: crop / standard / longline / tunic?
- Hem: straight / curved / stepped?
- Closure: buttons / zip / pullover / wrap?
- Collar: crew / mock / spread / shawl / hood?

### Bottom
- Rise: low / mid / high / super high?
- Leg: skinny / slim / straight / wide / barrel / palazzo?
- Length: short / cropped / full / floor?
- Waist: elastic / button / drawstring / fitted with zip?
- Pockets: patch / slash / welt / none?

### Dress / Jumpsuit
- Silhouette: A-line / sheath / shift / fit-flare / bias?
- Bodice fit: fitted / relaxed / ruched?
- Skirt: mini / knee / midi / maxi?
- Sleeve: sleeveless / cap / short / long?
- Neckline: round / V / square / halter / cowl?

## 📋 Pattern Brief Template

```
PATTERN BRIEF

Style Code: ________
Style Name: ________
Category: ________
Size Range: ________ (e.g., XS-XL, 5 sizes)
Base Size: ________ (usually M)

REFERENCE GARMENT:
- Attach tech pack + reference photo
- Sample ref garment (if tearing down existing)

KEY MEASUREMENTS (Size M body):
- Chest: ___
- Waist: ___
- Hip: ___
- Length CB: ___
- Shoulder (width): ___
- Sleeve length: ___
- Bicep: ___
- Cuff: ___

EASE / FIT:
- Ease over body: +___" at chest
- Desired silhouette: fitted / relaxed / oversized

CONSTRUCTION:
- Seam type: plain / flat-fell / French / overlock
- Top stitch: yes/no, width, density
- Seam allowance: 1 cm / 1.5 cm
- Hem: 2.5 cm double fold / rolled / serge

FABRIC:
- Main: ___ (gsm, composition)
- Lining: ___ (if any)
- Interfacing: ___ (fusible / sew-in, weight)

TRIMS:
- Buttons: size, material, quantity
- Zipper: length, type, brand
- Thread: cotton / polyester, tex
- Labels: brand label, care label, size label

GRADE RULES:
- Standard ASIA grade (2cm chest between sizes)
- OR: half-size grade for premium

TOLERANCES:
- Critical POMs ±0.5 cm
- Non-critical ±1 cm

SAMPLE REQUIREMENTS:
- First pattern + pattern card
- Proto sample 1 (fit sample)
- Fit sample 2 (after revisions)
- Top sample (production approval)

NOTES:
- Special instructions
- Reference image attachments
```

## 🌈 Color Story Building

### Palette Structure (typical 5-7 colors)

```
Hero / Signature (40%)
├── Primary neutral — 25%
├── Secondary neutral — 15%
Colors (40%)
├── Core color 1 — 15%
├── Core color 2 — 15%
├── Accent color — 10%
Blacks (20%)
├── True black — 10%
├── Off-black / charcoal — 10%
```

### Seasonal Reference

| Season | Palette Tendency |
|--------|------------------|
| SS | Light, brighter colors, white, pastels, saturated brights |
| FW | Deeper, warmer tones, browns, burgundy, deep greens |
| Pre-fall | Transition — earth tones, olive, rust |
| Resort | Vacation feel — bright, tropical, poolside |

### HEX → Pantone Tip

Pattern maker ต้องใช้ Pantone TPX (Textile Paper) ไม่ใช่ TCX:
- `#3F4550` → **Pantone 19-3921 TPX (Turbulence)**
- Tool: https://picular.co/ + Pantone Color Finder app

## 💰 Pricing Structure

```
Landed cost (fabric + trim + CMT + overhead): 25-30% of retail
Wholesale (to stockist): 50% of retail
Retail markup: 2-3x wholesale

Example:
Landed cost: 800 baht
Wholesale: 1,600 baht
Retail: 3,200-4,800 baht
```

**Margin tiers:**
- Mass brand: 25% retail margin
- Mid-tier: 50-65% retail margin
- Luxury: 70-80% retail margin

## ⚠️ Common Mistakes

❌ Designing without thinking about pattern maker reality
❌ Choosing fabric factory can't source at MOQ
❌ Ignoring size grading (only designing size M)
❌ Price that doesn't cover cost (race to bottom)
❌ Copying silhouette 1:1 from other brand (IP issue)
❌ Ignoring care labels / wash test (customer returns)
❌ Not budgeting for sample iteration (3-4 samples typical)
