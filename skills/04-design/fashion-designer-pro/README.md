# Fashion Designer Pro

> Capsule Collection Plan + Mood Board + Pattern Brief + Fabric Spec สำหรับแบรนด์แฟชั่น

## ⚡ ใช้ยังไง

```
/fashion-designer-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Capsule collection plan (10-20 pieces) ครบชุด
- ✅ Concept narrative + season story
- ✅ Color story (5-7 colors + HEX + Pantone)
- ✅ Fabric & material brief (composition + weight + source)
- ✅ Silhouette description (detailed สำหรับ pattern maker)
- ✅ Pattern maker brief + grade rules
- ✅ Line sheet (code + cost + retail)
- ✅ Production plan (MOQ, lead time, QC)
- ✅ Lookbook direction

## 💡 Use cases

- Streetwear drop
- Ready-to-wear capsule
- Resortwear collection
- Casual essentials line
- Luxury small batch
- Custom / made-to-order

## 🎓 Tips

- บอก **target market** ให้ชัด (ไทย / JP / EU) → fit + pricing ต่างกัน
- ส่ง **Pinterest mood ref** → extract color + silhouette ได้ตรง
- ใช้ `/photographer-pro` วาง lookbook shoot
- ใช้ `/graphic-designer-pro` ทำ logo + hangtag
