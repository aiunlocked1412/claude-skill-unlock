# Illustrator Pro

> Vector illustration + character + icon set — AI prompt + vector spec ครบในที่เดียว

## ⚡ ใช้ยังไง

```
/illustrator-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Editorial illustration (article hero, blog)
- ✅ Character / Mascot design + expression library
- ✅ Icon set (24/48px grid, consistent stroke)
- ✅ Isometric illustration (SaaS hero, app store)
- ✅ AI prompt 3 variants (Midjourney / Recraft / DALL-E)
- ✅ Color palette + style guide
- ✅ Vector export spec (SVG optimization)

## 💡 Use cases

- Mascot/Character บริษัท
- Icon set แอป/เว็บ
- Empty state illustration
- Onboarding illustration
- Editorial blog header
- Pattern design (packaging, textile)
- Sticker pack / Emoji

## 📦 ไฟล์ใน skill นี้

```
illustrator-pro/
├── SKILL.md                  # ไฟล์หลัก
├── README.md                 # คุณกำลังอ่านอยู่
├── templates/
│   ├── prompt-main.md        # AI prompt formula + style ref
│   └── output-template.md    # โครงสร้าง illustration brief
└── examples/
    └── example-output.md     # Mascot บริษัทเทค "Pixel"
```

## 🎓 Tips

- ระบุ **style + use case** ชัด → output แม่น
- ส่ง mood ref จาก Dribbble / Behance ได้
- ใช้ `/graphic-designer-pro` ถ้าต้องการ logo/brand
- ใช้ `/motion-designer` ต่อยอด animate icon
