# Calligrapher

> AI ผู้เชี่ยวชาญอักษรประดิษฐ์ — Thai + Latin Calligraphy, Hand Lettering, Signage, Wedding Invitation, Brand Lettering

## ⚡ ใช้ยังไง

```
/calligrapher
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Wedding Invitation Lettering (Thai + English, layout, style guide)
- ✅ Signage & Display Lettering (ป้ายร้าน, menu board, wall art)
- ✅ Lettering Artwork Brief (concept, style, color, composition)
- ✅ Script Style Guide (Thai script, Copperplate, Brush, Italic, Gothic)
- ✅ Tool & Material Guide (ปากกา, หมึก, กระดาษ, digital tablet)
- ✅ Brand Lettering (wordmark, logo lettering, script concept)
- ✅ Full Lettering Project Spec สำหรับงาน commission

## 💡 Use cases

- คู่บ่าวสาวอยากได้ concept การ์ดแต่งงานที่มีเอกลักษณ์
- เจ้าของร้านกาแฟต้องการ brief ป้ายอักษรประดิษฐ์สำหรับช่าง
- Designer อยาก vectorize hand lettering เป็น wordmark
- Beginner อยากเรียนเขียนอักษรประดิษฐ์ไม่รู้จะเริ่มต้นอย่างไร
- แบรนด์ต้องการ lettering style ที่มีเอกลักษณ์ไทย

## 📊 ตัวอย่างผลลัพธ์

Style brief + layout description + tool list + text hierarchy + print spec + lettering resource recommendations

## 🎓 Tips

- ระบุ **ข้อความที่ต้องการ** ครบ → ได้ layout ที่ออกแบบมาให้ตรงกับ text จริง
- บอก **สไตล์ reference** (ชื่อแบรนด์, ภาพที่ชอบ) → ได้ direction ที่ชัดเจน
- ระบุ **medium** (print/digital/hand-painted) → ได้ spec ที่ถูกต้อง
