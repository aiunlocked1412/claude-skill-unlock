# Motion Designer

> Animation brief + AfterEffects spec + Lottie export — สำหรับ UI, logo reveal, explainer

## ⚡ ใช้ยังไง

```
/motion-designer
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Motion brief + storyboard (keyframe + timing)
- ✅ Easing curve + duration spec
- ✅ AfterEffects layer/effect/expression spec
- ✅ Lottie export setting + size optimization
- ✅ 12 Animation Principles check
- ✅ Sound design cue + SFX recommendation

## 💡 Use cases

- Logo reveal / Brand intro/outro
- UI microinteraction (button, loader, transition)
- Explainer video / Motion graphic
- Lottie animation (web + mobile app)
- Character / mascot animation
- Data visualization animation

## 🎓 Tips

- บอก **duration + format** ตั้งแต่ต้น → spec ถูกตั้งแต่แรก
- ส่ง reference Vimeo/Dribbble link ได้
- ใช้ `/graphic-designer-pro` ก่อน เพื่อให้มี logo/asset
- ใช้ `/video-editor-pro` ต่อยอด final cut
