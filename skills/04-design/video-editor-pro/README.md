# Video Editor Pro

> Storyboard + Edit Timeline + Sound Cue + Color Grading Plan สำหรับงานตัดวิดีโอทุกประเภท

## ⚡ ใช้ยังไง

```
/video-editor-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Storyboard frame-by-frame (shot + duration ชัดเจน)
- ✅ Edit timeline พร้อม pacing plan
- ✅ B-roll list (15-30 clips per work)
- ✅ Music + SFX cue sheet (timestamp ตรง)
- ✅ Color grading recipe (DaVinci / Premiere values)
- ✅ Export settings (YouTube / IG / TikTok spec)
- ✅ Transition + speed ramp plan

## 💡 Use cases

- YouTube video (long form)
- Reels / Shorts / TikTok (short form)
- Wedding highlight
- Commercial / Advertisement
- Event recap
- BTS / behind-the-scenes
- Film / short film

## 🎓 Tips

- ส่ง URL reference video → เทียบ pacing ได้ตรง
- บอก **platform** ให้ชัด (YouTube vs IG) → aspect + length ต่างกัน
- ใช้ `/photographer-pro` ต่อยอดถ่าย behind-the-scene
- ใช้ `/my-marketing` ต่อยอดโพสโปรโมท
