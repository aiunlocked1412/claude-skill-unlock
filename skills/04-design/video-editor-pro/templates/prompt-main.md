## 🎬 Pacing Guide

| Style | Average Shot Length | BPM Range |
|-------|---------------------|-----------|
| Cinematic / documentary | 3-6 sec | 70-90 |
| YouTube long-form | 2-4 sec | 85-110 |
| Vlog | 1.5-3 sec | 95-120 |
| Commercial / Ad | 1-2.5 sec | 110-130 |
| Music video | 0.5-1.5 sec | 100-140 |
| Reels / TikTok montage | 0.3-1 sec | 120-160 |
| Action / Sports | 0.4-1.2 sec | 130-160 |

**Rule of thumb:** ทุก 30-60 วินาที ต้องมี "breathing moment" (hold 3-5 sec) เพื่อไม่ให้ viewer เหนื่อย

## 🎞️ Edit Formula — Wedding Highlight (3 min)

```
0:00-0:15   HOOK — Establishing shots + slow build
            - Wide venue aerial (3s)
            - Close-up details: rings, flowers, invitation (5s)
            - Groom preparing (3s)
            - Bride preparing (4s)
            Music: ambient piano build

0:15-0:45   PREP MONTAGE — bride/groom getting ready
            - Intercut 8-12 clips, ASL 2-3 sec
            - Intimate details + emotion
            Music: build continues, add strings

0:45-1:30   CEREMONY — the "moment"
            - First look (slow motion, 5s hold)
            - Walking down aisle
            - Vows (audio cue: real vow clip)
            - First kiss
            - Audio real vows at -10 dB, music at -20 dB
            Music: climax hits at first kiss

1:30-2:15   CELEBRATION — reception + dance
            - Faster cuts 1-2 sec
            - First dance slow-mo (3s hold)
            - Speeches (1 funny + 1 emotional line)
            - Cake cut
            Music: uplifting, tempo up

2:15-2:45   GOLDEN HOUR COUPLE PORTRAIT
            - Slow cinematic shots
            - Couple alone, backlit
            - 4-5 clips, 3-4 sec each
            Music: dreamy, tempo down

2:45-3:00   OUTRO — fade to title card
            - Names + date
            - Hold on final hug
            Music: resolve, fade out
```

## 🎞️ Edit Formula — Reels / TikTok Ad (30 sec)

```
0:00-0:03   HOOK (must-watch first 3 sec)
            - Text overlay: question or bold claim
            - Visual shock: before/after OR unusual angle
            - NO intro, NO logo yet

0:03-0:15   PROBLEM / STORY
            - 4-6 quick cuts showing the "before"
            - Relatable problem
            Music: tension building

0:15-0:25   SOLUTION / PRODUCT REVEAL
            - Product hero shot (2 sec hold)
            - Benefit demonstration
            - Text overlay: key benefits
            Music: resolution, drop

0:25-0:30   CTA
            - "ลองเลย / ซื้อเลย / follow"
            - Logo + handle appears
            - End frame holds 2 sec
```

**Vertical safe zones:**
- Top 250px: IG username/caption area
- Bottom 400px: IG UI (heart, comment, share, music)
- วาง text overlay ใน middle 60% ของหน้าจอ

## 🎞️ Edit Formula — YouTube Long-Form (8 min)

```
0:00-0:10   HOOK (must-hook or lose them)
            - No "สวัสดีครับ"
            - Shocking claim + visual
            - Promise what's inside

0:10-0:30   PROMISE + LOGO/INTRO
            - "ในคลิปนี้คุณจะได้..."
            - Channel logo animation (3 sec max)

0:30-6:30   BODY (3-5 chapters)
            - Each chapter = mini-hook → content → cliffhanger
            - Pattern interrupt every 60-90 sec:
              - Cut-in meme, graphic, zoom, SFX
              - Speed ramp, freeze frame
            - B-roll every spoken sentence

6:30-7:30   PAYOFF / CONCLUSION
            - Deliver what was promised
            - 3 key takeaways

7:30-8:00   CTA + END SCREEN
            - Like/comment/subscribe
            - Tease next video (thumbnail popup)
```

## 🎵 Music Cue Sheet Format

```
00:00 — Piano solo intro (mood: contemplative)
         Song: "First Light" by Tony Anderson (Musicbed)
         In: full volume
00:15 — Add strings layer (mood: hopeful)
00:45 — Full orchestra hits (emotional peak for first look)
01:30 — Tempo shift — upbeat acoustic
         Song: "Summer Days" by Josh Leake (Artlist)
         Crossfade 2 sec
02:15 — Return to piano (quiet/intimate)
02:45 — Fade out over 3 sec
```

**Always notate:**
- Exact timestamp
- Song title + artist + source
- Mood description
- Volume automation points
- Crossfade duration

## 🎨 Color Grading Recipes

### Recipe 1: Cinematic Teal & Orange

```
PRIMARY:
- Exposure: match across shots
- Contrast: +10
- White balance: neutral (no tint shift)

COLOR WHEELS:
Shadows: Hue 200 (teal), Sat 20, Lift -0.05
Midtones: neutral
Highlights: Hue 30 (orange), Sat 15, Gain +0.05

HSL:
- Skin tones (orange): desaturate surrounding -15, boost skin sat +5
- Green: shift toward teal (-10 hue)
- Blue: shift toward cyan (-5 hue), sat -10

SECONDARY:
- Power window vignette -0.3 edge
- Film grain: 5-8% medium grain

LUT:
- Apply Kodak 2393 at 30-40% opacity
OR
- Apply "Cinematic Teal Orange" at 50%
```

### Recipe 2: Natural Cinematic (True-to-life)

```
PRIMARY:
- Match exposure + WB
- Slight contrast boost +5

COLOR WHEELS:
- Shadows: -0.02 lift (slightly darker black)
- Midtones: neutral
- Highlights: +0.03 gain

HSL:
- Minimal — preserve realism
- Skin: slight sat boost +3

LUT:
- Rec.709 with mild S-curve
OR
- Arri Alexa 709 at 50%

FILM GRAIN: 3% fine
```

### Recipe 3: Film Emulation (Kodak Portra vibe)

```
PRIMARY:
- Exposure: +0.3 (slightly overexposed for film feel)
- Contrast: -15 (lower contrast)
- Lift: +0.05 (faded blacks)

COLOR WHEELS:
Shadows: Hue 220 (blue), Sat 15
Midtones: Hue 30 (warm), Sat 8
Highlights: Hue 45 (golden), Sat 12

HSL:
- Red: hue +5, lum +10
- Orange (skin): hue -5 (less orange), lum +10
- Yellow: sat -20 (muted yellows)
- Green: sat -30 (olive greens)

SECONDARY:
- Film halation: red glow on highlights (plugin FilmBox or manual)
- Heavy grain: 10-15% coarse

LUT:
- Apply Kodak Portra 400 at 60-70%
```

### Recipe 4: Reels / Social Pop

```
PRIMARY:
- Exposure: +0.2
- Contrast: +15 (more punch)
- Saturation: +10

COLOR WHEELS:
- Shadows: neutral
- Midtones: slight warm +5
- Highlights: punchy

HSL:
- All colors slightly boosted saturation
- Keep vibrant for mobile viewing

SHARPENING:
- Punch up detail (mobile compresses)
- Add subtle glow on highlights

NO GRAIN (mobile compression will eat it)
```

## 📤 Export Settings by Platform

### YouTube (4K 16:9)
```
Codec: H.264 or H.265
Resolution: 3840x2160
Frame rate: 24/30/60 (match source)
Bitrate: 50-80 Mbps
Audio: AAC 320kbps stereo, LUFS -14
Container: .mp4
```

### Instagram Reels / TikTok (9:16)
```
Codec: H.264
Resolution: 1080x1920
Frame rate: 30 fps
Bitrate: 8-12 Mbps
Audio: AAC 192kbps, LUFS -16
Duration: 15-90 sec
Container: .mp4
```

### Instagram Feed (1:1 or 4:5)
```
Resolution: 1080x1080 or 1080x1350
Duration: max 90 sec for Reels, 60 sec for feed video
Bitrate: 8 Mbps
```

### Facebook Ad
```
Resolution: 1080x1080 (feed) or 1080x1920 (stories)
Bitrate: 10 Mbps
Audio: LUFS -18 (Facebook auto-normalizes)
Caption: burn-in subtitle (85% watch muted)
```

### Commercial / TV
```
Codec: ProRes 422 HQ or DNxHD
Resolution: 1920x1080 or 3840x2160
Frame rate: 25 fps (broadcast)
Audio: -10 LUFS, dual mono, safe zones
```

## ⚠️ Common Edit Mistakes

❌ Using copyrighted music (YouTube ID claim)
❌ Hard cuts with no J-cut / L-cut (feels amateur)
❌ Same shot length ทั้งคลิป (no rhythm)
❌ Color graded inconsistently between shots
❌ Audio too loud (distortion) or too quiet
❌ No captions on mobile-first content
❌ Forgetting safe zones for vertical
❌ Export bitrate too low (pixelation)
❌ Using "trending sound" illegally (commercial use needs license)
