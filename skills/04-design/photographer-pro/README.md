# Photographer Pro

> Shot List + Mood Board + Lightroom Preset + Retouching Plan สำหรับงานถ่ายภาพทุกประเภท

## ⚡ ใช้ยังไง

```
/photographer-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Shot list ละเอียด (10-100 shots พร้อม lens, lighting, time)
- ✅ Mood board keywords + reference photographer
- ✅ Lighting diagram + power ratio
- ✅ Wardrobe / styling guide
- ✅ Lightroom preset recipe (HSL values + tone curve)
- ✅ Retouching checklist (skin, teeth, dodge & burn)
- ✅ Backup plan (ฝน / model late)

## 💡 Use cases

- Portrait (studio / outdoor)
- Product photography (e-commerce / editorial)
- Wedding (half-day / full-day)
- Family session
- Event / concert
- Landscape / Travel

## 🎓 Tips

- บอก **golden hour time** ของเมือง → พี่ Claude คำนวณตารางให้
- ส่ง mood image URL ได้ → extract HSL ออกมาเป็น preset
- ใช้ `/graphic-designer-pro` ออกแบบ branding ของ portfolio
- ใช้ `/video-editor-pro` ต่อยอดวิดีโอ behind-the-scene
