# Tattoo Designer

> Tattoo concept + style + placement + AI prompt + after-care guideline

## ⚡ ใช้ยังไง

```
/tattoo-designer
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Concept + symbolism statement
- ✅ Style recommendation (8 major styles)
- ✅ Sizing + placement strategy (pain scale + aging effect)
- ✅ Color/ink spec (longevity + allergy consideration)
- ✅ AI image prompt 3 variants (Midjourney + tattoo-specific)
- ✅ Stencil spec for artist (size, resolution, outline weight)
- ✅ Pre-tattoo + After-care guideline
- ✅ Warning signs + when to seek help

## 💡 Use cases

- First tattoo (consultation + safe choice)
- Memorial / Tribute design
- Cover-up old tattoo
- Sleeve / Back piece large project
- Cultural / Spiritual concept (with respect)
- Watercolor / Artistic statement
- Corporate-safe placement

## 🎓 Tips

- ระบุ **meaning + placement + size** ตั้งแต่ต้น
- ส่งรูป body ตำแหน่ง + concept reference
- ปรึกษา artist จริงก่อน sit (consultation 30 min)
- ใช้ `/illustrator-pro` ก่อนถ้าต้องการ stencil 2D

## 🔗 Skills ที่ใช้คู่กัน

- `/illustrator-pro` — vector stencil refinement
- `/graphic-designer-pro` — typography for script tattoo
- `/photographer-pro` — final tattoo documentation

## ⚠️ Disclaimer

Skill นี้สำหรับ **concept design + research only** — ต้องไป tattoo artist ที่ได้รับ certification เสมอ ห้าม DIY tattoo ที่บ้าน
