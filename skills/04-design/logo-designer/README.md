# Logo Designer

> AI ผู้เชี่ยวชาญออกแบบ Logo และ Brand Identity — Mark + Wordmark, Usage Guide, Rebrand, CMYK/RGB/Pantone Spec, File Format

## ⚡ ใช้ยังไง

```
/logo-designer
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Logo Concept Brief (brand personality → visual direction → concept)
- ✅ Logo System Design (mark, wordmark, combination, sub-brand, icon)
- ✅ Brand Identity Guide (color Pantone/CMYK/RGB/HEX, typography, usage)
- ✅ Rebrand Strategy (วิเคราะห์ logo เก่า + direction ใหม่)
- ✅ File Format Spec (AI, EPS, SVG, PNG, ICO พร้อม naming convention)
- ✅ Logo Review & Feedback (ตรวจงานที่มีอยู่แล้ว)
- ✅ Full Brand Identity Package ตั้งแต่ต้น

## 💡 Use cases

- Startup ต้องการ logo brief ก่อน briefing designer
- แบรนด์ SME อยากทำ brand guideline จากศูนย์
- Designer ต้องการ checklist file deliverables ให้ client
- แบรนด์เก่าอยาก rebrand โดยยังคง brand equity
- ทีม marketing ต้องการ color spec ที่ถูกต้องสำหรับ print + digital

## 📊 ตัวอย่างผลลัพธ์

Brand brief + Logo concept direction (3 options) + Color spec (Pantone/CMYK/RGB/HEX) + File package structure + Usage do/don't

## 🎓 Tips

- ระบุ **brand personality** (3-5 คำ) + **competitor logos** → ได้ direction ที่ชัดเจน
- บอก **usage context** (digital/print/signage) → ได้ file format spec ที่ครบ
- แชร์ **logo ที่ชอบ** (ไม่ต้องเป็น industry เดียวกัน) → ได้ visual language ที่ตรงใจ
