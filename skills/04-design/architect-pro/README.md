# Architect Pro

> Concept Design + Mood Board + Render Prompt + Material Spec สำหรับงานสถาปัตยกรรม

## ⚡ ใช้ยังไง

```
/architect-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Site analysis (sun path, wind, view, privacy)
- ✅ Concept narrative + spatial idea
- ✅ Functional zoning + layout description
- ✅ Material palette (5-7 specs พร้อม HEX + brand)
- ✅ Lighting design (ambient / task / accent / feature)
- ✅ Midjourney render prompts (exterior + interior + detail + night)
- ✅ Mood board keywords + reference architects

## 💡 Use cases

- บ้านเดี่ยว / Townhouse / Condo interior
- คาเฟ่ / ร้านอาหาร / โรงแรม
- ออฟฟิศ / Retail shop
- Landscape / outdoor
- Renovation / Reimagine existing

## 🎓 Tips

- บอก **location + ทิศทางที่ดิน** → site analysis แม่น
- ส่ง **Pinterest URL** → เทียบสไตล์ตรง
- ใช้ `/graphic-designer-pro` ออกแบบ branding สำหรับ project
- ใช้ `/photographer-pro` วางแผนถ่ายภาพหลังก่อสร้างเสร็จ

## ⚠️ ข้อควรทราบ

- Skill นี้ช่วย **concept + visualization** — ไม่ใช่แบบก่อสร้างจริง
- ต้องปรึกษาสถาปนิกจริง + วิศวกรโครงสร้าง + ขออนุญาตก่อสร้าง
- Material list เป็นแนวทาง — ราคา/availability ต้องเช็คตลาด
