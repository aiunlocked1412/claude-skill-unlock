## Diagnostic Output

หลังจากได้ข้อมูล วิเคราะห์ตาม 8 ข้อนี้:

1. **อัตราการออม** = (รายรับ - รายจ่าย) / รายรับ ≥ 20%?
2. **DTI** (Debt-to-Income) = ผ่อนหนี้ทั้งหมด / รายได้ ≤ 40%?
3. **Emergency Fund** ครอบคลุมกี่เดือน?
4. **Net Worth** = ทรัพย์สิน - หนี้สิน
5. **ดอกเบี้ยจ่าย** เทียบกับรายได้ ≤ 10%?
6. **มีประกันสุขภาพ** หรือไม่?
7. **มีลงทุนระยะยาว** (กอง/หุ้น) หรือไม่?
8. **Use SSF/RMF/ThaiESG** เต็ม limit หรือไม่ (ถ้ามีรายได้สูง)?

## Output Plan

จากนั้นเสนอแผน 3 ระดับ:
- **Conservative:** เก็บก่อน, จ่ายหนี้ก่อน, ลงทุนน้อย
- **Moderate:** บาลานซ์ทุกอย่าง
- **Aggressive:** ลงทุนเยอะ, ปลดหนี้ดอกต่ำช้าๆ
