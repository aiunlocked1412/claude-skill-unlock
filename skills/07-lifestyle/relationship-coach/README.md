# Relationship Coach — โค้ชความสัมพันธ์

ช่วยแก้ปัญหาความสัมพันธ์ทุกรูปแบบ — คู่รัก, dating, ครอบครัว, เพื่อน, เพื่อนร่วมงาน

## Features
- Couple problems (แต่งงาน/อยู่ด้วยกัน)
- Dating (เริ่มต้น, "talking" stage, DTR)
- ปัญหาครอบครัว (พ่อแม่, พี่น้อง, ญาติ, แม่สามี)
- เพื่อน / เพื่อนร่วมงาน
- Heartbreak / Healing 90-day plan
- Communication frameworks (NVC, I-statements)
- 5 Love Languages assessment + apply
- Gottman 4 Horsemen detection
- Conflict resolution roleplay
- Boundary setting

## Frameworks
- **NVC** (Nonviolent Communication) — Marshall Rosenberg 4 ขั้น
- **I-Statements** — สูตรแก้ blame language
- **5 Love Languages** — Gary Chapman
- **Gottman Method** — 4 Horsemen + Repair Attempts
- **Soft Start-up + Time-out** — conflict de-escalation

## บริบทไทย
- ครอบครัวขยาย (พ่อแม่ ญาติ)
- ความเกรงใจ vs boundary
- "ลูกของแม่" ตลอดชีวิต
- หน้าที่ลูกกตัญญู vs ชีวิตคู่
- script ภาษาไทยที่ยังเคารพ + ชัดเจน

## ⚠️ Disclaimer
**ผมเป็นโค้ช ไม่ใช่นักจิตวิทยา/นักบำบัด**
- ความรุนแรง → 1300 / 191
- ทำร้ายตัวเอง → 1323
- Trauma / Depression รุนแรง → จิตแพทย์ / นักจิตวิทยาคลินิก
- คู่ที่ปัญหาเรื้อรัง → couple therapist (Ooca, MUI Counseling)

## ตัวอย่างใช้งาน
```
/relationship-coach
/relationship-coach แฟนทะเลาะเรื่องเงิน
/relationship-coach แม่สามีก้าวก่าย
/relationship-coach อกหักครบ 1 เดือน
/relationship-coach NVC สำหรับเรื่องงานบ้าน
```

## Output
ไฟล์ `.md` ชื่อ `relationship-<topic>-YYYY-MM-DD.md` ประกอบด้วย:
- Acknowledge + Validate
- Situation Summary (objective)
- Diagnosis (Gottman, Love Languages, Communication Pattern)
- NVC Script (พูดแบบนี้ + อย่าพูดแบบนี้)
- Action Plan 90 วัน (4 phases)
- Roleplay (ฝึกก่อนคุยจริง)
- Red Flags + Refer Path
- Resources + Hotlines

## Workflow ตัวอย่าง
```
1. /relationship-coach แฟนทะเลาะเรื่องเงิน
   ↓ (สัปดาห์ที่ 1)
2. /relationship-coach roleplay ก่อนคุย
   ↓ (สัปดาห์ที่ 2)
3. /relationship-coach review หลังคุยครั้งแรก
   ↓ (เดือนที่ 1-3)
4. /relationship-coach maintain — monthly check-in
```

## เหมาะสำหรับ
- คู่รักทะเลาะบ่อย (ไม่ใช่ abusive)
- Dating ไม่ commit / "talking" stage stuck
- ครอบครัวก้าวก่าย (พ่อแม่ไทย, แม่สามี)
- เพื่อน/เพื่อนร่วมงาน toxic
- หลังเลิก / heartbreak recovery
- ฝึกพูดให้เป็น (introvert ที่อยาก improve)
