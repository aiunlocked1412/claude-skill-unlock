## NVC Worksheet (ฝึกพูด)

ฝึก 4 ขั้น สำหรับสถานการณ์ที่จะคุย:

1. **Observation (สังเกต fact):**
   "เมื่อ ___ (ระบุเหตุการณ์ ไม่ใช่ตัดสิน)"
   _____

2. **Feeling (ความรู้สึก):**
   "ฉันรู้สึก ___ "
   _____
   (ใช้คำ feeling: เหงา/เศร้า/กังวล/หงุดหงิด/ผิดหวัง — ไม่ใช่ "you make me angry")

3. **Need (ความต้องการ):**
   "เพราะฉัน need ___ "
   _____
   (ใช้ universal needs: connection/respect/safety/autonomy/appreciation)

4. **Request (ขอเฉพาะเจาะจง):**
   "อยากขอให้ ___ ได้ไหม?"
   _____
   (specific, doable, present-tense — ไม่ใช่ "เปลี่ยนเป็นคนดีกว่านี้")

## Love Languages Mini-Quiz

ให้คะแนน 1-5 ว่าอันไหนทำให้คุณรู้สึก loved ที่สุด:

- [ ] เขาบอกชื่นชม / ขอบคุณ / I love you (Words)
- [ ] เขาทำกับข้าว / ขับรถส่ง / ช่วยงานบ้าน (Acts)
- [ ] เขาให้ของขวัญที่คิดถึง (Gifts)
- [ ] เขาให้เวลาคุณภาพ ไม่มีมือถือ (Time)
- [ ] เขากอด / จับมือ / นวด (Touch)

→ **Top 1-2 = Primary Love Language ของคุณ**
→ ทำซ้ำสำหรับคู่ → จะเห็น mismatch

## Conflict De-escalation Script

**Step 1 — Time-out (ถ้า heated):**
> "เรื่องนี้สำคัญสำหรับเราทั้งคู่ ฉันอยากคุยให้ดี ตอนนี้รู้สึกยังเหนื่อยอยู่ — ขอเวลา 20 นาทีไปสงบใจ แล้วกลับมาคุยกันได้ไหม?"

**Step 2 — Soft Start-up:**
> "ขอบคุณที่ยอมคุย ฉันรักคุณนะ และอยากให้เราเข้าใจกันมากขึ้น
> ฉันมีเรื่องที่อยากเล่า ขอใช้สูตร I-statement หน่อยนะ..."

**Step 3 — Validate ก่อน defend:**
> "ฉันได้ยินที่คุณบอกว่า ___
> และฉันเข้าใจว่าทำไมคุณถึงรู้สึกแบบนั้น"

**Step 4 — Find common ground:**
> "เราทั้งคู่อยากให้ relationship นี้ดีขึ้น ใช่ไหม?
> ลองหาทางที่เราทั้งคู่รับได้ดีกว่ากัน"

**Step 5 — Repair attempt:**
> "ขอโทษที่พูดแรงไป
> ขอแก้ตัวด้วย ___
> Commit ว่า ___ "

## Red Flags Checklist (คนที่ผู้ใช้กำลังคบ)

- [ ] ทำร้ายร่างกาย แม้ครั้งเดียว
- [ ] ทำร้ายวาจา (ด่า เหยียด) ประจำ
- [ ] Control เงิน / เวลา / เพื่อนของคุณ
- [ ] Isolate คุณจากครอบครัว/เพื่อน
- [ ] โกหกประจำ / cheat ซ้ำ
- [ ] Substance abuse + ปฏิเสธรักษา
- [ ] Threaten suicide เมื่อคุณจะเลิก (manipulation)
- [ ] Gaslight (ทำให้คุณสงสัย sanity ตัวเอง)

→ **ถ้าเช็ค ≥ 2 ข้อ → strongly suggest professional support + safety plan**
