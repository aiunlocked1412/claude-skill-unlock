# Home Manager

> AI ผู้ช่วยจัดการบ้าน — KonMari decluttering, minimalism, household schedule, maintenance calendar, inventory

## ⚡ ใช้ยังไง

```
/home-manager
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Decluttering Plan ตาม KonMari method ทีละ category
- ✅ Household Schedule รายวัน/สัปดาห์/เดือน ที่ทำได้จริง
- ✅ Maintenance Calendar แยกตามฤดูกาลไทย (ร้อน/ฝน/หนาว)
- ✅ Inventory System สำหรับของใช้, อาหาร, ยา
- ✅ Minimalism Reset ลดของ 50%+ พร้อมแผนเป็นขั้นตอน
- ✅ Cleaning Routine daily habits + deep clean schedule

## 💡 Use cases

- คอนโดเล็กที่รู้สึกว่าของล้นไม่มีที่เก็บ
- ครอบครัวที่อยากมีระบบงานบ้านที่ทุกคนมีส่วนร่วม
- คนที่เคยจัดแล้วกลับมารกภายใน 2 สัปดาห์
- บ้านใหม่ที่ต้องการวางระบบตั้งแต่แรก
- อยากทำ annual declutter + maintenance reset

## 🎓 Tips

- บอก **ขนาดบ้าน + จำนวนสมาชิก** ก่อน → ได้ plan ที่ realistic
- บอก **ปัญหาหลัก** ชัดเจน (เช่น: "ตู้เสื้อผ้าล้น" หรือ "ห้องครัวรก") → ได้ quick wins ทันที
- ใช้คู่กับ `/gardening-pro` → จัดระเบียบพื้นที่สวนและระเบียงด้วย

## 🔗 Skills ที่ใช้คู่กัน

- `/gardening-pro` — จัดพื้นที่สวน/ระเบียง
- `/pet-care` — จัดบ้านให้ปลอดภัยสำหรับสัตว์เลี้ยง
- `/car-advisor` — maintenance schedule สำหรับรถด้วย
