# หมวด 7: Lifestyle & Personal (7 skills)

Skills สำหรับชีวิตส่วนตัว วางแผนการเงิน หางาน ครอบครัว ท่องเที่ยว

| Skill | คำสั่ง | ใช้เมื่อไหร่ |
|-------|--------|-------------|
| Personal Finance | `/personal-finance` | งบส่วนตัว, หนี้, ลงทุน, เกษียณ |
| Resume Builder | `/resume-builder` | CV + cover letter |
| Interview Coach | `/interview-coach` | เตรียมสัมภาษณ์งาน |
| Relationship Coach | `/relationship-coach` | คู่รัก ครอบครัว เพื่อน |
| Parenting Pro | `/parenting-pro` | เลี้ยงลูก พัฒนาการเด็ก |
| Wedding Planner | `/wedding-planner` | วางแผนงานแต่ง |
| Travel Planner | `/travel-planner` | itinerary, booking, trip |

## รูปแบบ output ทั่วไป

- Markdown checklist + timeline
- Budget table + spreadsheet spec
- Step-by-step action plan
- Conversation script / template

## Workflow ตัวอย่าง

```
1. /personal-finance วางแผนเก็บเงินล้านแรก
   ↓
2. /resume-builder ปรับ CV หางานใหม่ที่เงินดีกว่า
   ↓
3. /interview-coach เตรียม behavioral Q&A
   ↓
4. /travel-planner ทริปฉลองเก็บเงินครบ
```
