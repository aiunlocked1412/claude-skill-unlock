## STAR Answer Template

```
**S — Situation (15-20 วิ):**
"ตอนที่ผมอยู่ที่ [บริษัท] ปี [year], ทีมเรากำลัง [context]..."

**T — Task (10-15 วิ):**
"หน้าที่ของผมคือ [specific responsibility]..."

**A — Action (40-60 วิ — สำคัญสุด):**
"ผมเลย:
1. [action 1] เพราะ [reasoning]
2. [action 2]
3. [action 3]
มี challenge ตรง [obstacle], ผมแก้โดย [how]..."

**R — Result (15-25 วิ):**
"ผลคือ [quantified result]
- เพิ่ม X% / ลด Y% / ประหยัด Z บาท
- Lesson learned: [insight ที่ได้]"
```

## Power Phrases

- "ในประสบการณ์ของผม..." (sounds senior)
- "ผมเรียนรู้ว่า..." (humility)
- "ถ้าให้ทำใหม่ ผมจะ..." (growth mindset)
- "Data ที่ผมเห็นบอกว่า..." (analytical)
- "Hypothesis ของผมคือ..." (structured thinking)
