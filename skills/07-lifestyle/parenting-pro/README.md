# Parenting Pro — โค้ชเลี้ยงลูกตามพัฒนาการ

ช่วยพ่อแม่เลี้ยงลูกตามช่วงพัฒนาการ — ใช้หลักฐานจาก WHO, AAP, CDC + บริบทครอบครัวไทย

## Features
- 4 ช่วงอายุครอบคลุม:
  - **0-3 ปี** — Sleep, food, attachment, milestones
  - **4-6 ปี** — Discipline, social, เตรียมเข้า ป.1
  - **7-12 ปี** — การเรียน, เพื่อน, self-esteem
  - **13-18 ปี** — Autonomy, identity, mental health
- Screen Time Guidelines (ตาม AAP)
- Positive Discipline (แทนการตี/ดุ)
- Emotional Regulation Coaching
- ปัญหาโรงเรียน (bullying, การเรียน)
- จัดการปู่ย่าตายาย (ก้าวก่าย)

## Frameworks
- **Erikson's Stages** — Trust → Autonomy → Initiative → Industry → Identity
- **Attachment Theory** — Secure / Anxious / Avoidant
- **Connect Before Correct** — Dan Siegel
- **4S** — Safe, Seen, Soothed, Secure
- **Whole-Brain Parenting** — Siegel & Bryson

## บริบทไทย
- ครอบครัวขยาย (ปู่ย่า ตายาย)
- โรงเรียนไทย + ระบบสอบ entrance
- ค่าเลี้ยงดูเด็กในไทย
- ค่านิยม "ลูกกตัญญู"
- รักลูกแบบไทย vs Western

- สุขภาพจิตวัยรุ่น → จิตแพทย์เด็กและวัยรุ่น
- ฉุกเฉิน → 1669

## ตัวอย่างใช้งาน
```
/parenting-pro
/parenting-pro ลูก 2 ขวบ tantrum
/parenting-pro ลูก 9 ขวบ คะแนนตก
/parenting-pro ลูกวัยรุ่น 15 ไม่คุย
/parenting-pro screen time
/parenting-pro positive discipline
```

## Output
ไฟล์ `.md` ชื่อ `parenting-<topic>-<age>-YYYY-MM-DD.md` ประกอบด้วย:
- Child Profile + Developmental Stage
- What's normal vs what's concern
- Action Plan (Today / Week / Month / 3-Month)
- Scripts (พูดแบบนี้ vs อย่าพูด)
- Discipline Approach (logical consequences)
- Screen Time Plan (ตามอายุ)
- Red Flags (เมื่อต้องไปหาหมอ)
- Parent Self-Care Reminder
- Resources + Hotlines

## Workflow ตัวอย่าง
```
1. /parenting-pro ลูก 2 ขวบ tantrum
   ↓ (ลอง 4-6 สัปดาห์)
2. /parenting-pro review ผลการลอง — ปรับ plan
   ↓ (เด็กโตขึ้น 4 ขวบ)
3. /parenting-pro เตรียมเข้า ป.1
   ↓
4. /parenting-pro screen time ลูก 7 ขวบ
```

## Screen Time (AAP 2026)
- 0-18 เดือน: 0 (ยกเว้น video call)
- 18-24 เดือน: < 1 ชม. + co-viewing
- 2-5 ปี: ≤ 1 ชม./วัน
- 6-12 ปี: ≤ 2 ชม./วัน + device-free zone
- 13-18 ปี: ตกลงร่วมกัน + sleep priority

## Discipline Principles
- Connect Before Correct (Siegel)
- Time-In > Time-Out
- Logical Consequences ≠ Punishment
- Validate Feeling, Redirect Behavior
- Repair After Rupture (พ่อแม่ขอโทษได้)
