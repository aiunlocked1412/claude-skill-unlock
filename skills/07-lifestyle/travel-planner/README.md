# Travel Planner — ผู้ช่วยวางแผนทริปครบวงจร

วางแผนทริปทั้งในไทยและต่างประเทศ — itinerary, budget, visa, packing

## Features
- Destination Research (เปรียบเทียบ 3-5 ที่)
- Day-by-Day Itinerary (ละเอียดทุก activity)
- Budget Calculator (ตั๋ว, ที่พัก, อาหาร, ทัวร์)
- Flight Strategy (จองยังไงให้ถูก)
- Accommodation (โรงแรม, hostel, Airbnb, ryokan)
- Visa & Documents (สำหรับคนไทย)
- Packing List (ตามเดือน + activity)
- Food Guide (must-try local)
- Transport in-country
- Travel Apps + SIM/eSIM
- Travel Insurance + Safety
- ทริปในไทย (เกาะ, ภูเขา, อีสาน, เหนือ)

## Travel Style Supported
- Backpacker (ต่ำสุด งบประหยัด)
- Mid-range (สมดุล)
- Luxury (5 ดาว ทุกอย่าง)
- Family (เด็ก + ผู้สูงอายุ)
- Honeymoon (โรแมนติก)
- Solo (ปลอดภัย + เพื่อน)
- Business (efficient)

## Visa-Free สำหรับคนไทย (2026)
- ASEAN ทั้งหมด
- ญี่ปุ่น 15 วัน
- เกาหลี 90 วัน (K-ETA)
- ไต้หวัน 90 วัน
- ฮ่องกง 30 วัน
- จีน 30 วัน (ใหม่ 2024)
- ฯลฯ

## Budget Allocation
- ตั๋วเครื่องบิน: 30-40%
- ที่พัก: 25-30%
- อาหาร: 15-20%
- Transport: 5-10%
- Activity: 10-15%
- Shopping: 5-10%
- Insurance: 1-2%
- Buffer: 10%

## ตัวอย่างใช้งาน
```
/travel-planner
/travel-planner ญี่ปุ่น 7 วัน solo งบ 60k
/travel-planner เกาะลันตา 4 วัน คู่รัก งบ 25k
/travel-planner ยุโรป 14 วัน 2 คน งบ 250k
/travel-planner visa Schengen
/travel-planner packing list หนาว
```

## Output
ไฟล์ `.md` ชื่อ `trip-<destination>-YYYY-MM-DD.md`
- Day-by-day itinerary
- Budget breakdown
- Visa + document checklist
- Packing list
- Travel apps + emergency contacts
