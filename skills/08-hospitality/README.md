# หมวด 8: Hospitality & Service (5 skills)

Skills สำหรับธุรกิจบริการ ร้านอาหาร โรงแรม ทัวร์ สปา ร้านดอกไม้

| Skill | คำสั่ง | ใช้เมื่อไหร่ |
|-------|--------|-------------|
| Restaurant Owner | `/restaurant-owner` | menu, pricing, F&B operations |
| Hotel Concierge | `/hotel-concierge` | guest service, recommendation |
| Tour Guide Pro | `/tour-guide-pro` | itinerary, storytelling สถานที่ |
| Salon Owner | `/salon-owner` | booking, service menu, retention |
| Florist Pro | `/florist-pro` | arrangement, design brief, occasion |

## รูปแบบ output ทั่วไป

- Service menu + pricing
- Customer journey script
- Operational checklist
- Booking/reservation template

## Workflow ตัวอย่าง

```
1. /restaurant-owner ออกแบบ menu ใหม่
   ↓
2. /food-photographer (graphic-designer-pro) ภาพ menu
   ↓
3. /social-media-admin โปรโมท menu ใหม่
   ↓
4. /customer-service-pro CS script รับ booking
```
