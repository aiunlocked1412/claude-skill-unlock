# Prompt Formula — Florist Pro

## Template คำถาม context

```
โอกาส: <wedding / funeral / birthday / anniversary / sorry / corporate>
ผู้รับ: <เพศ + อายุ + relationship + ความชอบ>
งบ: ฿<ตัวเลขชัด>
Size: <small / medium / large / installation>
Style: <Western / Japanese / Modern / Thai / Tropical / Garden>
Delivery: <pickup / send> + <date> + <time> + <address>
Card message: "<ข้อความ>"
Special: <allergy / preference / cultural>
```

## Bouquet Design Prompt

```
ออกแบบช่อ <โอกาส> งบ ฿<X>:
Output design brief:
- Hero flower: <ดอก> × <จำนวน>
- Supporting: <2-3 ชนิด>
- Greenery: <ใบไม้>
- Color palette: <3-4 สี> + theory (mono/analogous/complementary)
- Vessel/wrap: <กระดาษ/ตะกร้า/vase>
- Style reference: <link หรือ keyword>
- Estimated cost breakdown:
  * Material: 30-40%
  * Labor: 15-20%
  * Packaging + delivery: 10-15%
  * Margin: 30-40%
```

## Wedding Arrangement Prompt

```
Wedding flower brief:
- Bride bouquet: style + color + size + flower list
- Bridesmaid bouquet (× N): smaller, complementary
- Boutonniere (× N): groom + groomsmen
- Ceremony arch / aisle
- Reception centerpiece × table
- Cake flower (optional)
- Sweetheart table

Logistics:
- Order flower 14 วันล่วงหน้า + buffer 20%
- Setup 4 ชม. ก่อนงาน
- Cold storage on-site
```

## Funeral / Memorial Prompt

```
Funeral arrangement:
- Type: พวงหรีด (wreath) / basket / casket spray / urn
- ศาสนา: <พุทธ / คริสต์ / อิสลาม> — flower selection ต่างกัน
- Color: ขาว / เหลือง / ม่วงอ่อน (ห้ามแดง/ชมพูสด)
- Card message: respectful, condolence
- Delivery: วัด / โบสถ์ + เวลา (ก่อน service 1 ชม.)
- Cultural note: พุทธ ใช้ มาลัยมะลิ + ดอกเบญจมาศ
```

## Birthday Bouquet Prompt

```
Birthday bouquet:
- ผู้รับ: <profile>
- Color: <สดใส pastel / vivid / soft>
- Hero flower: ตามความชอบ (rose / tulip / sunflower / peony)
- Add detail: บอลลูน, ตุ๊กตา, การ์ด, ขนม
- งบ ฿<X>:
  * 500-1,000 = mini bouquet 5-7 ดอก
  * 1,000-2,000 = standard 10-15 ดอก
  * 2,000-5,000 = premium 20-30 ดอก
  * 5,000+ = luxury / installation
```

## Order & Delivery System Prompt

```
ระบบ order + delivery:
1. Order in (LINE/IG/call): confirm brief + budget + delivery time
2. Quote + 50% deposit (order > 1,500)
3. Day before: prep flower + condition
4. Day of: arrange 4 ชม. ก่อน delivery
5. Delivery: insulated box + รถแอร์ + 2 ชม. หลังจัด
6. Photo confirm: ก่อน hand-off + หลัง delivery
7. Care card: cut stem + เปลี่ยนน้ำ + อายุ

Shelf life:
- Rose: 5-7 วัน (cold 4°C)
- Lily: 7-10 วัน
- Hydrangea: 3-5 วัน
- Orchid: 10-14 วัน
```

## Tips

- **ถาม budget ก่อน** design — กัน revise วน
- **Cultural taboo** ตรวจ — funeral ไม่ใช้แดง/ชมพูสด
- **Allergy** ผู้รับ — lily pollen, rose thorn
- **Photo confirm** ก่อน delivery ลด dispute 90%
- **Order wedding 14 วันล่วงหน้า** + buffer 20% — flower ขาดได้
