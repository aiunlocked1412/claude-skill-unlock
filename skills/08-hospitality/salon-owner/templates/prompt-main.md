# Prompt Formula — Salon Owner

## Template คำถาม context

```
ประเภท: <hair / nail / spa / brow / lash / multi>
Location + ระดับ: <ห้าง/ตึกแถว/boutique> + <mass/mid/premium>
ขนาด: <X> เก้าอี้/เตียง + <Y> ช่าง
รายได้: ฿<X>/เดือน + <Y> bookings/วัน
ราคาเฉลี่ย: ฿<X> ต่อบริการ
Pain: <ลูกค้าใหม่น้อย / repeat ต่ำ / ช่างลาออก / ราคาถูก>
```

## Service Menu Prompt

```
ออกแบบ service menu สำหรับร้าน <type>:
แบ่งเป็น 5 ประเภท:
1. Core (ขายตลอด) — 5-7 รายการ, margin 50-65%
2. Signature (Hero) — 3-5 รายการ, margin 60-75%
3. Add-on (Upsell) — 5-7 รายการ, margin 70-85%
4. Package (Bundle) — 3-5 sets, save 10-15%
5. Membership — 6-10 ครั้ง / 6-12 เดือน

แต่ละรายการ:
- ชื่อ memorable
- Time (นาที)
- Price (basic/premium/VIP)
- ใครทำได้ (junior/senior/director)
```

## Pricing Tiers Prompt

```
ตั้ง pricing 3 tiers:
- Basic: ลูกค้าใหม่ทดลอง, junior 1-3 ปี
- Premium: ลูกค้าประจำ, senior 3-7 ปี
- VIP: high-end, top stylist 7+ ปี

สูตร:
- Premium = Basic × 1.4-1.6
- VIP = Premium × 1.4-1.6

เปรียบเทียบกับ 3 ร้านคู่แข่งใน 1 กม.
ระบุเหตุผลทุกราคา (skill, product, time, experience)
```

## Booking System Prompt

```
ตั้งระบบจองด้วย LINE OA:
Flow:
1. Greeting → ถามบริการ + ช่าง + วัน-เวลา
2. Confirm + ส่งใบนัด (วันที่ + ราคา + บริการ)
3. Reminder T-1 วัน 16:00
4. Reminder T-2 ชม. + GPS link
5. Post-visit T+1 ขอบคุณ + review

No-show policy:
- Strike 1: เตือน + reschedule
- Strike 2: มัดจำ 30%
- Strike 3: ตัด preferred list
```

## Retention Plan Prompt

```
สร้าง retention lifecycle:
- T+0 (first visit): welcome card + IG follow
- T+7: thank you + after-care tip
- T+30: "อีก 6 สัปดาห์ถึงเวลาตัด" reminder
- T+90: birthday gift / special offer
- T+120 (ไม่กลับ): win-back free 30-min treatment
- VIP unlock (6 ครั้ง): private number + tier upgrade

KPI:
- 30-day return ≥ 40%
- 90-day return ≥ 65%
- LTV/CAC ≥ 3.0
```

## Upsell Script Prompt

```
เขียน upsell script "เห็น → แนะ → ขาย":
1. เห็น (diagnose): สังเกตสภาพผม/เล็บ/ผิว
2. แนะ (educate): อธิบายปัญหา + solution + benefit
3. ขาย (offer): ราคา + เวลา + ทำเลย/ครั้งหน้า

Add-on options + scripts:
- Hair: Olaplex, scalp spa, treatment
- Nail: cuticle care, gel topcoat, nail art
- Spa: scalp massage, foot reflex, mask

ห้าม: กดดัน, ขายของไม่จำเป็น, ขายระหว่าง relax
```

## Tips

- **Customer profile** บันทึกครบ — สูตรสี, allergy, มือถนัด
- **Reminder T-1** ลด no-show 30-50%
- **ราคาขึ้น** แจ้งลูกค้าประจำล่วงหน้า 30 วัน
- **Stylist binding** ผ่านระบบ ไม่ใช่ตัวคน — กัน turnover ลูกค้าหาย
- **Allergy test** ก่อนสี/ดัด ทุกครั้งสำหรับลูกค้าใหม่
