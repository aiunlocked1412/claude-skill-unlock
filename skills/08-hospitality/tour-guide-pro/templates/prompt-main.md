# Prompt Formula — Tour Guide Pro

## Template คำถาม context

```
Destination: <จังหวัด/ภูมิภาค> + <season>
Duration: <จำนวนวัน-คืน> + <date>
Group: <จำนวน> คน, อายุเฉลี่ย <X>, สัญชาติ <Y>
Theme: <culture / nature / adventure / food / wellness>
Budget: <economy / standard / premium> ฿<X>/หัว/วัน
Constraint: <senior / kids / vegetarian / halal / mobility>
```

## Itinerary Prompt

```
ออกแบบ itinerary <จำนวนวัน> สำหรับ <destination>:
ใช้สูตร 3-2-1 ต่อวัน:
- 3 highlight สำคัญ
- 2 meal คุณภาพ (ระบุร้าน + เมนู)
- 1 surprise (กิจกรรม off-brochure)

Format ต่อวัน:
- Morning (8-12): กิจกรรมหนัก
- Lunch (12-13:30): ร้าน + menu
- Afternoon (14-17): relaxing / shopping
- Dinner (18-20): experience meal
- Evening (20-22): show / market / free

Pacing rule: max 3 highlight, buffer 30 นาที
```

## Storytelling Framework Prompt

```
เขียน storytelling สำหรับ <สถานที่> (90-180 วินาที):
1. Hook (10 วิ): ตัวเลขแปลก / fact ลับ / คำถาม
2. History (30 วิ): สร้างเมื่อไร โดยใคร เพื่ออะไร
3. Culture (30 วิ): ทำไมสำคัญ + connect กับชีวิตคนปัจจุบัน
4. Anecdote (30 วิ): เรื่องเล่า guide-only (ไม่ใช่ Wikipedia)
5. Photo cue (10 วิ): มุมไหนสวย + เวลาแสงดี
6. Payoff (20 วิ): คำถามทิ้งคิด / connect สถานที่ถัดไป

ภาษา: <ไทย / EN / ทั้งสอง>
Tone: <educational / entertaining / spiritual>
```

## Group Dynamics Prompt

```
จัดการ group <type> <จำนวนคน>:
- Mixed (random): ice-breaker + buddy system
- Family: snack + toilet ทุก 90 นาที + kid activity
- Senior 60+: เดินช้า + เก้าอี้พัก + ปฐมพยาบาล ready
- Solo mixed: table mate matching + optional activity
- School: head count + 1:10 ratio
- Corporate: team building + awards + branding

Output: daily routine + risk + checklist
```

## Safety Protocol Prompt

```
เขียน safety protocol สำหรับ <destination> <duration>:
- Pre-tour checklist: emergency contact, ยาประจำตัว, ประกัน, first aid
- 4-tier emergency: Minor / Moderate / Major / Critical
- Local hospital list (ทุก stop ที่ไป)
- Contact: 1669 (medical), 191 (police), 1155 (tourist)
- Insurance hotline 24/7
- Repat plan ต่างชาติ
```

## Bus Microphone Script Prompt

```
เขียน script ไมค์ทัวร์:
- เปิดทัวร์ (วันแรก, 5 นาที):
  * Welcome + แนะนำตัว + คนขับ
  * Itinerary โดยรวม
  * Safety briefing
  * Group rule
  * Ice-breaker 2-3 นาที
- Daily commentary: 3-5 segment ระหว่างรถ
- Daily closing: recap + preview พรุ่งนี้
```

## Tips

- **Buffer time 30 นาที** ทุก segment — กัน traffic, toilet, photo
- **Head count ก่อน-หลัง** ทุก stop — ห้ามลืม
- **Verify reservation** ทุกร้าน + ที่พัก 1 วันก่อน
- **Backup plan** weather + ปิดถนน — เตรียมไว้ทุกวัน
- **Senior group** = เดินช้า + อาหารอ่อน + bathroom break เพิ่ม
