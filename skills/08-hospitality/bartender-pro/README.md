# Bartender Pro

> AI Mixologist — cocktail recipe, mixology technique, bar setup, menu engineering, food pairing

## ⚡ ใช้ยังไง

```
/bartender-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Cocktail Recipe classic + modern + signature พร้อม Thai local ingredients
- ✅ Technique Guide shake, stir, muddle, infusion, garnish ตั้งแต่มือใหม่
- ✅ Bar Setup equipment list เรียงตาม priority + mise en place workflow
- ✅ Menu Engineering food cost, pricing, seasonal menu rotation
- ✅ Ingredient Guide spirits, local substitutes, ของที่หาได้ในไทย
- ✅ Food & Cocktail Pairing จับคู่ cocktail กับอาหาร

## 💡 Use cases

- home bartender อยากทำ cocktail ดีๆ ที่บ้านด้วยงบไม่เยอะ
- เปิด bar ใหม่ต้องการ signature menu ที่ใช้ local ingredients
- bartender อยากเรียนรู้ classic cocktail patterns
- ร้านอาหารอยากเพิ่ม cocktail menu เสริม
- จัดงาน party อยากทำ mocktail สำหรับทุกคน

## 🎓 Tips

- บอก **spirit ที่มี + flavor profile ที่ชอบ** → ได้ recipe ที่ทำได้ทันที
- บอก **concept ร้าน** → ได้ signature menu ที่ match brand
- ใช้คู่กับ `/chef-pro` → pairing cocktail กับ tasting menu
