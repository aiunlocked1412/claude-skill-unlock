# Tutor Pro

> ติวเตอร์ส่วนตัว 1-on-1 — diagnostic test, learning style, แผนติวรายบุคคล, adaptive exercise

## ⚡ ใช้ยังไง

```
/tutor-pro
```

Skill จะถามข้อมูลนักเรียน (ระดับ + วิชา + เป้าหมาย + เวลา) แล้วเริ่มจาก diagnostic test เพื่อวินิจฉัยจุดอ่อนก่อน — ก่อนจะออกแผนติวที่เฉพาะตัว

## 🎯 ทำอะไรได้บ้าง

- ✅ Diagnostic test 15-20 ข้อ (วินิจฉัยจุดอ่อนก่อนติว)
- ✅ ประเมิน Learning Style (VARK — Visual / Auditory / Reading / Kinesthetic)
- ✅ แผนติวรายบุคคล 4/8/12 สัปดาห์ — Mastery-based
- ✅ แบบฝึก Adaptive (ปรับยากตามผลลัพธ์)
- ✅ Spaced Repetition Schedule (Anki-style)
- ✅ Mock exam + วิเคราะห์ผลละเอียด (O-NET, A-Level, TCAS, IELTS, SAT)
- ✅ Tutoring session script สอนสด 1 ชม.

## 💡 ต่างจาก /teacher-pro ยังไง?

| Teacher Pro | Tutor Pro |
|-------------|-----------|
| สอนห้อง 30-40 คน | ติว 1-on-1 |
| แผนการสอนตามหลักสูตร | แผนตามจุดอ่อนรายคน |
| Rubric กลุ่ม | Mastery รายคน |
| ใช้ในห้องเรียน | ใช้ที่บ้าน / ติวเตอร์ |

## 🎓 เหมาะกับใคร

- ติวเตอร์ที่รับสอนตัวต่อตัว
- ผู้ปกครองที่อยากสอนลูกเอง
- นักเรียนเตรียมสอบ TCAS / O-NET / GAT-PAT / IELTS / SAT / GED
- คนวัยทำงานที่อยากเรียนใหม่ (re-skill / up-skill)
