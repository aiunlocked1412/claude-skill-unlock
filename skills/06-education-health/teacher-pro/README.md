# Teacher Pro

> ผู้ช่วยครูมืออาชีพ — แผนการสอน + ใบงาน + quiz + rubric + การบ้าน ครบชุดในครั้งเดียว

## ⚡ ใช้ยังไง

```
/teacher-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ แผนการสอน 1 คาบ (50 นาที) ตามโครง 5E
- ✅ แผนการสอนรายสัปดาห์ (5 คาบ)
- ✅ ใบงาน / แบบฝึกหัด พร้อมเฉลย
- ✅ Quiz / แบบทดสอบ (ไล่ระดับ Bloom's)
- ✅ Rubric 4 ระดับ พร้อม descriptor ชัดเจน
- ✅ ไอเดียการบ้านสร้างสรรค์ 3 แบบ
- ✅ กิจกรรม warm-up / ice-breaker

## 💡 Use cases

- ครูประถม / มัธยม / มหาวิทยาลัย
- ครูสอนพิเศษ / ติว
- ครูพิเศษ / ผู้ปกครอง home school
- อาจารย์มหาวิทยาลัยออกแบบ course syllabus

ดู [`examples/example-output.md`](./examples/example-output.md) — แผนการสอนคณิตศาสตร์ ม.2 เรื่องทฤษฎีบทพีทาโกรัส 50 นาที ครบชุด

## 🎓 Tips

- บอก **ระดับชั้น + จำนวนนักเรียน** → ได้แผนที่เหมาะกับห้องจริง
- ถ้ามี **objective** ชัดเจน → rubric แม่นขึ้น
- ใช้ `/researcher-pro` ต่อยอดทำ research เพิ่มเติม
