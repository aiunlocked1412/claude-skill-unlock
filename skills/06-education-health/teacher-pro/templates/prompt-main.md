# Prompt Patterns สำหรับ Teacher Pro

## 1. Pattern: แผนการสอน 5E

```
ออกแบบแผนการสอนตามโครง 5E สำหรับ:
- ระดับชั้น: <...>
- วิชา: <...>
- เรื่อง: <...>
- เวลา: <...> นาที
- จำนวนนักเรียน: <...> คน

แบ่งเวลาเป็น:
Engage 10% / Explore 20% / Explain 30% / Elaborate 30% / Evaluate 10%

ทุกกิจกรรมต้อง:
- Actionable (ครูเปิดอ่านแล้วสอนได้เลย)
- มีคำถามกระตุ้นคิด 2-3 ข้อ
- ระบุสื่อ/อุปกรณ์ที่ต้องใช้
- มี plan B ถ้าอุปกรณ์ไม่พร้อม
```

## 2. Pattern: Quiz ไล่ระดับ Bloom's

```
สร้าง quiz <จำนวน> ข้อ เรื่อง <หัวข้อ> ระดับ <ชั้น>

โครงสร้าง:
- ข้อ 1-3: Remember (จำ) — multiple choice
- ข้อ 4-6: Understand (เข้าใจ) — fill-in หรือ short answer
- ข้อ 7-9: Apply (ใช้) — โจทย์ประยุกต์
- ข้อ 10-12: Analyze (วิเคราะห์) — case / scenario
- ข้อ 13+: Evaluate + Create — open-ended

ทุกข้อต้องมี:
- คำตอบที่ถูก
- เฉลยอธิบาย (step-by-step)
- คะแนน
- ข้อสังเกต (misconception ที่พบบ่อย)
```

## 3. Pattern: Rubric 4 ระดับ

```
สร้าง rubric ประเมิน <ชิ้นงาน> สำหรับ <ระดับชั้น>

4 ระดับ:
- 4 = ดีเยี่ยม (Exceeds expectation)
- 3 = ดี (Meets expectation)
- 2 = พอใช้ (Approaching)
- 1 = ควรปรับปรุง (Below)

เกณฑ์ 3-5 อย่าง — แต่ละเกณฑ์ต้อง:
- เขียน descriptor ชัดเจน (ไม่ใช่แค่ "ดีมาก")
- ใช้ verb observable (อธิบายได้, นำเสนอได้, วิเคราะห์ได้)
- มีตัวอย่าง "งานระดับ 4" และ "งานระดับ 2"
```

## 4. Pattern: ใบงานสร้างสรรค์

```
ออกแบบใบงาน <เรื่อง> สำหรับ <ระดับ> 1 หน้า A4

องค์ประกอบ:
- คำชี้แจง 2-3 บรรทัด
- ส่วนที่ 1: warm-up (2-3 ข้อง่าย)
- ส่วนที่ 2: core (5-7 ข้อหลัก)
- ส่วนที่ 3: challenge (1-2 ข้อยาก) — สำหรับเด็กเก่ง
- Real-life context อย่างน้อย 2 ข้อ
- เฉลยแยกหน้า พร้อมวิธีคิด
```

## 5. Pattern: การบ้านสร้างสรรค์

```
เสนอ 3 ตัวเลือกการบ้าน เรื่อง <หัวข้อ>:

1. Traditional — ฝึกโจทย์ เข้มข้น (สำหรับเด็กที่ชอบทำแบบเดิม)
2. Creative — ทำ visual (mind map / comic / poster / คลิป 60 วิ)
3. Real-life — ไปเก็บข้อมูลจริง / สัมภาษณ์ / สังเกตการณ์

ทุกตัวเลือก:
- ใช้เวลา 30-60 นาที
- เขียน clear deliverable (ส่งอะไร/กี่หน้า/กี่นาที)
- มีเกณฑ์ประเมินสั้นๆ
```
