# Prompt Patterns สำหรับ Mental Wellness

## 0. Pattern: Safety Screening (ใช้ก่อนทุก output)

```
Scan user input สำหรับ:
- Suicide ideation: "อยากตาย", "ไม่อยากอยู่", "หาวิธีจบ"
- Self-harm: "ทำร้ายตัวเอง", "กรีด"
- Abuse: "ถูกทำร้าย", "บังคับ"
- Severe symptoms: ฟังเสียง, เห็นภาพหลอน, panic ต่อเนื่อง

ถ้าพบ → STOP + refer:
- 1323 (สุขภาพจิต)
- 02-113-6789 (Samaritans)
- ER โรงพยาบาล

ห้าม proceed ด้วย skill content
```

## 1. Pattern: Daily Journal Prompts

```
INPUT:
- Time of day: morning / evening
- Context: regular / stress / decision / relationship / self-discovery
- Energy: high / medium / low

OUTPUT: 3-5 prompts (เลือกที่เหมาะสุด)

Format:
- Open-ended question
- Concrete enough to answer
- 2-3 ประโยคต่อคำตอบ
- ไม่บังคับ — "เลือก 1-2 ที่ตรงใจ"

Examples:
- Morning: "วันนี้ฉันอยากรู้สึก ___ (1 คำ)"
- Evening: "3 things ดีๆ วันนี้คือ..."
- Stress: "ความเครียดนี้บอกอะไรกับฉัน?"
```

## 2. Pattern: Gratitude Practice

```
Variation:
- 3 Blessings (Seligman)
- Gratitude Letter
- Gratitude Walk
- Mental Subtraction (imagine without)

Output:
- Setup
- Step-by-step instructions
- Time required
- Frequency (daily / weekly)
- Evidence base
- Common pitfall
- Adaptation for Thai culture
```

## 3. Pattern: Emotional Regulation Tool

```
INPUT:
- Emotion type: anger / fear / sadness / anxiety / overwhelm
- Intensity: 1-10
- Context

OUTPUT: 1-2 tools เลือกที่เหมาะ

Tools available:
- Name it to tame it (mild-moderate)
- TIPP (severe — ต้อง physical reset)
- STOP (general mindfulness)
- Opposite action (sustained emotion)
- Emotion wheel (clarification)

Format:
- Step-by-step
- Time: 1-5 min
- Why it works (brief science)
- When NOT to use
```

## 4. Pattern: CBT Thought Reframing

```
INPUT:
- Situation
- Automatic thought (raw)
- Mood + intensity

OUTPUT: Filled thought record

Steps:
1. Situation (objective)
2. Mood (1-10)
3. Automatic thought
4. Identify cognitive distortion (10 standard)
5. Evidence FOR
6. Evidence AGAINST
7. Balanced thought
8. New mood (1-10)

Tone: validating, not dismissive
"ความคิดนี้สมเหตุสมผลที่จะมี — แต่ลองดู evidence อีกมุมไหม?"
```

## 5. Pattern: Self-Compassion Exercise

```
Context: failure / shame / mistake / loss

Choose 1:
1. Self-Compassion Break (3 phrases + hand on heart)
2. Letter to self (300 words, perspective of compassionate friend)
3. "How would I treat a friend?" reframe

Format:
- Validate first ("ที่เธอรู้สึกแย่เป็นเรื่องธรรมชาติ")
- Then introduce exercise
- Specific instructions
- Time required
```

## 6. Pattern: Sleep Hygiene Audit

```
Ask user about current habits:
- Sleep time / wake time
- Caffeine intake (when, how much)
- Screen time before bed
- Bedroom environment
- Bedtime routine
- Issues (can't fall asleep / wake up middle / wake up tired)

Output:
- Audit (current state)
- Top 3 things to change (priority order)
- 7-day mini plan
- When to seek medical (insomnia > 3 weeks)
```

## 7. Pattern: 5-min Stress Reset

```
Combo of 4 mini-techniques:
- 5-4-3-2-1 grounding (1 min)
- Box breathing (2 min)
- Mini reframe (1.5 min)
- Action step (0.5 min)

Format:
- Time-stamped
- Verbatim cues
- Pause indicators
- ทำได้ที่โต๊ะทำงาน / ในห้องน้ำ / ในรถ
```

## 8. Pattern: Habit Builder (BJ Fogg)

```
INPUT:
- Goal habit
- Existing anchor habit
- Current state (never done / inconsistent / falling off)

OUTPUT:
- Tiny version of habit (< 5 min, < 50% commitment)
- Anchor formula: "After [existing], I will [tiny new]"
- Tracking sheet (21 days)
- Celebration (immediate after each rep)
- Rules: ห้าม miss 2 วันติด

Examples:
- Meditate 10 min → "After brushing teeth morning, I will meditate 2 min"
- Exercise → "After putting on PJs, I will do 5 push-ups"
```
