# Prompt Patterns สำหรับ Coach Pro

## 1. Pattern: Coaching Program 12 Sessions

```
ออกแบบ coaching program 12 sessions สำหรับ:
- ประเภท: <life/career/business/health>
- ลูกค้า: <persona + background>
- Presenting issue: <อะไร>
- Desired outcome (3 เดือน): <อะไร>

โครงสร้าง:
- Session 1 = Intake + Chemistry
- Session 2-3 = Goal + Reality
- Session 4-5 = Strengths + Obstacles
- Session 6-7 = Options + Will
- Session 8-10 = Integration + habits
- Session 11-12 = Celebration + next chapter

ทุก session ต้องมี:
- Theme ชัดเจน
- 3-5 powerful questions
- Action items 1-2 ข้อ
- Reflection prompt 1 ข้อ
```

## 2. Pattern: Powerful Questions

```
สร้าง powerful questions 20-30 ข้อ สำหรับหัวข้อ <topic>

หลักการ:
- Open-ended (ตอบได้ยาวๆ)
- ขึ้นต้นด้วย "อะไร / อย่างไร / เมื่อไหร่"
- หลีกเลี่ยง "ทำไม" (ทำให้ defensive)
- กระตุ้น reflection + insight
- ไม่มี leading question (นำคำตอบ)

แบ่งเป็น:
- Awareness questions (ช่วยเห็นตัวเอง)
- Values questions (อะไรสำคัญ?)
- Vision questions (อนาคตเป็นยังไง?)
- Action questions (ทำอะไรต่อ?)
- Reflection questions (เรียนรู้อะไร?)
```

## 3. Pattern: Session Plan GROW

```
ออกแบบ session plan 60 นาที ใช้ GROW:

GOAL (10 นาที):
- ลูกค้าอยากเอาอะไรออกจาก session นี้?
- Success looks like what?

REALITY (15 นาที):
- ตอนนี้เป็นอย่างไร?
- เคยลองอะไรไปแล้ว?
- อะไรที่ขวางอยู่?

OPTIONS (20 นาที):
- มีทางเลือกอะไรบ้าง? (อย่างน้อย 5 ทาง)
- ถ้า money no object?
- ถ้าไม่กลัวล้มเหลว?

WILL (15 นาที):
- เลือกทางไหน?
- First step เมื่อไหร่?
- อุปสรรคที่อาจเจอ?
- ใครช่วย support?
```

## 4. Pattern: Intake Form

```
สร้าง intake form 15-20 ข้อ สำหรับ <type> coaching

สาม section:
1. Current situation (ชีวิตตอนนี้)
2. Goals & vision (อยากไปไหน)
3. Coaching preferences (สไตล์ที่ชอบ)

รวมคำถาม scale 1-10 อย่างน้อย 5 ข้อ
(เช่น ความพอใจชีวิตตอนนี้, พลังงาน, ความมั่นใจ, ฯลฯ)

ลงท้ายด้วย:
- Coaching agreement checklist
- Confidentiality note
- Fee + schedule
```

## 5. Pattern: Progress Tracker

```
ออกแบบ progress tracker 3 layers:

1. Daily (2 นาที)
- Mood 1-10
- Today's focus (1 ข้อ)
- Evening reflection (1 ข้อ)

2. Weekly (10 นาที)
- Wins (3 ข้อ)
- Challenges (2 ข้อ)
- Next week priority (3 ข้อ)
- Coach check-in note

3. Monthly (30 นาที)
- Progress vs goal (quantitative + qualitative)
- What's working? / Not working?
- Adjustments needed
- Celebration
```
