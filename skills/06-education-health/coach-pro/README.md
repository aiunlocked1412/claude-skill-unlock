# Coach Pro

> ผู้ช่วย Coach มืออาชีพ — ออกแบบ coaching program ด้วย GROW model + powerful questions

## ⚡ ใช้ยังไง

```
/coach-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Coaching program 8-12 sessions (GROW-based)
- ✅ Intake form สำหรับลูกค้าใหม่
- ✅ Session plan template (60 นาที)
- ✅ Progress tracker (weekly / monthly / quarterly)
- ✅ Reflection journal prompts (30 ข้อ)
- ✅ Powerful questions bank
- ✅ Goal-setting worksheet (SMART + values)

## 💡 Use cases

- Life coach — เป้าหมายชีวิต, balance, purpose
- Career coach — เปลี่ยนสายงาน, ปั้นสายอาชีพ, promotion
- Business coach — เจ้าของธุรกิจ, SME, executive
- Health / Wellness coach — habit, energy, work-life

## 🎓 Tips

- บอก **background + presenting issue** ของลูกค้า → program แม่นขึ้น
- ใช้ quarterly review ปรับโปรแกรมตาม progress
- ใช้ `/psychologist-helper` คู่เมื่อเจอ emotional blocker (ระวัง scope)
