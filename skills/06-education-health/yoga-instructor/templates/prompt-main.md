# Prompt Patterns สำหรับ Yoga Instructor

## 1. Pattern: Class Sequence (60/75/90 min)

```
Style: <Vinyasa / Hatha / Yin / Restorative / Prenatal>
Level: <beginner / intermediate / advanced / mixed>
Length: <60 / 75 / 90 min>
Theme: <hip opening / heart opening / grounding / strength / etc.>
Peak pose: <optional>

Output structure (universal sequencing):
1. Centering (3-5 min) — sit, breath, intention
2. Warm-up (5-10 min) — joint mobility
3. Sun salutations A & B (5-10 min)
4. Standing sequence (15-20 min)
5. Peak pose (5-10 min) — apex
6. Cool-down (10-15 min) — fold, twist
7. Hip openers (5-10 min)
8. Backbend (5 min, optional)
9. Inversion (5 min, optional)
10. Savasana (5-10 min)

Each pose:
- Sanskrit + English + Thai
- Hold (breaths or seconds)
- Cue alignment 2-3 keys
- Modification (beginner/intermediate)
- Breath cue
```

## 2. Pattern: Asana Breakdown (Detailed)

```
Asana: <name>

Output sections:
- Category + difficulty
- Setup (step-by-step entry)
- Alignment (head to toe)
- Breath pattern
- Engagement (bandha + muscles)
- Contraindications
- Modifications 3 levels (Beginner / Intermediate / Advanced)
- Benefits (physical / mental / energetic)
- Counter pose
- Common mistakes
- References (Iyengar / Kaminoff)
```

## 3. Pattern: Pranayama Practice

```
Technique: <Nadi Shodhana / Ujjayi / Kapalabhati / Box / 4-7-8>

Output:
- Type (balancing / energizing / calming)
- Duration (5-10 min)
- Position
- Setup (mudra, eyes)
- Breath pattern (count)
- Number of rounds
- Cautions (esp. pregnancy, BP)
- Effects (nervous system, mental)
- Progression for next session
```

## 4. Pattern: Closing Meditation Script

```
Length: <5 / 10 / 15 min>
Position: savasana / sukhasana
Theme: <body scan / loving kindness / visualization>

Output:
- Settling (1 min)
- Main practice (60-80% of time)
- Integration / silence (10-20%)
- Return + closing (1 min)

Script verbatim:
- [Pause indicators in seconds]
- "..." (what to say)
- Voice quality cues (soft, slow)
```

## 5. Pattern: Prenatal Class

```
Trimester: 1 / 2 / 3
Class length: <60 min>
Special considerations: <e.g., low energy, back pain, prep for labor>

Output:
- Avoid list (per trimester)
- Modified poses
- Props essential
- Pace (slow + restorative for late trimester)
- Pelvic floor + breath emphasis
- Birth preparation poses (3rd trimester)
- Communication with partner if accompanying
```

## 6. Pattern: Therapeutic Sequence

```
Issue: <lower back / shoulder / hip / SI joint / sciatica>
Severity: acute / chronic / post-injury (cleared by PT)

Output:
- Disclaimer: refer PT if not improving
- Avoid list
- Recommended sequence (gentle)
- Prop usage heavy
- Hold times (longer for therapeutic)
- Frequency (daily 15-20 min often better than 1x weekly)
- Self-monitoring cues (when to back off)
```

## 7. Pattern: 30-Day Program

```
Goal: <flexibility / strength / mind / weight / general>
Level start: <beginner / continuing>
Time per day: 20-60 min

Output:
- Week 1: Foundation
- Week 2: Build
- Week 3: Deepen
- Week 4: Integration

Each day:
- Style + length
- Focus
- 1 new asana to learn
- Notes (rest day, recovery)

Progress markers:
- Week 1 end: should feel ___
- Week 4 end: should master ___
```
