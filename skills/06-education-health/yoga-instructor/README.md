# Yoga Instructor

> ครูสอนโยคะ — Hatha / Vinyasa / Yin / Restorative / Prenatal sequences + asana + pranayama

## ⚠️ คำเตือนด้านสุขภาพ

โยคะเป็น physical practice — ก่อนเริ่มควรปรึกษาแพทย์ ถ้า:
- 🤰 ตั้งครรภ์ (โดยเฉพาะไตรมาสแรก / มี complication)
- 🦴 ปัญหาหลัง / คอ / ข้อ / กระดูก
- ❤️ โรคหัวใจ / ความดัน / glaucoma
- 🩺 หลังผ่าตัด / บาดเจ็บ
- 💊 ทานยาที่กระทบ blood pressure / balance

**Skill นี้ไม่ใช่การรักษาโรค — ถ้ามีปัญหาเฉพาะ ปรึกษา PT หรือแพทย์**

## ⚡ ใช้ยังไง

```

```

(เลือก style + level + length + theme)

## 🧘 Styles ที่รองรับ

- 🔥 **Vinyasa** — flow, breath-linked movement
- 💪 **Hatha** — alignment, longer holds
- 🌙 **Yin** — passive, 3-5 min holds
- 🛌 **Restorative** — props, full support
- 🤰 **Prenatal** — ปลอดภัยสำหรับคนท้อง
- ⚡ **Power yoga** — strength + cardio
- 🪷 **Ashtanga** — primary series

## 🎯 ทำอะไรได้บ้าง

- ✅ Class sequence 60/75/90 min ตามธีม + level
- ✅ Asana breakdown ละเอียด (alignment + modification 3 ระดับ)
- ✅ Pranayama (Nadi Shodhana, Box, 4-7-8, Ujjayi)
- ✅ Meditation integration ปิดคลาส
- ✅ Prop guide (block / bolster / strap / wall)
- ✅ Prenatal modification (ทุก trimester)
- ✅ Therapeutic sequence (back / shoulder / hip / SI)
- ✅ Daily home practice 20-30 min
- ✅ 30-day progressive program

## 💡 จุดเด่น

- **Sequencing principle ครบ** — centering → warm → flow → peak → cool → savasana
- **Modification ทุก asana** — beginner / intermediate / advanced
- **Contraindication ชัด** — เตือนตั้งครรภ์/บาดเจ็บ
- **3 ภาษา** — Sanskrit + ไทย + English

## 👥 เหมาะกับใคร

- ครูสอนโยคะ (RYT-200/500)
- Studio owner ที่วาง schedule
- Practitioner ที่อยากฝึกที่บ้าน
- Personal trainer เพิ่มเสริม
- Yoga therapist (สอนเฉพาะ case)
