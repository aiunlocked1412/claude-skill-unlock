# TCM Helper

> ผู้ช่วยแพทย์แผนจีน — สมุนไพรจีน, ตำรับ, เส้นลมปราณ, Pattern Analysis (เพื่อการศึกษาเท่านั้น)

## ⚡ ใช้ยังไง

```
/tcm-helper
```

## 🎯 ทำอะไรได้บ้าง

- ✅ สมุนไพรจีน (中药) — รส, อุณหภาพ, ช่องทางเข้า, ข้อควรระวัง
- ✅ ตำรับยาคลาสสิก (方剂) — 君臣佐使, องค์ประกอบ, หลักการ
- ✅ Pattern Analysis — syndrome differentiation จาก 8 Principles
- ✅ เส้นลมปราณ 12 เส้น + จุดฝังเข็มสำคัญ
- ✅ Patient Intake Form แบบ TCM
- ✅ การวินิจฉัย 4 วิธี (四诊) — ดู/ฟัง/ถาม/คลำ

## ⚠️ ข้อจำกัดสำคัญ

**เนื้อหาทั้งหมดมีวัตถุประสงค์เพื่อการศึกษาเท่านั้น — ไม่ใช่คำแนะนำทางการแพทย์ หากมีอาการเจ็บป่วย ต้องปรึกษาแพทย์หรือผู้ประกอบวิชาชีพที่มีใบอนุญาตเสมอ**

## 💡 Use Cases

- นักเรียนแพทย์แผนจีนทบทวนสมุนไพรและตำรับ
- ผู้ปฏิบัติการ TCM ค้นข้อมูล reference อย่างรวดเร็ว
- ผู้สนใจ TCM อยากเข้าใจหลักการ yin-yang และ 8 principles
- นักวิจัยที่ศึกษา TCM approach ในงานวิชาการ

## 🎓 Tips

- ระบุ **ชื่อสมุนไพรภาษาจีนหรือ Pinyin** → ข้อมูลแม่นยำกว่า
- บอก **วัตถุประสงค์** (เรียน/ทบทวน/งานวิจัย) → ได้ระดับรายละเอียดที่เหมาะ
- ใช้คู่กับ `/thai-traditional-medicine` → เปรียบเทียบ 2 ระบบการแพทย์
