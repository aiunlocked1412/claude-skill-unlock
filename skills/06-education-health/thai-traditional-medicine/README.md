# Thai Traditional Medicine

> ผู้ช่วยแพทย์แผนไทย — สมุนไพรไทย, นวดแผนไทย, ธาตุเจ้าเรือน, สูตรตำรับ (เพื่อการศึกษาเท่านั้น)

## ⚡ ใช้ยังไง

```
/thai-traditional-medicine
```

## 🎯 ทำอะไรได้บ้าง

- ✅ สมุนไพรไทย — รส, สรรพคุณ, ธาตุที่แก้, ข้อควรระวัง
- ✅ ทฤษฎีธาตุสี่ (ดิน/น้ำ/ลม/ไฟ) และธาตุเจ้าเรือน
- ✅ นวดแผนไทย — เส้นประธานสิบ, ท่า, จุดบำบัด
- ✅ สูตรตำรับยาไทย (ยาต้ม, ยาลูกกลอน, ลูกประคบ)
- ✅ ซักประวัติแบบแผนไทย — ธาตุเจ้าเรือน + อาการตามธาตุ
- ✅ อ้างอิงตำราคัมภีร์และยาศิลาจารึก

## ⚠️ ข้อจำกัดสำคัญ

**เนื้อหาทั้งหมดมีวัตถุประสงค์เพื่อการศึกษาและอนุรักษ์ภูมิปัญญาไทยเท่านั้น ไม่ใช่คำแนะนำทางการแพทย์ หากมีอาการเจ็บป่วย ต้องปรึกษาแพทย์แผนไทยหรือแพทย์แผนปัจจุบันที่มีใบอนุญาตก่อนเสมอ**

## 💡 Use Cases

- นักเรียนแพทย์แผนไทยทบทวนสมุนไพรและตำรับ
- นักวิจัยที่ศึกษาภูมิปัญญาพื้นบ้านไทย
- ผู้สนใจธาตุเจ้าเรือนและการดูแลสุขภาพแบบองค์รวม
- หมอนวดแผนไทยที่ต้องการ reference เส้นประธานสิบ
