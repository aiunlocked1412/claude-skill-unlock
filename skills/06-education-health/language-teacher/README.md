# Language Teacher

> ครูสอนภาษาต่างประเทศแบบ TEFL — English / Chinese / Japanese / Korean สำหรับคนไทย

## ⚡ ใช้ยังไง

```
/language-teacher
```

(เลือกภาษา + กิจกรรม จากเมนู หรือบอกเป้าหมายเลย)

## 🌍 ภาษาที่รองรับ

- 🇬🇧 **English** (CEFR A1-C2, IELTS, TOEIC, TOEFL)
- 🇨🇳 **Chinese (Mandarin)** (HSK 1-6)
- 🇯🇵 **Japanese** (JLPT N5-N1)
- 🇰🇷 **Korean** (TOPIK I-II)

## 🎯 ทำอะไรได้บ้าง

- ✅ CEFR Placement Test (4 skills — R/L/W/S)
- ✅ Lesson Plan ตาม CEFR + CLT methodology
- ✅ Conversation practice / role-play สถานการณ์จริง
- ✅ Error correction แบบ supportive (recast)
- ✅ Vocabulary by chunks (lexical approach)
- ✅ Listening + shadowing
- ✅ Writing feedback (essay/email/business)
- ✅ Grammar explanation อธิบายภาษาไทย
- ✅ Test prep (TOEIC/IELTS/HSK/JLPT/TOPIK)

## 💡 จุดเด่น

- **เข้าใจ Thai L1 interference** — รู้ว่าคนไทยติดอะไร (tense, article, sound /θ/)
- **CEFR-aligned** — ทุก output ระบุระดับชัด
- **Chunk-based** — สอน collocation จริง ไม่ใช่ท่องคำเดี่ยว
- **Supportive correction** — ไม่ทำให้กลัวพูด

## 👥 เหมาะกับใคร

- ครูสอนภาษาที่ต้องเตรียมบทเรียน
- ติวเตอร์ภาษาตัวต่อตัว
- ผู้เรียนภาษาเอง (self-study)
- คนเตรียมสอบวัดระดับ
- คนเตรียมไปต่างประเทศ
