# Prompt Patterns สำหรับ Researcher Pro

## 1. Pattern: Literature Review (Thematic)

```
เขียน literature review แบบ thematic สำหรับ:
- Topic: <...>
- Field: <social/engineering/medical/humanities/business>
- Scope: <10 ปีหลัง / ทุกยุค / ช่วงเฉพาะ>
- Word count: <1,500-5,000 คำ>
- Citation style: APA 7 (default)
- จำนวน reference: <10-30>

โครงสร้าง:
1. Introduction (~10% ของ word count)
   - Importance
   - Scope
   - Roadmap

2. Theme 1 — <หลัก>
   - 3-5 key citations
   - Synthesis (ไม่ใช่ summary list)
   - Tension / debate

3. Theme 2 — <...>
4. Theme 3 — <...>

5. Research Gap
   - Knowledge / method / population gap

6. Conclusion (~10%)

หลัก:
- ทุกประโยคกล่าวอ้าง ต้องมี citation
- ใช้ signal verbs: argue, suggest, demonstrate, challenge
- สังเคราะห์ ไม่ใช่ summarize
- Balance voices (อ้างหลาย perspective)
- ชี้ตรงที่มี conflict ใน literature

References:
- Use APA 7 strictly
- Include DOI
- ถ้า AI ไม่แน่ใจ citation → flag "Verify via Google Scholar"
```

## 2. Pattern: Methodology (Quantitative)

```
เขียน methodology section สำหรับ quantitative study:
- Research question: <...>
- Design: <cross-sectional / RCT / quasi-exp / longitudinal>
- Word count: ~1,000-1,500

โครงสร้าง:
3.1 Research Design
- Design type + rationale
- Why not other design?

3.2 Participants
- Target population
- Sampling method (probability / non-probability)
- Sample size calculation (G*Power, formula)
- Inclusion / exclusion criteria
- Recruitment procedure

3.3 Instruments
- Each variable → instrument
- Validated scale ไหน? Cronbach's alpha
- Cultural adaptation (ถ้าใช้ใน Thai context)

3.4 Procedure
- Data collection process
- Timeline
- Quality control

3.5 Data Analysis
- Software (SPSS, R, Stata)
- Descriptive: mean, SD, frequency
- Inferential: t-test / ANOVA / regression / SEM
- Effect size
- Assumption checks
- Missing data handling

3.6 Ethical Considerations
- IRB/EC approval (number)
- Informed consent
- Anonymity / confidentiality
- Data security
```

## 3. Pattern: Methodology (Qualitative)

```
เขียน methodology section สำหรับ qualitative study:

3.1 Research Design
- Approach: phenomenology / grounded theory / ethnography / case study
- Paradigm: interpretivism / constructivism / critical
- Rationale

3.2 Participants
- Purposive sampling (criterion-based)
- Sample size: "until saturation" (ประมาณ 12-20)
- Inclusion criteria
- Recruitment

3.3 Data Collection
- Semi-structured interview / focus group / observation
- Interview guide (attach appendix)
- Duration, location
- Recording + transcription

3.4 Data Analysis
- Thematic analysis (Braun & Clarke 6 steps) OR
- Grounded theory (open-axial-selective coding) OR
- IPA (Smith 2007)
- Software (NVivo, ATLAS.ti, MAXQDA)
- Coding process (inter-rater reliability)

3.5 Trustworthiness (Lincoln & Guba 1985)
- Credibility (member checking, triangulation)
- Transferability (thick description)
- Dependability (audit trail)
- Confirmability (reflexivity)

3.6 Ethical Considerations
- IRB
- Informed consent (verbal + written)
- Anonymity (pseudonyms)
- Data protection
- Right to withdraw

3.7 Researcher Reflexivity
- Position of researcher
- Prior assumptions / biases
- How managed during research
```

## 4. Pattern: IMRaD Paper Outline

```
สร้าง paper outline (IMRaD) สำหรับ:
- Topic: <...>
- Target journal: <journal name + impact factor>
- Word limit: <5,000-8,000>
- Citation style: <APA/Vancouver/Chicago>

Sections:

Title (≤ 15 words, keywords)
Abstract (250 words, structured: Background/Methods/Results/Conclusion)
Keywords (5-8, MeSH/ERIC-compliant)

1. Introduction (10-15% word count)
2. Literature Review / Theoretical Framework (20-25%)
3. Methodology (15-20%)
4. Results (20-25%)
5. Discussion (20-25%)
6. Conclusion (5%)
7. References
8. Appendices

Each section:
- Sub-headings
- Word budget
- Key content to include
- Common mistakes to avoid
```

## 5. Pattern: Research Gap Analysis

```
วิเคราะห์ research gap ใน <topic>:

6 types of gaps:
1. Knowledge gap — "What is unknown?"
2. Methodological gap — "Different method needed?"
3. Population gap — "Whose perspective missing?"
4. Empirical gap — "Theory not tested?"
5. Conceptual gap — "New framework needed?"
6. Contextual gap — "Different setting (e.g., Thai/Asian)?"

Output:
- Summary of existing literature (5-10 key works)
- Each gap type:
  - What exists
  - What's missing
  - Why it matters
  - How this study addresses

Justify research contribution:
- Theoretical: ...
- Practical: ...
- Policy: ...
```

## 6. Pattern: Citation Reformat

```
Convert citations จาก <source style> → <target style>

Source:
1. (raw citation 1)
2. ...

Target style: APA 7 / Vancouver / Chicago 17 / MLA 9

Rules:
- ตรวจ author order (last, first vs first last)
- Journal italic / roman
- Year position
- DOI formatting
- Page range (en-dash vs hyphen)

⚠️ ถ้าไม่มีข้อมูลครบ → flag "Missing: [DOI/page/volume]"
```

## 7. Pattern: Conference Abstract

```
เขียน conference abstract สำหรับ <conference name>:
- Word limit: 250-300 (standard)
- Structure: IMRaD shortened

Structure:
- Background (2-3 sentences) — why this study matters
- Purpose / Research question (1 sentence)
- Methods (2-3 sentences) — design, sample, measures
- Results (3-4 sentences) — key findings with numbers
- Conclusion (1-2 sentences) — implication

Tips:
- ไม่ใส่ citations
- Active voice preferred
- Avoid jargon
- One idea per sentence
- End with impact statement
```
