## 📋 Quick Summary

- ADA 2025 ยกระดับ GLP-1 RA และ SGLT2i เป็น **first-line options** ใน T2DM + ASCVD/HF/CKD (ร่วมกับ metformin หรือแทนที่)
- **HbA1c goal** ปรับ individualized มากขึ้น — consider frailty, life expectancy
- เพิ่มบทใหม่เรื่อง **obesity-centric approach** — weight loss ≥15% = remission potential
- CGM (continuous glucose monitoring) แนะนำในผู้ป่วยใช้ insulin + บางกรณี non-insulin

## 📚 Key Recommendations (Graded)

### Class I (Strong recommendation) — "Should"

**1. Screening**
- Screen T2DM ทุกคนอายุ ≥35 ปี, repeat every 3 years if normal
- Screen in overweight adults (BMI ≥25, or ≥23 in Asian) with ≥1 risk factor
- Prediabetes: intensive lifestyle + consider metformin if high risk

**2. Glycemic targets**
- HbA1c < 7% for most non-pregnant adults
- Individualize based on age, comorbidities, hypoglycemia risk

**3. Cardiovascular disease management**
- T2DM + ASCVD → **GLP-1 RA with proven CVD benefit OR SGLT2i with proven CVD benefit**, independent of HbA1c
- T2DM + HF → **SGLT2i**, Class I evidence
- T2DM + CKD → **SGLT2i** + consider GLP-1 RA

**4. Blood pressure**
- Target <130/80 mmHg for most adults with DM
- First-line: ACEi/ARB (esp. with albuminuria)

**5. Lipids**
- Moderate-intensity statin age 40-75
- High-intensity statin if ASCVD or 10-y risk ≥20%

### Class IIa (Moderate) — "Reasonable"

**6. Obesity-focused care**
- Consider **tirzepatide or semaglutide** for weight loss ≥15% → diabetes remission possible
- Bariatric surgery if BMI ≥35 + poor glycemic control

**7. CGM**
- Consider in T2DM on insulin (Class I for T1DM)
- Consider in T2DM without insulin if motivated + financially feasible

### Class IIb (Weak) — "May consider"

**8. Technology**
- Automated insulin delivery for selected T1DM / intensive-therapy T2DM

### Class III (Against)

- Avoid sulfonylurea as first-line when SGLT2i/GLP-1 RA available
- Avoid thiazolidinedione in HF

## 🔄 Changes from 2024

### NEW in 2025
- ✅ Obesity section expanded — GLP-1 RA tirzepatide data integrated
- ✅ Finerenone recommended for CKD + albuminuria (Class IIa)
- ✅ New framework for "diabetes remission" definition (HbA1c <6.5% × 3 months off medication)

### CHANGED
- 🔄 Metformin no longer "always first-line" — can start with GLP-1 RA/SGLT2i directly in high-CV-risk
- 🔄 HbA1c target more individualized (frailty scale integrated)

### REMOVED
- ❌ Sliding scale insulin monotherapy (Class III — explicitly discouraged)

## 💡 Practical Pearls

### Decision algorithm (simplified)

```
T2DM Diagnosis
    │
    ├─ ASCVD / HF / CKD present?
    │       ├─ YES → GLP-1 RA OR SGLT2i (with CV benefit)
    │       │          ± Metformin
    │       └─ NO
    │           │
    │           ├─ Need major weight loss?
    │           │       ├─ YES → Tirzepatide / Semaglutide
    │           │       └─ NO → Metformin
    │           │
    │           └─ HbA1c not at goal after 3-6 mo?
    │                   → Add GLP-1 RA / SGLT2i / SU / DPP-4i / Insulin
    │
    └─ Monitor: HbA1c q3mo, BP, lipids, CKD, retina yearly
```

### Individualizing HbA1c target

| Patient profile | Target |
|-----------------|--------|
| Young, no comorbidity, long life expectancy | <6.5% |
| Most adults | <7% |
| Older, limited life expectancy, hypoglycemia risk | <8% |
| Frail, end-stage illness | Avoid symptoms, no target |

### Red flags requiring specialist
- Unable to achieve HbA1c <8% despite 2-3 agents
- Frequent hypoglycemia
- CKD stage 4-5
- Advanced retinopathy
- Pregnancy / pregnancy plan

## ⚠️ Limitations & Caveats

- **Cost of GLP-1 RA / SGLT2i** — not affordable for all patients in Thailand (check NHSO coverage)
- **Generalizability** — ADA guideline US-centric, some recommendations may not fit Thai context (cost, drug availability)
- **Drug interactions** — always check Lexicomp before adding (esp. statin + fibrate, SGLT2i + loop diuretic)
- **Pregnancy exception** — separate guideline applies for GDM
- **Latest evidence may supersede** — check for 2026 update Jan 2026

### Thai-specific considerations
- สปสช. coverage สำหรับ SGLT2i/GLP-1 RA จำกัด — ตรวจ eligibility
- ต้อง concur กับ Thai DM guideline (ราชวิทยาลัยอายุรแพทย์) ถ้าทำงานใน public sector
- ราคายา GLP-1 RA ยังแพงมาก — ต้องปรึกษา financial aspect กับผู้ป่วย

## 🧠 Key Takeaways

1. **Paradigm shift:** T2DM + CV/HF/CKD → start GLP-1 RA or SGLT2i (can be first-line, not after metformin failure)
2. **Obesity-centric:** ≥15% weight loss via tirzepatide/bariatric surgery can lead to remission
3. **Individualized HbA1c** — stop chasing 7% in frail elderly; focus on avoiding hypoglycemia
