# Psychologist Helper

> Clinical assistant สำหรับนักจิตวิทยา — session note / CBT worksheet / case conceptualization / DSM-5 reference

## ⚠️⚠️⚠️ DISCLAIMER สำคัญมาก ⚠️⚠️⚠️

**Skill นี้เป็น เครื่องมือช่วยค้นคว้าและจัดระเบียบข้อมูล สำหรับ นักจิตวิทยาคลินิก / จิตแพทย์ / นักจิตบำบัดที่มีใบประกอบวิชาชีพเท่านั้น**

- ❌ **ห้ามใช้แทนการวินิจฉัยของผู้ชำนาญ**
- ❌ **ห้ามใช้แทนการบำบัดจริง**
- ❌ **ห้ามให้คำแนะนำทางจิตวิทยาแก่ผู้ป่วยโดยตรงจาก output ของ skill**
- ❌ **ห้ามใช้สำหรับ self-diagnosis** (ผู้ป่วยใช้เองโดยไม่มีผู้ชำนาญ)
- ✅ ใช้ได้เพื่อ: จัด session note, refresh ทฤษฎี, เตรียม case conceptualization

**การใช้ผิดวัตถุประสงค์อาจก่อให้เกิดอันตรายต่อชีวิต**

## ⚡ ใช้ยังไง

```
/psychologist-helper
```

(Skill จะถาม confirm สถานะผู้ชำนาญก่อนทำงาน)

## 🎯 ทำอะไรได้บ้าง (สำหรับผู้ชำนาญ)

- ✅ Session note template (SOAP / DAP format)
- ✅ CBT worksheets — thought record, behavioral activation, cognitive distortions
- ✅ Case conceptualization (Beck model / behavioral model)
- ✅ DSM-5-TR / ICD-11 criteria reference
- ✅ DBT skills handouts (mindfulness / distress / emotion / interpersonal)
- ✅ ACT worksheets (defusion / values / committed action)
- ✅ Teaching material สำหรับ supervisee

## 💡 Use cases ที่เหมาะสม

- นักจิตวิทยาคลินิก เตรียม session note
- จิตแพทย์ refresh therapy framework
- Supervisor เตรียมสอน intern
- Psychology student (ใต้ supervision)

## ❌ Use cases ที่ไม่เหมาะสม (ห้ามใช้)

- Self-therapy / self-diagnosis
- ให้ output แก่ผู้ป่วยโดยตรงแทน therapy
- ตัดสิน diagnosis จาก output โดยไม่มี clinical interview
