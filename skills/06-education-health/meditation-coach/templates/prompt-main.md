# Prompt Patterns สำหรับ Meditation Coach

## 1. Pattern: Guided Meditation Script (Standard)

```
Length: <5 / 10 / 15 / 20 / 30 min>
Technique: <breathing / body scan / loving-kindness / mindfulness>
Audience: <beginner / experienced / group / individual / child>
Theme: <calm / focus / sleep / compassion / gratitude>
Position: <sit / lie down / walk>

Script structure:
1. Welcome + settling (10-15% of time)
2. Settling into body (5-10%)
3. Main practice (60-70%)
4. Integration / silence (10-15%)
5. Return / closing (5-10%)

Format:
- [Time stamp]
- [Pause indicators in seconds]
- "Verbatim text"
- (Voice quality: soft, steady, slow)

Tone:
- Invitation > command ("you might notice" not "notice")
- Trauma-sensitive ("if comfortable")
- Slow pacing
```

## 2. Pattern: Breathing Technique

```
Technique: <Box / 4-7-8 / Belly / Ujjayi / Bhramari>

Output:
- Origin / who developed
- When to use (anxiety / sleep / focus)
- When NOT to use (pregnancy, BP, heart)
- Step-by-step
- Round count + duration
- Common mistakes
- Progression
- Combined with other practices
```

## 3. Pattern: Sleep Meditation

```
Length: 20-30 min
Position: lying in bed

Special notes:
- Voice tone: very slow + soft
- Pace 30-50% slower than awake meditation
- End with intentional drift (not "open eyes")
- Optional: rain/ocean sounds in background
- Avoid: too much instruction, energizing visualization

Script:
- Long settling (3-5 min)
- Long body scan (15-20 min, 1-2 min per region)
- Drift transition (5 min)
- (No closing — let asleep)
```

## 4. Pattern: Anxiety Reset (Acute)

```
Length: 5 min (must be short — anxious person can't sit long)

Combine 3 techniques:
1. Grounding 5-4-3-2-1 (sensory anchor) — 2 min
2. Box breathing — 2 min
3. Affirmation — 1 min

Tone: gentle but slightly confident ("you're safe right now")
Avoid: prolonged silence (triggers more anxiety)
```

## 5. Pattern: Loving-Kindness (Metta) Multi-Phase

```
Length: 15-20 min
Format: 5 phases × 3-4 min each

Phrases (4 standard):
- "May I/you be happy"
- "May I/you be healthy"
- "May I/you be safe"
- "May I/you live with ease"

Phases:
1. Self
2. Loved one
3. Neutral person
4. Difficult person (mild — not enemy)
5. All beings

Cue + pause + repeat phrases × 3-5 each phase
```

## 6. Pattern: Body Scan (Detailed)

```
Length: 20-30 min

Direction: toes to head (relaxation) OR head to toes (energizing)

Per region (~1-2 min):
- Notice sensations
- No judgment
- No need to change
- Acknowledge if tense / numb
- Move on

Regions: toes, feet, ankles, calves, knees, thighs, hips, abdomen, lower back, chest, upper back, shoulders, arms, hands, neck, face, head

End: whole body awareness
```

## 7. Pattern: Group/Workplace Meditation

```
Setting: office, training room, online video
Length: 5-10 min (executives can't sit longer)

Considerations:
- People wear work clothes (no full lying down)
- Some can't close eyes (workplace dynamics)
- Time-sensitive (back to work)
- Diverse beliefs (avoid religious framing)

Format:
- Brief intro (1 min)
- Practice (5-7 min)
- Quick share (optional, 1 min)
- Return (1 min)

Tone: professional yet warm
```
