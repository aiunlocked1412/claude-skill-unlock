# Meditation Coach

> โค้ชสมาธิและ mindfulness — guided meditation script + breathing technique + theme practice

## ⚠️ คำเตือนด้านสุขภาพ

Meditation ดีต่อ wellbeing แต่ก่อนเริ่มควรปรึกษาผู้เชี่ยวชาญ ถ้า:
- 🧠 มีประวัติ trauma รุนแรง / PTSD (อาจ trigger)
- 😔 Depression / anxiety รุนแรง (อยู่ใต้การดูแลแพทย์)
- 🌀 Dissociation / depersonalization
- 💊 Psychotic disorder (กำลังรักษา)

**Meditation ไม่ใช่ therapy** — ถ้ามีปัญหาจริงจัง พบนักจิตวิทยาหรือจิตแพทย์
- สายด่วนสุขภาพจิต **1323** (ฟรี 24 ชม.)

## ⏱️ ความยาว

- ⚡ **5 min** — quick reset, anxiety, focus
- 🌅 **15 min** — daily practice
- 🌙 **30 min** — deep practice, sleep

## 🎯 Techniques ที่รองรับ

- 🫁 **Breathing** — Box (4-4-4-4), 4-7-8, Belly, Ujjayi
- 🧠 **Focused attention** — anchor practice
- 🌊 **Body scan** — head to toe awareness
- 💗 **Loving-kindness (Metta)** — for self / others
- 🌫️ **Mindfulness of thoughts**
- 😌 **Anxiety relief** — 5-4-3-2-1 grounding
- 😴 **Sleep meditation**
- 🎯 **Focus boost** — pre-work
- 🙏 **Gratitude practice**
- 🌳 **Visualization** — safe place
- 📿 **Walking meditation**
- 🪷 **Anapanasati** (Buddhist breathing)

## 💡 จุดเด่น

- **Pause time ชัด** — `[pause 30 sec]` ใน script
- **Trauma-sensitive** — เสนอทางเลือก "ลืมตาได้"
- **Bilingual** — Thai natural + Pali/Sanskrit ที่ relevant
- **Pacing เหมาะ** — แตกต่างกันตามจุดประสงค์

## 👥 เหมาะกับใคร

- คนทั่วไปที่อยากเริ่มฝึกสมาธิ
- ครูสอนสมาธิ / mindfulness coach
- Yoga teacher (ใส่ meditation ในคลาส)
- HR ที่จัด wellness program
- Therapist (เสริม therapy)
- ผู้ปกครอง (สอนลูก)
