# Fitness Trainer Pro

> Workout program 4 สัปดาห์ ที่มี periodization + progression + exercise library ครบ

## ⚡ ใช้ยังไง

```
/fitness-trainer-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Full program 4 สัปดาห์ (Linear Periodization)
- ✅ Splits: Full-body / Upper-Lower / PPL / Bro split
- ✅ Home (ไม่มีอุปกรณ์) / Gym program
- ✅ Exercise library — form cues + common mistakes + alternatives
- ✅ Tracking sheet (Excel-ready)
- ✅ Deload / recovery week
- ✅ Hybrid strength + cardio / HIIT

## 💡 Use cases

- เพิ่มกล้าม (hypertrophy)
- เพิ่มแรง (strength / powerlifting)
- ลดไขมัน (fat loss)
- วิ่ง / กีฬา (sport-specific)
- Rehab จากบาดเจ็บ (เบา)
- Home workout ไม่มีอุปกรณ์

## ⚠️ DISCLAIMER

**เครื่องมือนี้ให้คำแนะนำทั่วไป ไม่ใช่คำแนะนำทางการแพทย์**

- ❌ ไม่แทน certified personal trainer
- ❌ ไม่รักษาอาการบาดเจ็บ

**ควรปรึกษาแพทย์ก่อนเริ่มถ้า:**
- มีโรคประจำตัว (ความดัน, หัวใจ, เบาหวาน)
- ตั้งครรภ์
- ประวัติบาดเจ็บรุนแรง (หลัง, เข่า, ไหล่)
- ผู้สูงอายุ 65+
- ไม่เคยออกกำลังกาย > 5 ปี

## 🎓 Tips

- บอก **level + สถานที่ + อุปกรณ์** → program แม่นขึ้น
- ใช้คู่ `/nutritionist-pro` → workout + meal = ผลลัพธ์เร็ว
- Deload ทุก 4-6 สัปดาห์ สำคัญ อย่าข้าม
