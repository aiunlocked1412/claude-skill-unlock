# Prompt Patterns สำหรับ Fitness Trainer Pro

## 1. Pattern: Full Program 4 Weeks

```
ออกแบบ workout program 4 สัปดาห์ สำหรับ:
- Level: <beginner/intermediate/advanced>
- Split: <full-body/upper-lower/PPL/bro>
- สถานที่: <home/gym>
- อุปกรณ์: <barbell/dumbbell/bodyweight/ฯลฯ>
- วัน/สัปดาห์: <3-6 วัน>
- เป้าหมาย: <strength/hypertrophy/fat-loss>
- ข้อจำกัด: <บาดเจ็บ?>

Periodization 4 weeks:
- W1: Baseline (RPE 7, 70% 1RM)
- W2: Overload (+2.5 kg)
- W3: Peak (+2.5 kg, RPE 9)
- W4: Deload (60%, RPE 6)

ทุกวันต้องระบุ:
- Warm-up (5-10 min, specific to movement)
- Main exercises (3-5 ท่า, compound ก่อน)
- Accessory (2-4 ท่า, isolation)
- Cool-down / stretching

ทุกท่าต้อง:
- Set × Rep × Weight (หรือ bodyweight)
- RPE target
- Rest interval
- Form cue 1 ข้อ
```

## 3. Pattern: Home Workout

```
ออกแบบ home workout <X> สัปดาห์ ไม่มีอุปกรณ์:

ใช้เฉพาะ:
- Bodyweight (push-up, pull-up, squat)
- Resistance band (optional)
- Dumbbell / kettlebell ขนาด 1-2 ตัว (optional)

Duration: 30-45 นาที / session

Structure:
1. Warm-up 5 min (jumping jack, arm circle, hip rotation)
2. Circuit 3 rounds (4-5 exercises, 40 sec work / 20 sec rest)
3. Strength block (3 exercises × 3 sets)
4. Core finisher 5 min
5. Cool-down / stretch 5 min

Progression:
- W1-2: 3 rounds
- W3-4: 4 rounds หรือ harder variation
- W5+: ใช้ weighted backpack / resistance band
```

## 4. Pattern: Fat Loss Program

```
ออกแบบ fat loss program <X> สัปดาห์:

Principles:
- Strength 3x/week (ไม่ใช่ cardio only!)
- HIIT 2x/week (20-30 min)
- LISS cardio 1-2x/week (เดิน 45 min)
- Rest 1 day

Strength focus:
- Compound lifts (big 5)
- 4-8 rep range (strength) หรือ 8-12 (hyper)
- Progressive overload

HIIT template:
- 5 min warm-up
- 20 min: 30 sec all-out / 30 sec rest × 20 rounds
- 5 min cool-down

Diet recommendation:
- Caloric deficit 300-500 kcal
- Protein 2.0-2.4 g/kg (รักษากล้าม)
- Refer to /nutritionist-pro

Tracking:
- Weight: สัปดาห์ละ 1 ครั้ง
- Measurement: สัปดาห์ละ 1 ครั้ง (รอบเอว / สะโพก / ต้นขา)
- Photo: 2 สัปดาห์ครั้ง
```

## 5. Pattern: Deload Week

```
ออกแบบ deload week หลัง 4-6 สัปดาห์หนัก:

Principles:
- ลดปริมาตร (volume) 40-50%
- ลดความหนัก 30-40%
- คง intensity (RPE 6-7)
- ยังต้องมาออก ไม่ใช่ rest สนิท

Structure:
- 2-3 sets แทน 4-5
- 50% ของน้ำหนักสัปดาห์ peak
- Reps เดิม
- เน้น form + mobility

Extra:
- Sleep 8+ hours
- Stretching 20 min daily
- Walk/swim/yoga
- Eat at maintenance (ไม่ bulk/cut)

Return week (หลัง deload):
- Start at 95% ของสัปดาห์ peak
- Push new PR
```
