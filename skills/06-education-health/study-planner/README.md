# Study Planner

> โค้ชเทคนิคการเรียนรู้ — Spaced Repetition, Active Recall, Feynman, Exam Plan, Schedule

## ⚡ ใช้ยังไง

```
/study-planner
```

## 🎯 ทำอะไรได้บ้าง

- ✅ สร้างแผนการเรียน (study schedule + timeline จากเวลาจริง)
- ✅ Spaced Repetition System (Anki setup + review intervals)
- ✅ Active Recall Methods (blank paper, self-testing, Q&A)
- ✅ Feynman Technique (ฝึก concept ที่ยากให้เข้าใจจริง)
- ✅ Exam Prep Plan (30/60/90 วัน + subject prioritization)
- ✅ Flashcard System (Anki / physical card workflow)
- ✅ Focus & Productivity (Pomodoro, deep work blocks)

## 💡 Use Cases

- ม.ปลายเตรียมสอบ TCAS มีหลายวิชา ไม่รู้จะเริ่มตรงไหน
- นักศึกษามีสอบ 2 สัปดาห์ อยากวางแผนให้ครอบคลุม
- ผู้เรียน online course ที่ดูเสร็จแต่จำไม่ได้เลย
- Professional เตรียมสอบ certification (CPA, CFA, IT Cert)
- ใครก็ตามที่ "เรียนเยอะ แต่สอบไม่ผ่าน"

## 🎓 Tips

- บอก **วันสอบ + เวลาที่มีจริง** → แผนทำได้จริง ไม่ over-plan
- ระบุ **จุดอ่อน** ที่รู้อยู่แล้ว → ช่วย prioritize ก่อนหลัง
- อัปเดต progress รายสัปดาห์ → ปรับแผนได้ทัน
