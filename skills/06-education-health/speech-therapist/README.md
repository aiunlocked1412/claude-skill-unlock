# Speech Therapist

> ผู้ช่วยนักอรรถบำบัด — Articulation, Stuttering, Language Milestone, Session Plan, Parent Coaching (เพื่อการศึกษาเท่านั้น)

## ⚡ ใช้ยังไง

```
/speech-therapist
```

## 🎯 ทำอะไรได้บ้าง

- ✅ พัฒนาการภาษาตามวัย (milestone 0-6 ปี) + Red Flag
- ✅ Articulation Disorders (เสียงผิด, Age of Acquisition ไทย)
- ✅ Stuttering / Fluency (Lidcombe principle, กิจกรรมที่บ้าน)
- ✅ Language Development (ล่าช้า, ASD, สองภาษา)
- ✅ Swallowing & Feeding (IDDSI texture levels)
- ✅ Session Plan Template สำหรับนักอรรถบำบัด
- ✅ Parent Coaching — กิจกรรมกระตุ้นพัฒนาการที่บ้าน

## ⚠️ ข้อจำกัดสำคัญ

**เนื้อหาเพื่อการศึกษาและสนับสนุนผู้ดูแลเท่านั้น — ไม่ใช่การวินิจฉัยหรือรักษา หากเด็กมีอาการล่าช้า ต้องพบนักอรรถบำบัดที่มีใบอนุญาตโดยตรง**

## 💡 Use Cases

- พ่อแม่กังวลลูก 2 ขวบพูดช้า อยากรู้ว่าปกติไหม
- นักอรรถบำบัดต้องการ session plan template
- ครูปฐมวัยอยากรู้ milestone ภาษาตามวัย
- พ่อแม่เด็กติดอ่างอยากรู้วิธีช่วยที่บ้าน
- นักศึกษาอรรถบำบัดทบทวน articulation errors
