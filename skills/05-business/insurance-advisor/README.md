# Insurance Advisor

> AI ผู้ช่วยวิเคราะห์ความต้องการประกันสำหรับคนไทย — ชีวิต, สุขภาพ, รถ, บ้าน

## ⚡ ใช้ยังไง

```
/insurance-advisor
```

## 🎯 ทำอะไรได้บ้าง

- ✅ ประกันชีวิต — DIME analysis คำนวณทุน
- ✅ ประกันสุขภาพ — IPD/OPD/CI compare
- ✅ ประกันรถยนต์ — ชั้น 1/2+/3+/3 เลือกอย่างไร
- ✅ ประกันบ้าน/ทรัพย์สิน — Fire + flood
- ✅ ประกันเดินทาง — Schengen + อเมริกา
- ✅ ประกันอุบัติเหตุ (PA)
- ✅ Term vs Whole Life — แบบไหนคุ้ม
- ✅ Full Insurance Audit — ตรวจสุขภาพประกันรวม

## 💡 Use cases

- คนเริ่มทำงานยังไม่เคยซื้อประกัน
- ครอบครัวใหม่ที่มีลูก — ต้องมีกี่ทุน
- คนที่ตัวแทนเสนอ Whole Life — เช็คว่าคุ้มไหม
- ซื้อรถใหม่ต้องเลือกชั้นไหน
- ผ่อนบ้านต้องมีประกันอะไรบ้าง
- กำลังจะไป Schengen ต้องเตรียม €30,000 ไหม
