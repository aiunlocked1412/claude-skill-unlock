# Prompt Formula — Insurance Advisor

## Template คำถาม context

```
อายุ: <ปี>
เพศ: <ช/ญ>
อาชีพ: <Class 1: ออฟฟิศ / Class 2: outdoor / Class 3: high risk>
สถานะ: <โสด / สมรส / บุตร X คน>
รายได้/ปี: <บาท>
หนี้สิน: <บัตร / รถ / บ้าน + ยอด>
โรคประจำตัว: <มี/ไม่มี + รายละเอียด>
ประกันที่มี: <ทุน + ประเภท>
งบเบี้ย/ปี: <บาท หรือ % ของรายได้>
```

## DIME Recipe

```
คำนวณทุนประกันชีวิตด้วย DIME:
- D (Debt): หนี้สินทั้งหมด
- I (Income): รายได้/ปี × 5-10 (ทดแทนครอบครัว)
- M (Mortgage): หนี้บ้านคงเหลือ
- E (Education): ค่าเรียนลูก (1.5-3 ลบ./คน จนจบ ป.ตรี)
รวม = Sum Insured ที่แนะนำ
```

## Term vs Whole Life

```
เปรียบเทียบ Term vs Whole Life:
- Term (10/20/30 ปี): เบี้ยถูก, ไม่มีคืน, เหมาะวัยมีหนี้/บุตร
- Whole Life: เบี้ยแพง 5-10 เท่า, มีมูลค่าเวนคืน, เหมาะ wealth transfer
Rule: 90% ของคนใช้ Term ก็พอ
```

## Health Insurance Recipe

```
ประเมินประกันสุขภาพ:
- โรงพยาบาลเป้าหมาย (รัฐ / เอกชน A / B / C)
- ค่าห้อง: 5,000-10,000/วัน (เอกชน BKK)
- IPD Sum Insured: ≥ 1 ลบ./ปี
- OPD: nice to have ถ้าใช้บ่อย
- CI (โรคร้ายแรง): 1-3 ลบ.
- Co-pay 20% / Deductible 30k → ลดเบี้ย ~30-45%
```

## Auto Insurance Recipe

```
แนะนำชั้นประกันรถ:
- รถใหม่ < 5 ปี + มูลค่า > 500k → ชั้น 1
- รถ 200-500k → 2+ / 3+
- รถเก่า > 10 ปี → 3 + พ.ร.บ.
- น้ำท่วมพื้นที่ → ต้องมีชั้น 1 (แค่ 2+/3+ ไม่ครอบ)
- เปรียบเทียบ 3 บริษัทขึ้นไป
```

## Tips

- คำนวณเบี้ยรวมไม่ควรเกิน 10-15% ของรายได้
- IPD ต้องมีเป็นอันดับแรก ก่อนทุกอย่าง
- ถ้ามีประกันสังคม + ประกันบริษัทแล้ว → ลดทุนประกันส่วนตัวได้
- เด็ก/ผู้สูงอายุ — เบี้ยพุ่งสูง ต้องคำนวณตอนสมัครหนุ่ม
- ทุก output ต้องมี disclaimer + แนะนำปรึกษา คปภ.
