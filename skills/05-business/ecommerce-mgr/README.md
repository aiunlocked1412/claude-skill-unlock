# E-commerce Manager

> AI ผู้ช่วยจัดการร้านออนไลน์ — Shopee/Lazada/TikTok Shop/own site, conversion, fulfillment, return

## ⚡ ใช้ยังไง

```
/ecommerce-mgr
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Conversion Funnel Analysis (Visit → Cart → Buy)
- ✅ Cart Abandonment (benchmark 70%)
- ✅ Pricing Strategy (cost+/competitor/value)
- ✅ Fulfillment Strategy (self/3PL/FBS)
- ✅ Return Policy & Process
- ✅ Multi-channel Strategy (Shopee/Lazada/TikTok/own)
- ✅ Review & Rating Management
- ✅ Customer Lifetime Value (CLV) + Retention
- ✅ Promo Strategy (11.11, 12.12)
- ✅ Full Store Audit

## 💡 Use cases

- ร้าน Shopee CR 1% — อยากปรับให้ถึง 3%
- Cart abandonment 80% — ลด return ยังไง
- จัด fulfillment 300 order/วัน
- เลือก platform ใหม่ — TikTok Shop คุ้มไหม
- เตรียม 11.11 mega sale
- สร้าง loyalty program ใน Line OA
- Pricing สำหรับ multi-channel
