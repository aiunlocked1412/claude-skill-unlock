## Funnel Recipe

```
ตรวจ funnel:
Impressions → CTR % → Visits
Visits → ATC % → Cart
Cart → Checkout % → Checkout
Checkout → Purchase % → Order

CR overall = Visits / Purchase

Industry benchmark:
- CTR: 2-5%
- ATC: 5-15%
- Cart-Checkout: 50-70%
- Checkout-Purchase: 60-80%
- Overall CR: 2-3%

หาจุด drop ที่ใหญ่สุด → focus optimize
```

## Cart Abandonment Recipe

```
Cart abandonment % = (cart - purchase) / cart × 100
Benchmark: ~70% (industry avg)

Top reasons:
1. Extra cost (shipping)
2. Force account creation
3. Long checkout
4. Trust issue
5. Slow delivery
6. Unclear return

Fix actions:
- Show shipping early
- Guest checkout
- < 4 step checkout
- Trust badges
- Express delivery option
- Clear return policy

Recovery email: 1hr / 24hr / 72hr
Recovery rate target: 10-20%
```

## Pricing Recipe

```
Cost-plus:
Price = Cost × (1 + markup)
Markup by category:
- FMCG: 30-50%
- Fashion: 100-200%
- Electronics: 15-30%
- Beauty: 100-300%

Multi-channel pricing:
- Shopee/Lazada base price
- Own site -5 to -10%
- TikTok same as Shopee
ระวัง platform policy
```

## Fulfillment Recipe

```
Volume estimation:
- < 100 order/day → self
- 100-500 → 3PL (Flash, Kerry, J&T)
- > 500 → FBS or own warehouse

KPI:
- Processing < 24h
- Ship BKK 1-2 day, ตจว. 2-4 day
- On-time > 95%
- Damage < 1%
```

## Return Recipe

```
Return rate by category:
- Electronics: 5-8%
- FMCG: 1-3%
- Fashion: 15-25%
- Beauty: 2-5%

Reduce return:
- Detailed size chart
- Multi-angle photos + video
- True-to-life photos (no over-edit)
- Pre-purchase Q&A
- Customer reviews with size feedback

Policy:
- Window: 7-14 days
- Customer pays return shipping (size) vs seller (defect)
- Restocking fee 0-15%
```

## Multi-channel Recipe

```
- Margin per channel (after commission, voucher)

```
