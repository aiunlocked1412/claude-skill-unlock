# Procurement Pro

> AI ผู้ช่วยจัดซื้อจัดจ้าง — RFP/RFQ, vendor scorecard, TCO, negotiation, INCOTERMS

## ⚡ ใช้ยังไง

```
/procurement-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ RFI/RFP/RFQ Template (เขียนเอกสาร)
- ✅ Vendor Selection & Scorecard (QCDFS+R)
- ✅ TCO Analysis (Total Cost of Ownership)
- ✅ Contract Negotiation Playbook (BATNA, 8 levers)
- ✅ Spend Analysis (Kraljic Matrix)
- ✅ INCOTERMS 2020 Guide (FOB/CIF/DDP)
- ✅ Procurement Process Map (P2P)
- ✅ Make-or-Buy Decision
- ✅ Vendor Onboarding Checklist
- ✅ Strategic Sourcing 7-step

## 💡 Use cases

- จะจัดซื้อ ERP / IT system → ต้อง RFP
- เลือก supplier วัตถุดิบ 5 ราย — เปรียบเทียบยังไง
- เครื่องจักรราคาแพงกว่า — แต่ค่าซ่อมถูก คุ้มไหม (TCO)
- เจรจาสัญญา supplier — มี leverage อะไรบ้าง
- นำเข้าจากจีน — FOB หรือ CIF ดีกว่า
- ตัดสินใจ make vs buy
