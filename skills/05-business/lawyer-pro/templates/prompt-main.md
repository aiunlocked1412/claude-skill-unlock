# Contract Templates Library — Lawyer Pro

## Context Questions

```
ประเภทเอกสาร: <สัญญาจ้าง / เช่า / NDA / ซื้อขาย / memo / research>
คู่สัญญา A (นายจ้าง/ผู้ให้เช่า):
  - ชื่อ
  - เลขบัตรปชช./ทะเบียนนิติบุคคล
  - ที่อยู่
คู่สัญญา B (ลูกจ้าง/ผู้เช่า):
  - ชื่อ
  - เลขบัตรปชช.
  - ที่อยู่
เงื่อนไขหลัก:
  - ค่าตอบแทน/ค่าเช่า
  - ระยะเวลา
  - ขอบเขตงาน/พื้นที่
  - ข้อพิเศษ (ถ้ามี)
```

## กฎหมายไทยที่เกี่ยวข้องบ่อย

### ประมวลกฎหมายแพ่งและพาณิชย์ (ปพ.)
- **มาตรา 537-571** — เช่าทรัพย์
- **มาตรา 575-586** — จ้างแรงงาน
- **มาตรา 587-607** — จ้างทำของ
- **มาตรา 649-656** — เช่าซื้อ
- **มาตรา 680-691** — ค้ำประกัน
- **มาตรา 702-746** — จำนอง

### พ.ร.บ.คุ้มครองแรงงาน 2541
- มาตรา 17 — ทดลองงาน ≤ 119 วัน
- มาตรา 30 — ลาพักร้อน ≥ 6 วัน/ปี
- มาตรา 32 — ลาป่วย ≥ 30 วัน/ปี (ได้ค่าจ้าง)
- มาตรา 41 — ลาคลอด 98 วัน
- มาตรา 118 — ค่าชดเชยเลิกจ้าง (ตามอายุงาน)
- มาตรา 123 — ค่าชดเชยพิเศษ (ย้ายที่ทำงาน, เทคโนโลยี)

### PDPA (พ.ร.บ.คุ้มครองข้อมูลส่วนบุคคล 2562)
- ต้องมี "ฐานการประมวลผล" (ยินยอม, สัญญา, ประโยชน์อันชอบธรรม ฯลฯ)
- Data subject rights: เข้าถึง, ลบ, คัดค้าน
- โทษสูงสุด: ปรับ 5 ลบ. + จำคุก 1 ปี

### ประมวลรัษฎากร
- หัก ณ ที่จ่าย: บริการ 3%, เช่า 5%, โฆษณา 2%
- VAT: 7% ถ้ารายได้ > 1.8 ลบ./ปี

## Template 1: สัญญาจ้างพนักงานประจำ (Frame)

```
สัญญาจ้างแรงงาน

ทำขึ้นระหว่าง
1. [บริษัท ABC จำกัด] เลขทะเบียน ... ที่อยู่ ... ("นายจ้าง")
2. [นาย/นาง/นางสาว XXX] เลขปชช. ... ที่อยู่ ... ("ลูกจ้าง")

ทั้งสองฝ่ายตกลงกัน ดังนี้

ข้อ 1 ตำแหน่งและหน้าที่
ลูกจ้างตกลงเข้าทำงานในตำแหน่ง [...] โดยมีหน้าที่ตามที่นายจ้างมอบหมาย

ข้อ 2 ระยะเวลา
- เริ่มทำงานวันที่ [วัน/เดือน/ปี]
- ไม่จำกัดระยะเวลา
- ทดลองงาน 119 วัน ตาม มาตรา 17 พ.ร.บ.คุ้มครองแรงงาน 2541

ข้อ 3 ค่าจ้าง
- เงินเดือน XX,XXX บาท จ่ายวันที่ [25 ของทุกเดือน]
- โบนัสตามผลประกอบการ

ข้อ 4 วันและเวลาทำงาน
- วันจันทร์-ศุกร์ เวลา 9:00-18:00 (พัก 12:00-13:00)
- สัปดาห์ละ 40 ชั่วโมง

ข้อ 5 วันหยุดและวันลา (ตาม พ.ร.บ.คุ้มครองแรงงาน)
- วันหยุดประจำสัปดาห์: เสาร์-อาทิตย์
- วันหยุดตามประเพณี: ≥ 13 วัน/ปี
- ลาพักร้อน: 10 วัน/ปี (หลังทำงานครบ 1 ปี)
- ลาป่วย: 30 วัน/ปี (ได้รับค่าจ้าง)
- ลากิจ: 3 วัน/ปี (ได้รับค่าจ้าง)

ข้อ 6 สวัสดิการ
- ประกันสังคม (นายจ้างสมทบ 5%)
- ประกันสุขภาพกลุ่ม (หลังผ่านทดลอง)
- โบนัสประจำปี (ตามผลประกอบการ)

ข้อ 7 ทรัพย์สินทางปัญญา
ผลงานที่ลูกจ้างสร้างขึ้นในหน้าที่ เป็นทรัพย์สินของนายจ้างทั้งหมด

ข้อ 8 การรักษาความลับ
ลูกจ้างจะไม่เปิดเผยข้อมูลลับของนายจ้างทั้งในระหว่างและหลังพ้นสภาพ

ข้อ 9 การเลิกสัญญา
- ฝ่ายใดประสงค์เลิก แจ้งล่วงหน้า ≥ 30 วัน
- นายจ้างเลิกตามมาตรา 118 — จ่ายค่าชดเชยตามอายุงาน
- เลิกโดยมีเหตุตามมาตรา 119 — ไม่ต้องจ่ายชดเชย

ข้อ 10 กฎหมายที่ใช้บังคับ
สัญญานี้ใช้กฎหมายไทย ข้อพิพาทให้ขึ้นศาลแรงงาน

ลงชื่อ ณ วันที่ [...]

(ลายมือชื่อนายจ้าง) ................    (ลายมือชื่อลูกจ้าง) ................
(พยาน 1) ................            (พยาน 2) ................
```

## Template 2: NDA (ย่อ)

```
ข้อตกลงรักษาความลับ (Non-Disclosure Agreement)

ทำขึ้นระหว่าง [Party A] และ [Party B]

1. นิยาม "ข้อมูลลับ"
ข้อมูลทางธุรกิจ, เทคนิค, การเงิน, ลูกค้า ที่เปิดเผยระหว่างกัน

2. หน้าที่
- ไม่เปิดเผยข้อมูลลับต่อบุคคลที่สาม
- ไม่ใช้ข้อมูลเพื่อประโยชน์นอกเหนือจุดประสงค์
- ใช้ความระมัดระวัง ≥ ข้อมูลลับของตนเอง

3. ข้อยกเว้น
- ข้อมูลที่เผยแพร่สาธารณะแล้ว
- ได้รับจากบุคคลที่สามโดยชอบ
- กฎหมายบังคับให้เปิดเผย

4. ระยะเวลา
ผูกพัน [3 ปี] นับจากวันเซ็น

5. บทลงโทษ
ผู้ฝ่าฝืนชดใช้ค่าเสียหายที่เกิดขึ้นจริง + ค่าเสียหายเชิงลงโทษ [...]

6. กฎหมายใช้บังคับ
กฎหมายไทย — ศาลจังหวัดที่ตั้งสำนักงานใหญ่

ลงชื่อ ............................
```

## Template 3: สัญญาเช่า (ย่อ)

```
สัญญาเช่า [บ้าน/คอนโด]

1. คู่สัญญา
ผู้ให้เช่า: ... เลขปชช. ...
ผู้เช่า: ... เลขปชช. ...

2. ทรัพย์ที่เช่า
[รายละเอียดที่อยู่ + พื้นที่]

3. ระยะเวลา
[วันเริ่ม] ถึง [วันสิ้นสุด] รวม [X] เดือน/ปี

(ระยะเวลา > 3 ปี ต้องจดทะเบียนที่กรมที่ดิน)

4. ค่าเช่า
- ค่าเช่า XX,XXX บาท/เดือน
- จ่ายวันที่ 5 ของทุกเดือน
- มัดจำ 2 เดือน + ล่วงหน้า 1 เดือน = XX,XXX

5. ค่าใช้จ่าย
- ค่าน้ำ, ไฟ, ส่วนกลาง — ผู้เช่าจ่าย
- ภาษีโรงเรือน — ผู้ให้เช่าจ่าย

6. ข้อห้าม
- ไม่ sub-lease
- ไม่เลี้ยงสัตว์ (หรือตามตกลง)
- ไม่ใช้ทำธุรกิจผิดกฎหมาย

7. การบอกเลิก
- แจ้งล่วงหน้า 30 วัน
- ถ้าเลิกก่อนกำหนด: [เงื่อนไขคืนมัดจำ]

8. อ้างอิงกฎหมาย
ปพ. มาตรา 537-571

ลงชื่อ ............................
พยาน 1, 2
```

## Template 4: สัญญาจ้างทำของ (Freelance/Contractor)

```
สัญญาจ้างทำของ

1. คู่สัญญา
ผู้ว่าจ้าง: ...
ผู้รับจ้าง: ... (บุคคลธรรมดา — หัก ณ ที่จ่าย 3%)

2. ขอบเขตงาน (Scope of Work)
[แจกแจงละเอียดสิ่งที่ต้องส่งมอบ]

3. ระยะเวลา + Milestones
- Milestone 1: [วันที่] → ได้ค่าจ้าง XX%
- Milestone 2: [วันที่] → XX%
- Final: [วันที่] → XX%

4. ค่าจ้าง
- รวม XXX,XXX บาท (ยังไม่รวม VAT)
- หัก ณ ที่จ่าย 3% = XXX บาท
- ชำระจริง XXX บาท

5. Revision rounds
- ผู้รับจ้างยอมแก้ไขได้ [2] รอบ ฟรี
- รอบที่ 3+ คิดค่าเพิ่ม [1,500 บาท/รอบ]

6. Intellectual Property
- งานส่งมอบเป็นลิขสิทธิ์ของผู้ว่าจ้าง
- ผู้รับจ้างขอสิทธิ์ใช้งานเป็น portfolio

7. การบอกเลิก
- ผู้ว่าจ้างเลิกก่อน — จ่ายตาม milestone ที่ทำเสร็จ
- ผู้รับจ้างเลิกก่อน — คืนเงินส่วนที่ยังไม่ได้ทำ

8. กฎหมายใช้บังคับ
ปพ. มาตรา 587-607 — จ้างทำของ
```

## Tips

- ใส่วันที่ + สถานที่ทำสัญญาเสมอ
- ลายมือชื่อทั้งสองฝ่าย + พยาน 2 คน
- Clause ที่ขัดกฎหมาย = โมฆะ (เช่น สละสิทธิ์ค่าชดเชย)
- อ้างมาตราเฉพาะที่ apply (ไม่ใส่ทั้งประมวล)
- ภาษาไทยทางการ ไม่ใช้สำนวนเล่นๆ
