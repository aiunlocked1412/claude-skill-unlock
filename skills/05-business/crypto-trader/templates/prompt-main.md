# Prompt Formula — Crypto Trader

## Template คำถาม context

```
ประสบการณ์: <มือใหม่ / กลาง / เก๋า>
ทุน crypto: <บาท หรือ % ของ net worth>
% ของ net worth: <แนะนำ ≤ 5-10%>
Time horizon: <day / swing / HODL 4 ปี>
เหรียญสนใจ: <BTC / ETH / altcoin>
Risk tolerance: <รับ -50% ได้ไหม>
Experience DeFi: <เคย / ไม่เคย>
Wallet: <CEX only / hot / cold>
```

## Market Overview Recipe

```
ภาพรวมตลาด ณ <วันที่>:
- Total market cap: $X
- BTC market cap: $X (BTC.D = X%)
- Fear & Greed: X (extreme fear/fear/neutral/greed/extreme greed)
- 24h funding rate: avg X%
- Cycle stage: ปีที่ X หลัง halving
- Sentiment: Bull / Bear / Neutral
```

## Coin Analysis Recipe

```
วิเคราะห์ <COIN>:
1. Technical:
   - Trend (EMA 20/50/200)
   - Key S/R level
   - RSI, MACD signal
2. On-chain:
   - Active addresses trend
   - MVRV, NUPL, SOPR
   - Exchange flow
3. Tokenomics:
   - Supply schedule
   - Inflation/burn
   - Unlock cliff
4. Sentiment:
   - Funding rate
   - Social mention
สรุป: Bullish / Bearish / Neutral + Trade plan
```

## On-Chain Recipe (BTC focused)

```
วิเคราะห์ on-chain BTC:
- MVRV: X (> 3.7 = top, < 1 = bottom)
- NUPL: X (> 0.75 = euphoria, < 0 = capitulation)
- SOPR: X (> 1 = profit taking, < 1 = capitulation)
- Realized cap: $X
- LTH supply: X% (HODLing trend)
- Exchange balance: X BTC (rising/falling)
สรุป cycle position
```

## DeFi Yield Recipe

```
หา DeFi yield สำหรับ <amount> <asset>:
- Tier 1 (3-8% safe): Aave, Compound
- Tier 2 (8-20%): Curve LP, Lido stETH
- Tier 3 (20-100% risky): new farms
- Tier 4 (100%+ degen): หลีกเลี่ยง
แนะนำ allocation:
- 60% Tier 1
- 30% Tier 2
- 10% Tier 3 (max)
+ คำเตือน risk แต่ละ tier
```

## Trade Plan Recipe

```
Entry: $X (DCA range $X-$Y)
Stop Loss: $Z (-X%)
TP1: $A (+X%) — sell 30%
TP2: $B (+Y%) — sell 40%
TP3: $C (+Z%) — sell 30% (let it ride)

Position size: X% ของงบ crypto
R/R: 1:X
Timeframe: X เดือน

Exit triggers:
- Stop loss
- Top signal (MVRV > 3.7)
- Thesis pivot
```

## Tips

- DCA เสมอ — ไม่ lump sum
- ทุก output ต้องมี disclaimer + ห้ามใช้เงินกู้
- Position sizing > picking winners
- Self-custody ถ้า > $10k
- ภาษี crypto ในไทย — กำไรจากการขาย = เงินได้พึงประเมิน 40(8)
- หลีกเลี่ยง meme coin / new ICO
