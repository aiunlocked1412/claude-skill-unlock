# Crypto Trader

> AI ผู้ช่วยวิเคราะห์ตลาด crypto — BTC/ETH/altcoin, on-chain, DeFi yield, cycle analysis

## ⚡ ใช้ยังไง

```
/crypto-trader
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Market Overview (BTC dominance, fear/greed, total market cap)
- ✅ Coin Analysis (BTC/ETH/altcoin)
- ✅ On-Chain Metrics (MVRV, NUPL, SOPR, exchange flow)
- ✅ Altcoin Season Indicator
- ✅ DeFi Yield Strategy (4 tier risk-return)
- ✅ Cycle Analysis (4-year halving cycle)
- ✅ Trade Plan (entry, stop, TP)
- ✅ Concept Explainer (DCA, HODL, cold wallet)

## 💡 Use cases

- คนสนใจ BTC แต่ไม่รู้เริ่มยังไง
- มี portfolio crypto อยากรู้ตอนนี้ bull/bear
- อยาก yield USDT ที่ฝากไว้เฉยๆ
- กำลังหาจุด exit จาก altseason
- มือใหม่อยากเข้าใจ on-chain analysis
