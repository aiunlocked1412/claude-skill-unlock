### 3. Shipped
```

เช็คสถานะ → [link]
```

### 4. Out of Stock
```

แนะนำสินค้าใกล้เคียง:
🌟 [รุ่น A] - ราคา / feature
🌟 [รุ่น B] - ราคา / feature

```

### 5. Refund Approved
```
ขอโทษในความไม่สะดวกครั้งนี้นะคะ 🙏

ขอให้ส่ง:
1. เลขบัญชีธนาคาร
2. ชื่อเจ้าของบัญชี

โอนคืนภายใน 3 วันทำการค่ะ
```

### 6. Refund Rejected (สุภาพ + ชัด)
```

ทางเราได้ตรวจสอบเรื่องแล้ว น่าเสียดายที่กรณีนี้ไม่เข้าเงื่อนไขคืนเงิน
เพราะ [เหตุผลชัดเจน 1 บรรทัด]

แต่เราขอเสนอ [alternative — เช่น ส่วนลด 15% ในครั้งถัดไป / ซ่อมฟรี / เปลี่ยนรุ่นได้]
สะดวกรับข้อใด?
```

### 7. Complaint Response (3 ระดับ)

**ระดับต่ำ — ของมาช้า:**
```
ขอโทษที่ของช้ากว่าปกตินะคะ
เช็คให้แล้ว ขนส่งคาดว่าของถึงพรุ่งนี้ก่อน 18:00
ขอชดเชยด้วยส่วนลด 100 บาทสำหรับครั้งถัดไปค่ะ
```

**ระดับกลาง — ของผิด:**
```
ขอโทษจริงๆ นะคะที่ได้ของผิด 🙏
ทางเราจะส่งของที่ถูกต้องให้ภายในวันนี้ (EMS ส่งด่วน)
+ แถมของเล็กๆ น้อยๆ เป็นการขอบคุณที่อดทนรอ

ของผิดเก็บไว้ได้เลยค่ะ ไม่ต้องส่งคืน
```

**ระดับสูง — ชำรุด/Allergy:**
```
ขอโทษในเหตุการณ์ครั้งนี้อย่างสูงค่ะ 🙏
ทางเรารับเรื่องและจะดำเนินการทันที:
1. คืนเงินเต็มจำนวน (3 วันทำการ)
2. เก็บเคสเป็น priority สำหรับ QC review
3. ผู้จัดการจะโทรคุยกับคุณด้วยตัวเองภายในวันนี้

ขอเบอร์โทรที่สะดวกรับสายได้ที่ไหม?
```

## 📊 SLA + KPI

### SLA (Service Level Agreement)
- **First response:** ≤ 15 นาที (peak hours 10:00-20:00)
- **Resolution:** ≤ 24 ชม. (simple), ≤ 72 ชม. (complex)
- **Offline response:** ภายใน 10:00 เช้าวันถัดไป

### KPI ที่ต้องวัดทุกเดือน
1. **First Response Time (FRT)** — target < 15 นาที
2. **Resolution Rate** — target > 90%
3. **CSAT Score** (rating หลังปิดเคส) — target > 4.2/5
4. **Escalation Rate** — target < 15%
5. **Inbox เฉลี่ยต่อคน/วัน** — benchmark capacity
