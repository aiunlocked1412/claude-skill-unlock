# Customer Service Pro

> AI วาง CS system ครบวงจร — chatbot, FAQ, escalation, reply template

## ⚡ ใช้ยังไง

```
/customer-service-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Chatbot flow (decision tree) — Line OA / FB Inbox / IG DM
- ✅ FAQ Bank 20+ คำถาม-ตอบ (pre-sale, post-sale, policy)
- ✅ Escalation Rule (เมื่อไรส่งต่อให้คน)
- ✅ Reply Template (inbox, email, complain, refund)
- ✅ CS Playbook เต็มรูปแบบ
- ✅ SLA + KPI (response time, resolution rate)
- ✅ Complaint Handling (ลูกค้าโกรธ)
- ✅ Bad Review Response (Shopee/Lazada)

## 💡 Use cases

- ร้านออนไลน์มีออเดอร์เยอะจนตอบไม่ทัน
- แบรนด์อยากมี brand voice สม่ำเสมอใน CS
- ทีม CS ใหม่ต้อง training
- ย้ายระบบมา Manychat / Zoho / HubSpot
