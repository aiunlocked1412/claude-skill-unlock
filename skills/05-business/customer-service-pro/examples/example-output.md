## 🤖 Chatbot Flow (Line OA)

### Greeting Message
```
สวัสดีค่ะ GlowUp ยินดีต้อนรับค่า 🌸
สนใจเรื่องอะไรคะ?

1️⃣ ดูสินค้า / ขอคำแนะนำ skincare
2️⃣ เช็คออเดอร์ที่สั่งแล้ว
3️⃣ คืน / เคลมสินค้า
4️⃣ โปรโมชั่นวันนี้
5️⃣ คุยกับแอดมินค่า 💁‍♀️
```

### Branch 1: ดูสินค้า
```
→ "สนใจสินค้าหมวดไหนคะ?"
   ├── 1.1 Skincare (cleanser, toner, serum, ครีม)
   ├── 1.2 Makeup (คุชชั่น, ลิป, บรัช)
   ├── 1.3 Sunscreen
   ├── 1.4 Best seller
   └── 1.5 ขอคำแนะนำจากพี่เลี้ยง (ส่งต่อ CS คน)
```

### Branch 2: เช็คออเดอร์
```
→ "ขอเลข order 8 หลักที่ได้รับหลังสั่งนะคะ 📦"
   → [ผู้ใช้พิมพ์เลข]
   → Bot ดึง tracking จากระบบ + ส่ง link Kerry
   → "ถ้ามีอะไรเพิ่มเติม tag แอดมินได้เลยค่ะ"
```

### Branch 3: คืน/เคลม
```
→ "ขอเลือก 1 ตัวเลือกนะคะ:
    3.1 สินค้ายังไม่ได้ใช้ — อยากเปลี่ยนรุ่น / คืนเงิน
    3.2 สินค้าชำรุด / เสียหายจากการขนส่ง
    3.3 ได้รับของผิด
    3.4 แพ้สินค้า (allergy)"

→ [3.2 ชำรุด] → "ขอภาพสินค้า + กล่อง + ใบเสร็จค่ะ"
→ [3.4 Allergy] → handoff ทันที (P1 priority)
```

## ❓ FAQ Bank (25 ข้อ)

### 🛒 Pre-sale

**Q1: ของแท้ไหม?**
```
✅ ของแท้ 100% ทุกชิ้นค่ะ

**Q2: ส่งกี่วันถึงคะ?**
```
📦 Kerry/Flash = 1-2 วัน (กทม.) / 2-3 วัน (ตจว.)
📮 EMS = 2-4 วัน
ส่งทุกวันจันทร์-เสาร์ (งดอาทิตย์)

สั่งก่อน 14:00 = ส่งวันนั้นเลยค่า 🚀
```

**Q3: ค่าส่งเท่าไร?**
```
ค่าส่งเริ่ม 50 บาท
🎁 ครบ 500 บาท = ส่งฟรี Kerry
🎁 ครบ 1,500 บาท = ส่งฟรี + ของแถม surprise
```

**Q4: ชำระเงินยังไง?**
```

💰 ปลายทาง (COD) +30 บาท
