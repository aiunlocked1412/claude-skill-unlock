# Prompt Formula — Business Consultant

## Template คำถาม context

```
ธุรกิจ: <ชื่อ + สินค้า/บริการ>
Stage: <idea / pre-launch / <1 ปี / >1 ปี>
ตลาด: <B2C / B2B / online / offline>
รายได้: <จำนวน/เดือน ถ้ามี>
ทีม: <จำนวนคน + บทบาท>
Goal: <เป้าหมาย 6-12 เดือน>
ปัญหาใหญ่ที่สุดตอนนี้: <1 ประโยค>
```

## SWOT Prompt Recipe

```
วิเคราะห์ SWOT ของ <ธุรกิจ>:
- Strengths: จุดแข็งภายใน 4-6 ข้อ (ทีม, สินค้า, brand, ต้นทุน, network)
- Weaknesses: จุดอ่อนภายใน 4-6 ข้อ (ต้องซื่อสัตย์)
- Opportunities: โอกาสภายนอก 4-6 ข้อ (เทรนด์, regulation, เทคโนโลยี)
- Threats: ภัยคุกคามภายนอก 4-6 ข้อ (คู่แข่ง, เศรษฐกิจ)
- สรุป 3 Strategic Moves ที่ควรทำใน 90 วัน
```

## Business Canvas Prompt

```
ทำ Business Model Canvas สำหรับ <ธุรกิจ> 9 ช่อง:
1. Customer Segments — แบ่ง persona 2-3 ตัว
2. Value Proposition — คุณค่าหลัก + รอง
3. Channels — online + offline + referral
4. Customer Relationships — แบบบริการตัว, self-service, community
5. Revenue Streams — แต่ละ stream + % สัดส่วน
6. Key Resources — physical, IP, human, financial
7. Key Activities — production, problem solving, platform
8. Key Partners — supplier, alliance, outsourcing
9. Cost Structure — fixed vs variable, economies of scale
```

## Porter's 5 Forces Prompt

```
วิเคราะห์ Porter's 5 Forces ของอุตสาหกรรม <อุตสาหกรรม>:
ให้คะแนนแต่ละแรง Low/Medium/High + เหตุผล 2-3 ข้อ
สรุป: อุตสาหกรรมนี้น่าเข้าแค่ไหน? (1-10)
```

## Pitch Deck Recipe

```
ร่าง pitch deck 10 slides สำหรับ <ธุรกิจ> ขอเงิน <จำนวน>:
- Slide headline (ข้อความใหญ่)
- Slide content (bullet 3-5 ข้อ)
- Slide visual suggestion (ใส่อะไร)
- Speaker notes (พูดอะไร 30-60 วิ)
```

## Financial Forecast Recipe

```
ทำ 3-year forecast:
- สมมติฐาน: price, unit sold, growth rate, CAC, churn
- 3 scenarios: conservative (-30%), base, optimistic (+30%)
- Breakdown: Revenue, COGS, Gross Profit, OpEx, EBITDA, Net Profit
- Key metrics: Gross Margin %, Net Margin %, Payback period
```

## Tips

- ตัวเลขทุกตัวต้องมี "ที่มา" — ระบุว่าเป็นสมมติฐานหรือข้อมูลจริง
- SWOT ควร Weaknesses ซื่อสัตย์ ไม่ใช่จุดแข็งที่แต่งว่าเป็นจุดอ่อน
- Pitch deck slide 3 (Solution) สำคัญที่สุด — ต้อง 10 วิเข้าใจ
- Financial forecast — conservative > optimistic เสมอ เมื่อ pitch นักลงทุน
