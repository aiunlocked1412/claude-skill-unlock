# Import-Export

> ผู้เชี่ยวชาญการค้าระหว่างประเทศ — HS Code, Incoterms, LC, พิธีการศุลกากรไทย

## ⚡ ใช้ยังไง

```
/import-export
```

## 🎯 ทำอะไรได้บ้าง

- ✅ HS Code Lookup & Classification (พิกัดศุลกากร 8 หลัก + อัตราภาษี)
- ✅ Incoterms Selector (FOB, CIF, EXW, DAP, DDP — เลือกให้เหมาะ)
- ✅ เอกสารนำเข้า/ส่งออก (Invoice, BL, CO, Packing List, FDA)
- ✅ การเงินการค้า (LC, T/T, D/P, D/A — ความเสี่ยงและวิธีป้องกัน)
- ✅ พิธีการศุลกากรไทย step-by-step (e-Customs, NSW Portal)
- ✅ Landed Cost Calculator (ต้นทุนรวมจริงรวมอากร + VAT + handling)

## 💡 Use Cases

- SME ไทยจะเริ่มนำเข้าสินค้าจากจีน/ยุโรป ไม่รู้จะเริ่มตรงไหน
- ผู้ส่งออกไทยต้องการ CO ใช้สิทธิ FTA ลดภาษี
- บริษัทที่เปิด LC ครั้งแรก ไม่แน่ใจขั้นตอน
- ต้องการคำนวณ landed cost จริงก่อนตัดสินใจนำเข้า
- พบปัญหา Red Lane + ของค้างที่ท่าเรือ

## 🎓 Tips

- บอก **ชื่อสินค้าละเอียด** (วัสดุ + การใช้งาน) → HS Code แม่น
- ระบุ **ประเทศต้นทาง** → ตรวจสิทธิ FTA และ anti-dumping duty
- ใช้คู่กับ `/startup-founder` หรือ `/franchise-owner` → วางแผนต้นทุนนำเข้าสินค้า

- `/startup-founder` — สินค้า import เพื่อขายใน startup
- `/franchise-owner` — วัตถุดิบจาก franchisor ต่างประเทศ
- `/business-consultant` — วิเคราะห์ margin + competitive pricing
