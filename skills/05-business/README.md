# หมวด 5: Business & Service (7 skills)

Skills สำหรับที่ปรึกษาธุรกิจ, บัญชี, HR, กฎหมาย, อสังหา

| Skill | คำสั่ง | ใช้เมื่อไหร่ |
|-------|--------|-------------|
| Business Consultant | `/business-consultant` | SWOT, canvas, pitch |
| Accountant | `/accountant-pro` | งบการเงิน, ภาษี, cash flow |
| HR Officer | `/hr-officer-pro` | recruit, JD, interview |
| Customer Service | `/customer-service-pro` | chatbot, FAQ, escalation |
| Stock Trader | `/stock-trader-pro` | technical, news, risk |
| Lawyer | `/lawyer-pro` | สัญญา, คดี, memo |
| Real Estate | `/real-estate-pro` | listing, market, tour |

## รูปแบบ output ทั่วไป

- Structured doc (ตาราง + bullet)
- Template เอกสาร (สัญญา, JD, proposal)
- Analysis report พร้อม recommendation
- FAQ / Script สำหรับพนักงาน

## ⚠️ Disclaimer

Skills ในหมวดนี้เป็น **ผู้ช่วยทำงาน** ไม่ใช่ผู้เชี่ยวชาญแทน:
- `/accountant-pro` — ต้องให้นักบัญชีตรวจก่อนยื่นภาษีจริง
- `/lawyer-pro` — ต้องให้ทนายตรวจก่อนใช้เอกสารจริง
- `/stock-trader-pro` — ให้ข้อมูลเทคนิค ไม่ใช่คำแนะนำลงทุน

## Workflow ตัวอย่าง

```
1. /business-consultant SWOT + canvas
   ↓
2. /hr-officer-pro JD + recruit team
   ↓
3. /customer-service-pro FAQ + chatbot
   ↓
4. /accountant-pro วางแผน cash flow
   ↓
5. /lawyer-pro ร่างสัญญาจ้าง
```
