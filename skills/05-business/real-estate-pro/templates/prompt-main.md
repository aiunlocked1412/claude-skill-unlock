# Prompt Formula — Real Estate Pro

## Context Questions

```
ประเภท: <condo / detached house / townhome / land / commercial>
ขาย/เช่า: <sale / rent>
ราคา: <ขาย X ลบ. / เช่า XX,XXX/เดือน>
ที่ตั้ง:
  - โครงการ: ...
  - ซอย/ถนน: ...
  - เขต/แขวง: ...
  - จังหวัด: ...
ขนาด:
  - พื้นที่ใช้สอย: XX ตรม.
  - ห้องนอน: X + ห้องน้ำ: X
  - ชั้น: X (จาก X ชั้น)
  - ทิศ: ...
Features:
  - เฟอร์นิเจอร์: full / partial / unfurnished
  - เครื่องใช้ไฟฟ้า: (list)
  - วิว: river/city/garden/pool
  - ห้องครัว: open / closed / pantry
Project Facilities:
  - pool / gym / sauna / co-working / kids area / playground
Proximity (ระยะ):
  - BTS/MRT: ...
  - ห้าง: ...
  - โรงพยาบาล: ...
  - โรงเรียน: ...
  - ทางด่วน: ...
Target:
  - Young professional / family / expat / investor
USP (ข้อเด่น 1 อย่าง): ...
```

## Listing Vocabulary

### Words that SELL (ใช้ได้)
- "ติดริมน้ำ / ริมแม่น้ำ"
- "วิว 360 องศา"
- "ใกล้ BTS เดินแค่ 5 นาที"
- "ห้องมุม / double view"
- "High floor"
- "Fully furnished"
- "ห้องเพดานสูง"
- "ธรรมชาติเยอะ / ร่มรื่น"
- "สภาพเหมือนใหม่"
- "ราคาต่อรองได้"

### Words to AVOID
- ❌ "ถูกที่สุด" (ไม่มีข้อพิสูจน์)
- ❌ "Luxury" โดยไม่มี feature justify
- ❌ "Perfect" / "ไม่มีที่ติ" (overclaim)
- ❌ "เจ้าของขายเอง ไม่รับ agent" (หลีกเลี่ยง filter)
- ❌ ตกแต่งอิตาเลี่ยนแท้" (ถ้าไม่มีใบรับรอง)

## CMA Structure

```
Subject Property: [name + details]

Comparables (3-5):
1. [Same building] — XX sqm / X.X M / features
2. [Near building] — ...
3. [Walking distance] — ...

Analysis:
- Price/sqm average: XXXk
- Subject position: premium / market / below
- Expected time on market: XX วัน
- Recommended list price: X.X M
```

## Negotiation / Pricing Tips

- ราคาตลาดจาก CMA = ราคาขายได้ใน 30-60 วัน
- ตั้งสูงกว่า target 3-5% (เผื่อต่อ)
- ขายเร็ว (< 30 วัน) = ยอมลด 3-5%
- ขายช้า (> 90 วัน) = ลด 5-10%
- รับโอนเร็ว + cash = ลดเพิ่ม 1-2%
- **ระวัง:** ราคาต่อ ตรม. ใกล้เคียงตลาดจะขายได้ไว

## ภาษีการโอนที่ควรรู้ (สำหรับ seller)

| ภาษี/ค่า | อัตรา | ใครจ่าย |
|----------|-------|---------|
| ค่าโอน | 2% ของราคาประเมิน | ตกลงกัน (ปกติแบ่งครึ่ง) |
| อากรแสตมป์ | 0.5% | seller (ถ้าถือ > 5 ปี) |
| ภาษีธุรกิจเฉพาะ | 3.3% | seller (ถ้าถือ < 5 ปี — แทนอากรแสตมป์) |
| ภาษีเงินได้หัก ณ ที่จ่าย | ตามอายุการถือ | seller |

**ตัวอย่างคอนโด 6.5 ลบ.:**
- ถือ 3 ปี → ภาษีธุรกิจเฉพาะ 215k + ค่าโอน 130k (แบ่งครึ่ง)
- ถือ 6 ปี → อากรแสตมป์ 33k + ค่าโอน 130k
