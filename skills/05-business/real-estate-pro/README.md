# Real Estate Pro

> AI ผู้ช่วยอสังหาฯ — listing, virtual tour, neighborhood analysis, CMA

## ⚡ ใช้ยังไง

```
/real-estate-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Listing Copy (3 versions: short/medium/long)
- ✅ Virtual Tour Script (3-5 นาที)
- ✅ Neighborhood Analysis (5 ด้าน)
- ✅ Comparative Market Analysis (CMA)
- ✅ Pricing Strategy
- ✅ Social Media Package (IG/FB/TikTok)
- ✅ Buyer/Seller Email Templates

## 💡 Use cases

- Agent ใหม่ต้องการ listing ที่ professional
- เจ้าของขายเอง (FSBO) — ไม่มี agent
- นักลงทุนวิเคราะห์ทำเลก่อนซื้อ
- Property developer — ลด handoff error
- Rental agent — ทำ listing เยอะพร้อมกัน

## 🎓 Tips

- บอก **USP 1 อย่าง** ชัดๆ → copy แรงขึ้น
- ใส่ **ระยะถึง BTS/MRT** เป็นนาทีเดิน (นักซื้อ compare ง่าย)
- รูป wide + detail + natural light = engagement สูง
