# DJ Pro

> AI DJ มืออาชีพ — Setlist Design, BPM Matching, Crowd Reading, Mixing Technique สำหรับทุก gig type

## ⚡ ใช้ยังไง

```
/dj-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Setlist Design พร้อม energy curve (warm-up → peak → closing)
- ✅ BPM & Key Mixing Plan (Camelot Wheel, transition strategy)
- ✅ Crowd Reading Framework (signal + action guide)
- ✅ Mixing Technique Tips (blend, cut, drop, acapella)
- ✅ Gig Preparation Checklist (club, wedding, corporate)
- ✅ Track Recommendation ตาม genre + mood + era

## 💡 Use cases

- DJ มือใหม่ต้องการ setlist structure สำหรับงานแรก
- DJ ประจำ club ต้องการ plan setlist ใหม่ทุกสัปดาห์
- Wedding DJ จัดเรียง flow ตั้งแต่ cocktail hour ถึง last dance
- Event planner ต้องการ brief ส่งให้ DJ ว่าต้องการอะไร
- DJ อยากเรียน Camelot Wheel mixing และ BPM transition

## 📊 ตัวอย่างผลลัพธ์

Setlist Sheet ครบถ้วนพร้อม BPM flow, energy curve, track list แต่ละ phase และ backup tracks สำหรับทุก gig type

## 🎓 Tips

- บอก **crowd profile ละเอียด** (อายุ, สัญชาติ, vibe) → setlist ตรงกลุ่มขึ้น
- บอก **venue size + soundsystem** → เลือก genre ที่เหมาะสม
- ใช้คู่กับ `/music-producer` → เข้าใจ production ของ track ที่เล่น
- ขอ **backup tracks** เสมอ — งาน live ต้องมีแผนสำรอง

## 🔗 Skills ที่ใช้คู่กัน

- `/music-producer` — เข้าใจ beat structure ของ track
- `/musician-pro` — วิเคราะห์ key + BPM ของ track
- `/hotel-concierge` — coordinate กับ venue เรื่อง audio setup
