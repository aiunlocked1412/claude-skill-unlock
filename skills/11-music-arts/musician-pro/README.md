# Musician Pro

> AI นักดนตรีและ Composer — Chord Progression, Arrangement, Music Theory, Compose Brief รองรับดนตรีไทย+สากล

## ⚡ ใช้ยังไง

```
/musician-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Chord Progression ตาม genre + key + mood (pop, R&B, jazz, ลูกทุ่ง)
- ✅ Song Structure & Arrangement (intro → outro พร้อม energy map)
- ✅ Music Theory อธิบาย (scale, mode, harmony, voice leading)
- ✅ Instrument Part Writing (piano, guitar, bass, drums, strings)
- ✅ DAW Workflow Tips (Logic Pro X, Ableton Live, FL Studio)
- ✅ Compose Brief สรุปสำหรับโปรดิวเซอร์/นักดนตรี

## 💡 Use cases

- นักดนตรีมือใหม่อยากเข้าใจ chord function และ music theory
- Songwriter ต้องการ chord progression แบบใหม่ที่ไม่ซ้ำใคร
- โปรดิวเซอร์ต้องการ arrange เพลงใหม่และต้องการ brief ครบ
- Band ต้องการ instrument part สำหรับซ้อม
- ครูดนตรีต้องการอธิบาย theory ให้นักเรียนเข้าใจง่าย

musician-pro/

## 📊 ตัวอย่างผลลัพธ์

Compose Brief ครบถ้วนสำหรับเพลงป๊อปไทย: key, chord progression, structure, instrumentation, reference tracks และ DAW notes

## 🎓 Tips

- บอก **reference track** ที่ชอบ → Claude จับ feel ได้แม่นยิ่งขึ้น
- ระบุ **เครื่องดนตรีที่มีจริง** → arrangement จะปฏิบัติได้จริง
- ใช้คู่กับ `/songwriter` → chord + lyrics ไปด้วยกัน
- ใช้คู่กับ `/music-producer` → ต่อยอดเป็น production brief

## 🔗 Skills ที่ใช้คู่กัน

- `/songwriter` — เนื้อร้อง + hook ที่เข้ากับ chord
- `/music-producer` — beat making + mixing brief
- `/dj-pro` — remix และ mashup idea จาก chord analysis
