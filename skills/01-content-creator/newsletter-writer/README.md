# Newsletter Writer

> AI Email Newsletter Specialist ที่ทำให้คนรอเปิดอ่าน — ไม่ใช่ archive

## ⚡ ใช้ยังไง

```
/newsletter-writer
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Newsletter issue (1 ฉบับ — Hook-Story-Offer framework)
- ✅ Editorial calendar (4-12 weeks)
- ✅ Welcome sequence (5-7 emails สำหรับ new subscriber)
- ✅ Growth playbook (จาก 0 → 1K → 10K subs)
- ✅ Monetization plan (paid tier / sponsor / affiliate)
- ✅ Deliverability audit (spam check + technical)

## 💡 Use cases

- Independent newsletter operator (Substack, beehiiv, ConvertKit)
- B2B SaaS founder ส่ง weekly update
- Creator ที่อยากเป็นเจ้าของ audience เอง (ไม่อิง algorithm)
- Course creator ส่ง drip campaign
- Community manager ที่ดูแล member newsletter

## 🎓 Tips

- ระบุ **niche** ชัด → เขียนตรงใจ + growth ง่าย
- 1 main idea ต่อ 1 issue → engagement สูงกว่ายัด 5 เรื่อง
- Subject line ทดสอบ A/B เสมอ → +20-30% open rate
- ใช้คู่กับ `/copywriter-pro` เพื่อปรับ subject line ให้ punchy
- ใช้คู่กับ `/blogger-pro` แปลง issue เก่าเป็นบทความ SEO
