# Comic Writer Templates — Panels, SFX, Dialogue, Pacing

## Panel Composition Patterns

### Panel Sizes & Purpose

| Size | Use case |
|------|----------|
| Splash (full page) | Major reveal, action peak, scene establishment |
| Half page | Important moment, dramatic pause |
| Standard (1/4-1/6) | Regular dialogue/action |
| Inset / strip | Reaction, transition, beat |

### Panel Count per Page (Manga)
- Action page: 4-6 panels
- Dialogue page: 5-8 panels
- Quiet/atmosphere: 2-3 panels
- Reveal: 1 splash

### Camera Angles (specify ทุก panel)

| Angle | Effect |
|-------|--------|
| Wide / establishing | Set scene, isolation, scale |
| Medium | Standard dialogue |
| Close-up (face) | Emotion, tension |
| Extreme close-up (eyes/object) | Intense emotion, key detail |
| Over-shoulder (OTS) | Conversation, POV bias |
| POV (1st person) | Immersion, surprise reveal |
| Bird's eye / God view | Vulnerability, scope |
| Low angle / Worm view | Power, intimidation |
| Dutch angle (tilted) | Unease, chaos |

## Dialogue Templates

### Bubble Types

| Type | Use |
|------|-----|
| Round (regular) | Normal speech |
| Spiky (jagged) | Shouting, anger, alarm |
| Cloud (fluffy) | Thoughts, dreams |
| Square / rectangle | Caption (narrator), TV/radio |
| Whisper (dotted) | Whisper, weak voice |
| Dripping (horror) | Demonic, sinister |

### Dialogue Density Rules

| Format | Bubbles/panel | Words/bubble |
|--------|---------------|--------------|
| Manga | 1-3 | ≤25 |
| Western | 1-4 | ≤30 |
| Webtoon | 0-1 | ≤20 |

**Total page word count target:**
- Manga page: ≤200 คำ
- Western page: ≤250 คำ
- Webtoon segment (10 panels): ≤150 คำ

### Dialogue Smell Test
Read aloud — ถ้าฟังไม่เป็นธรรมชาติ revise

❌ "As you know, Sarah, our village was attacked 5 years ago by the demon king."
✅ "ถ้ามันรู้ว่าเรารอด..." / "5 ปีมาแล้วนะ ซาราห์"

❌ Over-explanation ("ผมรู้สึกเศร้า เพราะแม่ผมตาย")
✅ Show via panel + minimal dialogue ("...แม่")

## Pacing Tools

### Beat Per Page (BPP)
- Action: 1-2 beats/page (slow + impactful)
- Dialogue: 3-4 beats/page (conversational)
- Comedy: 4-6 beats/page (rapid-fire)
- Horror: 1 beat/page + dread builds

### Decompression vs Compression

**Decompression (modern manga, webtoon):**
- 1 action across 4-6 panels
- เน้น emotion + atmosphere
- Slow burn

**Compression (golden age Western, gag comic):**
- หลาย action ใน 1 panel
- Dense plot per page
- Fast paced

**เลือกใช้ตาม genre:**
- Slice of life, drama → Decompression
- Battle shonen, gag → Mix
- Mystery thriller → Compression for clues, decompression for tension

## Webtoon Vertical Layout

### "Long Strip" Pacing
- ใช้ vertical space เป็น "เวลา"
- Panel ห่างกันเยอะ = ช่วงนิ่ง / ช่วงสะเทือน
- Panel ติดกัน = continuous action
- "Page break" = scene change

### Webtoon Rhythm
```
[Panel: wide establishing]
[gap — let reader settle]
[Panel: close-up character]
[Panel: dialogue 1]
[Panel: dialogue 2 + action]
[gap — beat]
[Panel: reaction face]
[BIG gap — tension]
[Splash: reveal]
```

### Webtoon Cliffhanger Hook
- จบ chapter ที่ panel "shocking face" + 1 line dialogue
- ห้ามจบกลาง dialogue — หาที่จบด้วย image-driven

## Series Bible Structure

```markdown
# Series Bible

## Premise (1 sentence)
"[Character] in [setting] must [goal] before [stake]"

## Logline (3-4 sentences)
...

## Genre + Tone
...

## World
- Setting time/place
- Rules (magic system, tech level, social order)
- Aesthetic reference (3-5 ภาพ ref)

## Main Characters (3-7 ตัว)
For each:
- Name + age + role
- Visual description (height, hair, clothing signature)
- Personality (3 adjectives + 1 contradiction)
- Backstory (1 paragraph)
- Goal (external + internal)
- Arc (start → end)

## Supporting Cast
- Name, role, function in story (sentence)

## Story Arc (long-term)
- Arc 1 (Chapter 1-15): Discovery
- Arc 2 (Chapter 16-30): Conflict escalates
- Arc 3 (Chapter 31-50): Resolution

## Themes
- Primary: ...
- Secondary: ...

## Visual Style Notes (สำหรับ artist)
- Line weight
- Shading style (screen tone / digital / minimal)
- Panel layout preference
- Color palette (ถ้า color)

## Tone Reference
- Inspired by: [Series A] meets [Series B]
- Avoid: [Series C] tone (เพราะ...)
```

## Format Conversion (Manga → Webtoon)

### Splitting Manga Page → Webtoon Panels

1. **Identify beats** ใน manga page (4-8 panels)
2. **Each manga panel = 1-2 webtoon panels**
3. **Add "breath panels"** ระหว่าง beat — เช่น close-up object, background detail
4. **Resize:**
   - Manga panel ratio: ~3:4 หรือ 4:3
   - Webtoon panel ratio: 4:3 หรือ 16:9 (กว้างกว่าสูง)
5. **Reading direction:**
   - Manga: right-to-left (R→L)
   - Webtoon: top-to-bottom (T→B)
6. **Dialogue:** ลด text ลง 30% — webtoon scroll เร็ว
