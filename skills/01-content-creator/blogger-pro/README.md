# Blogger Pro

> AI Blog Writer ที่เขียนบทความติด Google + อ่านสนุก พร้อม SEO ครบชุด

## ⚡ ใช้ยังไง

```
/blogger-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ บทความ how-to (วิธีทำอะไรสักอย่าง)
- ✅ Listicle (5 วิธี, 10 เทคนิค)
- ✅ รีวิวสินค้า/บริการ
- ✅ บทวิเคราะห์ (deep-dive / opinion piece)
- ✅ Keyword research + outline (ก่อนเขียน)
- ✅ SEO metadata (title, meta, slug, schema)
- ✅ Content calendar 30 บทความ

## 💡 Use cases

- Blogger / นักเขียนส่วนตัว
- Content marketer บริษัท
- Affiliate marketer
- เจ้าของ eCommerce
- Freelance writer

## 🎓 Tips

- ให้ **primary keyword** ชัด → บทความติดอันดับเร็วขึ้น
- ระบุ **target audience** (มือใหม่ / มือเก๋า) → โทนแม่นขึ้น
- ใช้คู่กับ `/copywriter-pro` เพื่อปรับ headline ให้ CTR ดี
- ใช้คู่กับ `/translator-pro` เพื่อทำ version ภาษาอังกฤษ

## 🔗 Skills ที่ใช้คู่กัน

- `/copywriter-pro` — ปรับ headline + meta description ให้ CTR สูง
- `/translator-pro` — แปลบทความ Thai ↔ English
- `/graphic-designer-pro` — สร้าง featured image
- `/social-media-admin` — ทำ snippet ไปโพสโปรโมท
