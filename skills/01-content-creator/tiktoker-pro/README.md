## 🎓 Tips

- **First second** ต้อง hook — algorithm 2026 ตัด video ใน 1 วิแรก
- ใช้ **captions overlay** ตลอด — 80% mute
- จำกัด **hashtag 3-5** — มากไป algorithm งง
- ใช้คู่กับ `/youtuber-pro` ถ้าทำ Shorts/Reels parallel
- ใช้คู่กับ `/copywriter-pro` ปรับ caption + comment bait
