# TikToker Templates — Hooks, Sound, Algorithm 2026, Comment Bait

## TikTok Algorithm 2026 — Signal Hierarchy

### Tier 1 (สำคัญสุด)
1. **Watch time / completion rate** — % คนดูจบ
2. **Re-watch rate** — คนดูซ้ำ (loop video!)
3. **Share rate** — DM share + share to other platforms

### Tier 2
4. **Comment rate** — comment / view
5. **Like rate** — like / view (less important than 2024)
6. **Save rate** — saves
7. **Follow conversion** — view → follow %

### Tier 3
8. **Profile visit rate**
9. **Sound usage** (ถ้าเป็น original sound, คนใช้ sound ของคุณ → boost)
10. **Hashtag relevance** (3-5 ตัวที่ตรง niche)

### Negative signals
- Quick scroll-away (ใน 1.5 วิ) — kills reach
- Watermark from other platform (Reels/Shorts logo)
- Repeated content จาก 24 ชม. ก่อน
- Spam hashtag / link in caption
- Shadowban triggers (controversial keyword)

## First-Second Hook Patterns (10 Templates)

### Pattern 1: Visual Shock
**Format:** Open with unexpected visual + zero context
> Example: เปิดด้วยกล้องซูม face คนตกใจ → "ทำไมเขาถึง...?"

### Pattern 2: Question Hook
**Format:** Bold question text overlay + curious face
> "ทำไมแฟนบอกฉันว่า 'ไม่เป็นไร'?"

### Pattern 3: Before/After Tease
**Format:** Show "after" first, tease "before"
> "ผมหายปวดหลังใน 7 วันด้วยเทคนิคนี้..."

### Pattern 4: Counter-intuitive Statement
**Format:** Bold claim ที่ขัดความเชื่อ
> "หยุดดื่มน้ำ 8 แก้ว/วัน — มันทำร้ายไตคุณ"

### Pattern 5: POV Drop
**Format:** "POV: [scenario]" overlay
> "POV: คุณเป็นเด็กม.6 ตื่นเช้าวันสอบ"

### Pattern 6: Number Promise
**Format:** ตัวเลข specific + promise
> "5 คำที่ทำให้ HR เรียกสัมภาษณ์ทันที"

### Pattern 7: Story Cold Open
**Format:** Drop into mid-story
> "...แล้วเขาก็บอกว่า 'นี่ไม่ใช่ครั้งแรก'"

### Pattern 8: Tutorial Tease
**Format:** Show end result, tease how
> "ทำกาแฟร้านในบ้าน ฉันใช้แค่ 3 อย่าง"

### Pattern 9: Confession
**Format:** "ผมไม่ได้บอกใคร แต่..."
> "ผมจะบอกความจริงเรื่องเงินเดือนผมตอนนี้"

### Pattern 10: Trending Sound + Visual Match
**Format:** Match video beat กับ trending sound drop
> Sound drop ใน 1 วิ → cut ไปภาพหลัก

## Trending Sound Strategy

### Sound Lifecycle (ใช้ stage ไหน?)

| Stage | Usage count | Strategy |
|-------|-------------|----------|
| Emerging | < 50K | High risk, high reward — first-mover bonus |
| Growing | 50K - 500K | **SWEET SPOT** — boost from algorithm |
| Peak | 500K - 2M | Saturated — only use if super relevant |
| Declining | > 2M | Avoid — algorithm penalty |

### How to find trending sounds

1. **TikTok Creator Search:** filter "trending" + your niche
2. **Watch For You Page** สัก 30 video — note sound ที่ซ้ำ
3. **TrendTok / TikTok Creative Center**
4. **Look for ↑ arrow** ใน sound info (TikTok shows trending indicator)

### Sound Selection Rules

- **Match niche:** sound trend ต้อง make sense กับ content คุณ
- **Match emotion:** dramatic sound + comedic content = jarring (unless intentional)
- **Match format:** dance sound + tutorial = mismatch
- **Original > trending** ถ้าเป็น brand long-term — but slower growth

## Caption + Hashtag Strategy

### Caption Length & Style

| Length | Use case |
|--------|----------|
| 0-50 chars | Strong hook video, let video speak |
| 50-100 chars | Standard — question or context |
| 100-150 chars | Story tease or detailed CTA |
| 150+ | RARELY — only if essential context |

### Caption Formulas

1. **Question:** "คุณคิดว่ายังไงบ้าง?"
2. **Statement + Tease:** "ผมเปลี่ยนมา 3 เดือน ผลลัพธ์ใน video"
3. **Direct CTA:** "Save ไว้ก่อนลืม"
4. **Mystery:** "เกือบไม่กล้าโพสต์อันนี้..."
5. **Comment bait:** "ใครเคยเจอแบบนี้บ้าง? Comment เลย"

### Hashtag Strategy (3-5 ตัว)

```
1× ULTRA-NICHE   → #DIYดริปกาแฟ      (small but exact audience)
2× NICHE         → #กาแฟ #cafeที่บ้าน  (broader but relevant)
1× CATEGORY      → #foodie หรือ #lifestyle
1× TRENDING      → #2026trend (ถ้ามี + relevant)
```

❌ **อย่าใช้:** #fyp #foryou #viral #thailand (ทุกคนใช้ → algorithm ignore)

## Comment Bait Patterns (6 Templates)

### Pattern 1: "Wrong on Purpose"
ใส่ข้อมูลผิดเล็กๆ ที่คนรู้จะแก้
> Video: "แม่ค้าผมขายมา 5 ปี ทำกาแฟมา 100 แก้ว"
> Comment: "พี่คะ 5 ปี = 5×365 = 1825 วัน × 50 แก้ว/วัน = ไม่ใช่ 100 นะ!"

### Pattern 2: "What's your..."
ตั้งคำถามเปิด
> "comment ของหนัก/เบาที่สุดที่คุณซื้อสไลด์เดียว"

### Pattern 3: "Part 2 if..."
> "comment 'PART 2' ถ้าอยากรู้ Step 4-5"

### Pattern 4: "Did you see..."
ซ่อน detail
> "ดู second 12 ดีๆ — มีอะไรในกระจก"

### Pattern 5: "Tag someone who..."
> "tag friend ที่ทำแบบนี้ทุกวันแน่นอน"

### Pattern 6: Easter Egg / Hidden detail
สร้าง mystery, คน comment สมมติฐาน
> Video เผยภาพ T-shirt ลึกลับ → "นี่ teaser เรื่องอะไร?"

## Loop Video Technique (Re-watch Booster)

จุดประสงค์: คน rewatch = signal ดีต่อ algorithm

### How to loop:

1. **Visual loop:** Frame สุดท้ายของ video = Frame แรก (smooth cut)
2. **Audio loop:** Sound ตอนจบ = sound ตอนเปิด (กระแสไหลกลับ)
3. **Story loop:** Cliffhanger ที่จบ → "ตอบไปแล้วใน video" (คนกลับไปหา)
4. **Hook loop:** ตอนจบ tease "ดู comment สำหรับ surprise" (คนวนกลับมาดู)

### Example structure:
```
0:00 — "ผมจะบอก 3 mistakes ที่คนทำกาแฟดริป" (hook)
0:25 — "ส่วน mistake #1 อยู่ตอนเปิด video — เห็นไหม?" (loop trigger)
[Player rewinds to find #1 — re-watch counted!]
```

## Algorithm Audit Checklist

ใช้เมื่อ video ไม่ติด FYP — diagnose ทีละข้อ:

### Hook Quality
- [ ] First second มี visual + audio + text hook?
- [ ] First 3 วิ ห้าม intro / "สวัสดี"?
- [ ] Hook ตรง niche ที่ algorithm จัดให้?

### Watch Time
- [ ] Average watch time > 60% of length?
- [ ] Drop-off chart smooth (ไม่ cliff)?
- [ ] Re-watch rate > 5%?

### Engagement
- [ ] Comment > 0.5% of views?
- [ ] Share > 0.3% of views?
- [ ] Comment bait มีไหม?

### Technical
- [ ] No watermark from other platform?
- [ ] Resolution 1080×1920?
- [ ] No copyrighted music (non-trending)?
- [ ] Caption ไม่มี link?

### Niche Consistency
- [ ] โพสต์ niche เดียวกัน 5+ video ล่าสุด?
- [ ] Hashtag ตรง niche?
- [ ] Bio + profile บอก niche ชัด?

### Posting Frequency
- [ ] โพสต์อย่างน้อย 1 ครั้ง/วัน (สำหรับ growth)?
- [ ] ไม่ post-then-delete (algorithm ลงโทษ)?
- [ ] เวลา post ตรง audience peak (ดู Analytics)?
