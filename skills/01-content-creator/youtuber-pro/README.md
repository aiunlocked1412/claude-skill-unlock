# YouTuber Pro

> AI Script Writer สำหรับครีเอเตอร์ไทย ครบตั้งแต่ idea → script → thumbnail → metadata

## ⚡ ใช้ยังไง

```
/youtuber-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ สคริปต์ YouTube 5-20 นาที (พร้อม hook, pacing, CTA)
- ✅ Shorts / Reels / TikTok (15-60 วิ)
- ✅ Hook 3 วิแรก (5-10 ตัวเลือก)
- ✅ Thumbnail prompt (Midjourney/DALL-E ready)
- ✅ Title options (5 แบบ/คลิป)
- ✅ Description + Tags + Timestamps
- ✅ Content calendar 30 วัน (30 ideas)

## 💡 Use cases

- รีวิวสินค้า
- สอน / tutorial
- Vlog / lifestyle
- Storytelling / reaction
- Finance / productivity
- Gaming / entertainment

- ใช้ `/graphic-designer-pro` ต่อยอด thumbnail prompt
