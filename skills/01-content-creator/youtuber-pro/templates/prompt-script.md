# Script Template — Hook Patterns & Recipes

## Hook Patterns (3 วิแรก)

### Pattern 1: Shocking Statement
> "90% ของคนที่ถ่ายรูปด้วยกล้องฟิล์ม ไม่รู้ว่ามีปุ่มนี้อยู่"

### Pattern 2: Question Hook
> "ทำไมกล้องราคา 3,000 ถ่ายได้ดีกว่าราคา 50,000?"

### Pattern 3: Curiosity Gap
> "ผมใช้กล้องตัวนี้มา 3 ปี แล้วเพิ่งค้นพบ 1 ฟีเจอร์ที่เปลี่ยนทุกอย่าง"

### Pattern 4: Contrarian
> "ลืมทุกสิ่งที่คนสอนคุณเรื่องกล้องฟิล์ม มันผิดหมด"

### Pattern 5: Result-First
> "นี่คือรูปที่ถ่ายด้วยกล้อง 500 บาท"
> *(แสดงรูปสวยงาม)*

### Pattern 6: Time-Sensitive
> "ถ้าคุณจะซื้อกล้องฟิล์มในปี 2026 ดูคลิปนี้ก่อน"

### Pattern 7: Story Cold Open
> "ตอนที่ผมซื้อกล้องตัวนี้ ร้านบอกผมว่ามันพัง..."

## Pattern Interrupt Recipes (ทุก 60-90 วิ)

ใช้เพื่อรักษา retention — เปลี่ยนทุกๆ ~1 นาที:

1. **Camera angle change** — close-up → wide → over-shoulder
2. **B-roll cut-in** — ตัดไปภาพประกอบ 3-5 วิ
3. **Text overlay** — คำ/ประโยคสำคัญขึ้นจอ
4. **Sound effect** — whoosh, ding, page-turn
5. **Music tempo change** — เปลี่ยนเพลงตามอารมณ์
6. **Location change** — ย้ายสถานที่ถ่าย
7. **Expression change** — หยุด, สีหน้า, ท่าทาง

## Body Chapter Recipe

แต่ละ chapter ใช้โครง PROBLEM → SOLUTION → PROOF → TRANSITION:

```
[0:30] Chapter 1: ปัญหาที่ 1
    PROBLEM:   "คนส่วนใหญ่เจอปัญหานี้..."
    SOLUTION:  "ผมแก้ด้วยวิธีนี้..."
    PROOF:     "ลองดูผลลัพธ์..."
    TRANSITION: "แต่ถ้าคุณเจอปัญหาที่ 2..."
```

## CTA Templates

### CTA สุภาพ (educational channel)
> "ถ้าคลิปนี้มีประโยชน์ กด Like ช่วยให้ YouTube แนะนำให้คนอื่นเห็นด้วย แล้ว comment บอกผมหน่อยว่าคุณอยากให้ผมทำเรื่องอะไรต่อ"

### CTA สนุก (entertainment channel)
> "ถ้าดูจบแล้วยังไม่กด subscribe ถือว่า... ผมว่าคุณเป็นคนใจร้ายมาก (หยอก) แต่จริงๆ แค่อยากให้รู้ว่าผมปล่อยคลิปทุกวัน..."

### CTA แบบ curiosity (keep watching)
> "แต่ก่อนจะจบ มี 1 เรื่องที่ผมอยากให้คุณรู้ — ดูคลิปนี้ต่อเลย [แนะนำคลิปถัดไป]"

## Retention Tricks

1. **Open loop** — "เดี๋ยวจะบอกตอนท้ายว่า..." (สร้างความอยากรู้ตลอดคลิป)
2. **Callback** — อ้างถึงสิ่งที่พูดไว้ตอนต้นคลิป
3. **Promise ladder** — เพิ่ม promise เรื่อยๆ "แต่ยังไม่ใช่แค่นั้น..."
4. **Mini-cliffhanger** — จบ chapter ด้วย "...แต่ที่ช็อคจริงๆ คือ..."
5. **Visual payoff** — แสดง before/after, transformation ชัดเจน

## Shorts/Reels Hook Recipe

ต่างจาก long-form — **1 วินาทีต้องจบ**:

### Visual Hook (บังคับ)
- Text ใหญ่ขึ้นจอทันที (question / shocking number / promise)
- Movement แปลกๆ (zoom, shake, reveal)
- Color ตัดกันแรง

### Hook Examples (Shorts)
- "POV: คุณใช้ ChatGPT ทำ..."
- "ห้ามซื้อสินค้านี้ถ้า..."
- "3 เทคนิคที่ไม่มีใครบอก"
- "ผมลองอันนี้ 30 วัน ผลคือ..."

## Algorithm Optimization Notes (2026)

**YouTube:**
- CTR (thumbnail + title) — ต้อง > 4%
- AVD (average view duration) — ต้อง > 50%
- Session time — คนต้องดู channel ต่อ

**TikTok/Reels:**
- Completion rate — ต้อง > 60%
- Replay — encourage rewatch
- Save + Share > Like
- Comment bait (question ในคลิป)

**Shorts:**
- First 3 sec = everything
- Loop ลื่นๆ (วนกลับได้เนียน)
- Sound trending (แต่ไม่จำเป็นเสมอ)
