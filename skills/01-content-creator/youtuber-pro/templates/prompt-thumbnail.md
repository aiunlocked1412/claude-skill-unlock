# Thumbnail Prompt Formula

> สูตรสร้าง Midjourney/DALL-E prompt สำหรับ YouTube thumbnail ที่ CTR สูง

---

## Formula มาตรฐาน

```
<SUBJECT> <ACTION/EXPRESSION>, <BACKGROUND>, <LIGHTING>,
<TEXT OVERLAY: "คำใหญ่ 3-5 คำ">,
<STYLE>, <COLOR PALETTE>, <COMPOSITION>,
--ar 16:9 --v 6 --style raw
```

---

## 7 Elements ของ Thumbnail ที่ได้ CTR สูง

### 1. Subject (ต้องมี!)
- **ใบหน้าคน + expression แรง** (ตกใจ, ดีใจ, สงสัย)
- ถ่ายใกล้ (close-up — เห็นแค่หน้า + ไหล่)
- ตาจ้องกล้อง

### 2. Action / Object
- สิ่งของที่เกี่ยวข้องกับเนื้อหา (กล้อง, เงิน, อาหาร, ฯลฯ)
- ชี้นิ้วไป object หรือ reaction ต่อ object

### 3. Background
- เบลอ / สีพื้น / gradient
- ห้ามรก (แข่งกับ text)

### 4. Text Overlay
- **3-5 คำใหญ่** ตัดกับพื้นหลัง
- Font หนา (bold, black)
- สีตัดกับพื้นหลังแรง (yellow/red/white)
- วาง 1 ใน 3 ของภาพ (ซ้าย/ขวา/บน)

### 5. Lighting
- Key light จากด้านหน้า
- Rim light จากด้านหลัง (ให้ subject ลอยออกมา)
- ห้ามมืด — ต้อง bright, contrast สูง

### 6. Color Palette (High Contrast)
- ตัวเลือก 1: **Yellow + Red + Black** (classic MrBeast)
- ตัวเลือก 2: **Blue + Orange** (complementary)
- ตัวเลือก 3: **White + Red** (clean + urgent)

### 7. Composition (Rule of Thirds)
- Subject วาง 1 ใน 3 ของภาพ (ไม่วางกลาง)
- Text วางฝั่งตรงข้าม subject
- มี negative space สำหรับ breathing

## Don'ts ของ Thumbnail

❌ ตัวอักษรเล็กเกิน (ดูบนมือถือไม่รู้เรื่อง)
❌ Subject อยู่กลางจอ (boring composition)
❌ สีกลมกลืน (low contrast — ไม่เด่น)
❌ ไม่มีใบหน้า (expression คือ magnet)
❌ Text ยาวเกิน 5 คำ
❌ Background รก (แข่งกับ subject)

---

## Thumbnail Variants (ถ่าย 3-5 แบบ → A/B test)

สร้าง 3 variants ให้ YouTube เลือกเอง:

1. **Emotion-heavy** — expression สุดโต่ง + คำสั้น
2. **Curiosity-driven** — สิ่งของแปลก + คำถาม
3. **Result-driven** — before/after + ตัวเลข

YouTube Studio → Test & Compare feature จะเลือก version ที่ CTR สูงสุดให้

## Title + Thumbnail Pairing

Title กับ Thumbnail ต้องเสริมกัน **ไม่ซ้ำกัน**:

| Title | Thumbnail Text |
|-------|---------------|
| "10 วิธีลงทุนที่คนไม่บอก" | "สูตรลับ!" |
| "ทำไม iPhone แพงขึ้น 30%" | "$$$ เพิ่ม 30%" |
| "รีวิวกล้อง Fuji XT-5 ใช้ 1 ปี" | "ห้ามซื้อ!" |

Title ให้ความสุภาพ/ข้อมูล, Thumbnail ให้ emotion/hook
