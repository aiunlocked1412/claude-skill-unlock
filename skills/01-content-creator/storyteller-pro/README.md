# Storyteller Pro

> AI นักเล่าเรื่องที่สร้าง plot + ตัวละคร + dialogue ที่สมจริง สำหรับเรื่องสั้น/นิยาย/บทหนัง/เกม

## ⚡ ใช้ยังไง

```
/storyteller-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ เรื่องสั้น (1,000-3,000 คำ)
- ✅ นิยายตอน / chapter (3,000-8,000 คำ)
- ✅ บทหนังสั้น (5-10 นาที — screenplay format)
- ✅ Game narrative (quest / cutscene / lore)
- ✅ Character profile (ลึก — want/need/wound/arc)
- ✅ Plot outline (3-act + beat sheet)
- ✅ Dialogue sample (ฝึกเสียงตัวละคร)

## 💡 Use cases

- นักเขียนนิยาย / web novel
- Screenwriter มือใหม่
- Game designer ทำ narrative
- Content creator ทำ short video storytelling
- ครูสอนเขียนที่ต้องการ example

## 📦 ไฟล์ใน skill นี้

```
storyteller-pro/

```

## 🎓 Tips

- บอก **premise** ชัด 1-2 ประโยค → plot จะโฟกัส
- ระบุ **theme** (สิ่งที่อยากให้คนอ่านรู้สึก) → เรื่องมี soul
- ให้ **genre + tone** → convention ถูกต้อง
- ตัวละครยิ่ง specific ยิ่งดี — อาชีพ อายุ นิสัย contradiction
