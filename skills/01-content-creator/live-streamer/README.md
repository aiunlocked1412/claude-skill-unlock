# Live Streamer

> AI โปรดิวเซอร์ไลฟ์สตรีม — Scene, Overlay, Chat Engagement, Monetization, Sponsor สำหรับทุกแพลตฟอร์ม

## ⚡ ใช้ยังไง

```
/live-streamer
```

## 🎯 ทำอะไรได้บ้าง

- ✅ ออกแบบ Scene & Overlay (layout, widget, alert, transition animation)
- ✅ Chat Engagement Plan (poll, กิจกรรม, ! commands, milestone)
- ✅ Monetization Setup (donation ไทย, sub tier, Ko-fi, Patreon)
- ✅ Stream Schedule & Rundown (ตาราง + script opening/closing)
- ✅ Sponsor Integration (tier, mention script, disclosure ถูกกฎหมาย)
- ✅ Growth Strategy (title formula, clip strategy, highlight weekly)
- ✅ Full Stream Package (ทุกอย่างรวมกัน)

## 💡 Use cases

- Streamer มือใหม่อยากตั้ง stream ให้ดูมืออาชีพตั้งแต่ต้น
- Gaming channel อยากเพิ่ม viewer และสร้าง community ที่แข็งแกร่ง
- Content creator อยากเพิ่ม monetization จาก sponsor
- ร้านค้าอยากทำ FB Live / TikTok Live ขายของให้มีระบบ
- Talk show / tutorial channel อยากออกแบบ production ให้ดู professional

## 📊 ตัวอย่างผลลัพธ์

เมื่อ run `/live-streamer full stream package` จะได้:
- Scene list พร้อม layout spec
- Chat engagement calendar รายสัปดาห์
- Donation + sponsor tier table
- Stream rundown script (opening → main → ending)
- Growth KPI targets

## 🎓 Tips

- บอก **แพลตฟอร์มชัดเจน** — rule TikTok ≠ Twitch ≠ YouTube
- บอก **ประเภทคอนเทนต์** (gaming/talk/tutorial/ขายของ) → overlay design ต่างกันมาก
- บอก **ขนาด audience ปัจจุบัน** → ปรับ monetization strategy ให้เหมาะ
- ใช้คู่กับ `/content-planner` → วางตารางคอนเทนต์รายเดือน

## 🔗 Skills ที่ใช้คู่กัน

- `/content-planner` — วางแผนคอนเทนต์รายเดือน
- `/community-manager` — ดูแล Discord/FB Group ของ community
- `/ads-strategist` — ยิง ads เพื่อหา viewer ใหม่
