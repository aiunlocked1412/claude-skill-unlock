## 🎯 Transformation Promise

**ก่อนเรียน คุณ:**
- พิมพ์ prompt สั้นๆ แบบ "ช่วยสรุปหน่อย" แล้วได้ output งั้นๆ
- เสียเวลา edit สิ่งที่ AI ออกมาเกือบเท่ากับเขียนเอง
- ไม่กล้าใช้ AI กับงานสำคัญเพราะกลัว hallucinate

**หลังเรียน คุณจะ:**
- เขียน prompt ที่ออกมาตรงใจตั้งแต่ครั้งแรก ลด iteration 70%
- ลดเวลางาน routine (เขียน email, สรุป meeting, ทำ slide) ลงได้ 60-80%
- มี prompt library 50+ templates ใช้งานจริงในออฟฟิศ

## ✅ Learning Outcomes

หลังเรียนจบ ผู้เรียนจะสามารถ:

1. **Apply** 7 prompt patterns (Role, Few-shot, CoT, ReAct, Persona, Constraint, Self-critique) กับงานจริง
2. **Design** prompt template สำหรับ workflow ตัวเอง (อย่างน้อย 5 templates)
3. **Critique** output ของ AI และวิจารณ์เชิง factual + style
4. **Build** "AI Assistant ส่วนตัว" บน ChatGPT GPTs หรือ Claude Projects
5. **Implement** AI workflow ในทีม โดยจัดการเรื่อง security + IP ได้
