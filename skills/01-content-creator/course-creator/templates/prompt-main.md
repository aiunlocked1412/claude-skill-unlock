# Course Creator Templates — Curriculum, Lesson, Pricing Patterns

## Backward Design Framework

### Step 1: เขียน Outcome ที่วัดได้

ใช้ verb ที่วัดได้ — หลีกเลี่ยง "เข้าใจ", "รู้", "ตระหนัก" (วัดยาก)

| Bloom Level | Verb ที่ใช้ | ตัวอย่าง |
|-------------|-----------|---------|
| Remember | list, name, recall | "list 5 prompt patterns" |
| Understand | explain, summarize | "explain why CoT works" |
| Apply | implement, use, demonstrate | "implement RAG ใน LangChain" |
| Analyze | compare, contrast, debug | "compare GPT-4 vs Claude output" |
| Evaluate | critique, justify, assess | "critique AI-generated code" |
| Create | design, build, compose | "build a chatbot end-to-end" |

### Step 2: ออกแบบ Assessment ก่อนเนื้อหา

| Outcome Level | Assessment Type |
|---------------|-----------------|
| Remember/Understand | Quiz, flashcard, MCQ |
| Apply | Hands-on exercise, sandbox |
| Analyze | Case study, code review |
| Evaluate | Critique session, peer review |
| Create | Capstone project, portfolio |

## Curriculum Structure Patterns

### Pattern 1: Linear Progression
```
Module 1 (Foundation) → Module 2 (Core) → Module 3 (Advanced) → Capstone
```
ใช้กับ: skill ที่มี dependency ชัด (เช่น coding, ภาษา)

### Pattern 2: Hub & Spoke
```
Module 1 (Core concept)
    ├─ Module 2 (Application A)
    ├─ Module 3 (Application B)
    └─ Module 4 (Application C)
```
ใช้กับ: framework ที่ apply หลาย domain (เช่น productivity, design thinking)

### Pattern 3: Spiral Curriculum
```
Module 1 (Topic A basic) → Module 2 (Topic A intermediate)
   → Module 3 (Topic A advanced) → Module 4 (Topic A expert)
```
ใช้กับ: deep mastery 1 เรื่อง

## Lesson Plan Template

### Hook Patterns (1-2 นาที)
1. **Pain story** — "ผมเคยเสีย 3 ชั่วโมงเขียน prompt ที่ผิด..."
2. **Outcome preview** — "หลัง lesson นี้ คุณจะสร้าง prompt ที่ output ดีขึ้น 3x"
3. **Counterintuitive claim** — "ส่วนใหญ่ใช้ ChatGPT ผิด — นี่คือสาเหตุ"
4. **Stat shock** — "85% ของ AI prompt ใน production ใช้แค่ 1 pattern"

### Concept Delivery (5-10 นาที)
- เริ่มด้วย **mental model / visual**
- อธิบายด้วย **3 บรรทัดสุด** ก่อนยกตัวอย่าง
- ใช้ **analogy** ที่ผู้เรียนคุ้นเคย
- หลีกเลี่ยง jargon ที่ไม่ได้นิยามไว้

### Demo (5-10 นาที)
- แสดงจริง — screen recording หรือ live coding
- พูด **"why"** ก่อน **"what"** — ทำไมเราเลือกแบบนี้
- เผื่อ pause point ให้ผู้เรียน reflect

### Practice (3-5 นาที)
- ให้ task ที่ผู้เรียนต้องทำเอง
- เตรียม starter file / sandbox
- ใส่ **time box** ชัด ("ลอง 5 นาที")

## Bonus Stack Formula

ราคาคอร์ส 4,990 + bonus stack มูลค่ารวม 19,990:

1. **Templates / swipe files** (มูลค่า ~3,000) — reusable assets
2. **Private community access 6 เดือน** (มูลค่า ~5,000) — accountability
3. **Live Q&A 4 ครั้ง** (มูลค่า ~5,000) — direct support
4. **Bonus module: case study** (มูลค่า ~2,000) — extra depth
5. **Lifetime updates** (มูลค่า ~5,000) — long-term value

## Completion Rate Levers

### กฎ 1: Quick Win ใน 24 ชม.แรก
- Lesson 1 ต้องสั้น (10 นาที) + ได้ผลจับต้องได้
- ส่ง email "ยินดีด้วย คุณทำ X ได้แล้ว" ตอน lesson 1 จบ

### กฎ 2: Cohort > Evergreen เพื่อ completion
- Live session มี deadline → คนพยายามทันเพื่อนร่วมรุ่น
- Slack/Discord channel → peer pressure ดี

### กฎ 3: Streak / Gamification
- ใส่ progress bar ทุกหน้า
- Email reminder ถ้าหยุดเรียน 7 วัน
- Certificate / badge ตอนจบ module

### กฎ 4: Office Hours
- 1 ชม./สัปดาห์ — ผู้เรียนถามได้สด
- Cost ต่ำ, impact สูง

### กฎ 5: Capstone with Public Demo
- จบคอร์สด้วย project ที่ผู้เรียน "โชว์ได้"
- Showcase ใน community → social proof สำหรับรอบหน้า

## Sales Page Structure

```
1. Hero (5 วินาทีแรก)
   - Headline: outcome + audience
   - Subhead: how + timeline
   - CTA button + price

2. Pain agitation (1 scroll)
   - "ถ้าคุณ X, Y, Z..."

3. Promise transformation
   - Before / After comparison

4. Curriculum overview (collapsible)

5. Bonus stack (anchor pricing)

6. Testimonials (3-5)
   - Result-focused, ไม่ใช่ "ดีมากค่ะ"

7. Instructor bio
   - Why qualified to teach this

8. FAQ (objection handling)
   - "ฉันยังมือใหม่ทำได้ไหม?"
   - "ถ้าไม่มีเวลาเรียน?"
   - "Refund policy?"

9. Final CTA + scarcity
   - Cohort: "เปิดรับ 30 ที่นั่ง"
   - Evergreen: "ราคานี้ถึง 31 ธ.ค."
```
