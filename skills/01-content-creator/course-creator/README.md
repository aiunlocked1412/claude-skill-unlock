# Course Creator

> AI Course Designer ที่ออกแบบคอร์สให้คนเรียนจบ ไม่ใช่แค่ขายได้

## ⚡ ใช้ยังไง

```
/course-creator
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Curriculum รวม (โครงคอร์สเต็ม + learning outcomes)
- ✅ Lesson plan (script + slide outline + assignment)
- ✅ Assessment / Quiz / Final project (วัดผลตาม Bloom's Taxonomy)
- ✅ Sales page (landing page + bonus stack + pricing)
- ✅ Launch plan (Cohort vs Evergreen + ตารางเปิดขาย)
- ✅ Completion rate optimization (ทำให้คนเรียนจบจริง)

## 💡 Use cases

- Course creator / online educator
- Coach / consultant ที่อยาก productize ความรู้
- Corporate trainer / L&D
- มหาวิทยาลัย / สถาบันการศึกษาออนไลน์
- Influencer ที่อยากแปลง audience เป็น student

## 📦 ไฟล์ใน skill นี้

course-creator/

## 🎓 Tips

- ระบุ **transformation** ชัด → ออกแบบเนื้อหาง่ายขึ้น
- เลือก **Cohort vs Evergreen** ตั้งแต่ต้น → กลยุทธ์ขายต่างกันมาก
- **Backward design** เสมอ — เริ่มจาก outcome ก่อนเขียนเนื้อหา
- ใช้คู่กับ `/copywriter-pro` เพื่อปรับ sales page ให้ convert
- ใช้คู่กับ `/youtuber-pro` เพื่ออัด promo video

## 🔗 Skills ที่ใช้คู่กัน

- `/copywriter-pro` — เขียน sales page + email sequence
- `/youtuber-pro` — promo video + course trailer
- `/blogger-pro` — บทความ SEO ดึง lead
- `/newsletter-writer` — email sequence onboarding ผู้เรียน
