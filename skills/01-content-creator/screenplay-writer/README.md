# Screenplay Writer

> AI นักเขียนบทภาพยนตร์/ซีรี่ส์/โฆษณา — 3-Act, Beat Sheet, Dialogue, Scene Heading, Character Arc ฟอร์แมตบทไทยมาตรฐาน

## ⚡ ใช้ยังไง

```
/screenplay-writer
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Story Concept & Logline (แกนเรื่อง + ประโยคขายที่ดึงดูด)
- ✅ Beat Sheet (Save the Cat 15 beats / 3-Act structure)
- ✅ Character Bible (want vs need, fatal flaw, arc, voice)
- ✅ Scene Writing (ฟอร์แมตบทไทยมาตรฐาน scene heading + action + dialogue)
- ✅ Episode Outline (ซีรี่ส์ — outline รายตอน)
- ✅ Screenplay โฆษณา TVC (15/30/60 วิ + short film)
- ✅ Full Script Package (concept + beats + ฉากนำร่อง 10 หน้า)

## 💡 Use cases

- นักเขียนบทมือใหม่ต้องการ structure ที่แข็งแกร่งก่อนเริ่มเขียน
- Production house ต้องการ pitch document สำหรับ investor / broadcaster
- แบรนด์ต้องการ TVC script ที่มี story arc ไม่ใช่แค่ showcase สินค้า
- Creator อยากทำ web series แต่ไม่รู้จะ structure ยังไง
- นักเขียนนิยายอยาก adapt งานเป็น screenplay

## 📊 ตัวอย่างผลลัพธ์

เมื่อ run `/screenplay-writer full script package` จะได้:
- Logline + premise statement
- Beat sheet 15 beats พร้อม page target
- Character bible ตัวละครหลัก 3-4 คน
- ฉากนำร่อง 10 หน้า ฟอร์แมตมาตรฐาน

## 🎓 Tips

- บอก **inspired by** (เทียบเรื่องที่ชอบ) — ช่วย calibrate tone ได้ชัดมาก
- บอก **platform target** (โรงหนัง/Netflix/YouTube/TV ไทย) → ส่งผล runtime และ content
- แนบ **ไอเดียดิบ** มาเลย แม้ยังไม่ครบ → จะช่วยปั้นให้เป็น concept ได้
- ใช้คู่กับ `/voice-over-artist` → ได้ script พร้อม VO direction ในคราวเดียว
