# Ghostwriter Pro

> AI Ghostwriter ที่เขียน "เหมือนเจ้าของ voice" — ไม่ใช่สไตล์ AI

## ⚡ ใช้ยังไง

```
/ghostwriter-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Voice analysis (ถอดสไตล์ → voice profile 7 มิติ)
- ✅ Byline writing (LinkedIn post / blog / op-ed / speech)
- ✅ Long-form ghosting (book chapter, e-book, white paper)
- ✅ Ghost agreement (สัญญา + scope + IP + payment)
- ✅ Pricing & billing (rate card + project quote)
- ✅ Ethical disclosure (เมื่อไหร่ disclose / ไม่ต้อง)

## 💡 Use cases

- Ghostwriter ที่รับงาน CEO/founder/exec
- Agency ที่ทำ thought leadership ให้ลูกค้า
- PR/comms team ที่เขียนแทน leadership
- Co-author ของ book / autobiography
- LinkedIn ghostwriter (niche กำลังเติบโต)

## 🎓 Tips

- ขอ **5-10 writing samples** ก่อนเริ่ม — voice ที่ดีต้องมี data
- ทำ **voice profile** ครั้งเดียว → reuse ได้ทุก piece
- จำกัด **2 revision rounds** ในสัญญา — กัน scope creep
- ใช้คู่กับ `/copywriter-pro` เพื่อปรับ headline ให้ punchy
- ใช้คู่กับ `/blogger-pro` สำหรับ long-form ghosting
