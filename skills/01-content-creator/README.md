# หมวด 1: Content Creator (5 skills)

Skills สำหรับนักสร้างเนื้อหา ครีเอเตอร์ นักเขียน

| Skill | คำสั่ง | ใช้เมื่อไหร่ |
|-------|--------|-------------|
| YouTuber / Reels Creator | `/youtuber-pro` | ทำคลิป YouTube, Shorts, Reels, TikTok |
| Blogger / Article Writer | `/blogger-pro` | เขียนบทความ blog, SEO writing |
| Translator / Localizer | `/translator-pro` | แปลภาษา, subtitle, localization |
| Podcaster | `/podcaster-pro` | สคริปต์ podcast, show notes |
| Storyteller / Screenwriter | `/storyteller-pro` | นิยาย, บทหนัง, plot |

## รูปแบบ output ทั่วไป

- Markdown file (`.md`) พร้อม frontmatter
- แยก section: Hook / Body / CTA
- Prompt สำหรับ AI image (Midjourney/DALL-E) ที่เกี่ยวข้อง
- Title/Headline ตัวเลือก 5-10 แบบ

## Workflow ตัวอย่าง

```
1. /youtuber-pro สคริปต์รีวิวกล้องฟิล์ม 8 นาที
   ↓
2. /copywriter-pro เขียน title + description
   ↓
3. /graphic-designer-pro thumbnail prompt
   ↓
4. /social-media-admin โพสโปรโมทคลิป
```
