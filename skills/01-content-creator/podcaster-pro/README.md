# Podcaster Pro

> AI Script Writer สำหรับ Podcaster — สคริปต์ฟังไม่เบื่อ + show notes + teaser social

## ⚡ ใช้ยังไง

```
/podcaster-pro
```

## 🎯 ทำอะไรได้บ้าง

- ✅ สคริปต์ solo podcast (20-45 นาที)
- ✅ สคริปต์สัมภาษณ์ (30-60 นาที — พร้อมคำถาม)
- ✅ Narrative / storytelling podcast
- ✅ Show notes + timestamps ครบ
- ✅ Key quotes (สำหรับ social quote card)
- ✅ Teaser clips 30-60 วิ (IG Reels / TikTok)
- ✅ Season plan 10 ตอน

## 💡 Use cases

- Podcaster มือใหม่ที่ไม่รู้จะเริ่มยังไง
- Podcaster มืออาชีพที่อยาก scale output
- สัมภาษณ์แขก — เตรียมคำถาม + flow
- ทำ promo content สำหรับ social
- Content creator ที่อยากขยายไปเสียง

## 🎓 Tips

- ระบุ **listening context** (ขับรถ / เดิน / ออกกำลัง) → pacing แม่น
- บอก **host voice** ชัด → สคริปต์ไม่แข็ง
- ใส่ **personal story** ในสคริปต์เสมอ — podcast ที่ดีต้องมี human element
- Record แล้ว? ใช้คู่กับ `/translator-pro` ทำ subtitle EN
