# Book Author

> AI ผู้ช่วยเขียนหนังสือเต็มเล่ม — Chapter Outline, Voice Guide, Manuscript Plan, Self-Publish Roadmap สำหรับตลาดหนังสือไทย

## ⚡ ใช้ยังไง

```
/book-author
```

## 🎯 ทำอะไรได้บ้าง

- ✅ Book Concept & Positioning (แกนหนังสือ + target reader + USP)
- ✅ Chapter Outline (สารบัญ + สรุปแต่ละบท ทั้ง nonfiction และ fiction)
- ✅ Chapter Draft (เขียนบทตัวอย่าง + hook เปิดบท)
- ✅ Voice & Style Guide (tone, ระดับภาษา, ตัวอย่างย่อหน้า)
- ✅ Manuscript Plan (timeline, word count target, milestone)
- ✅ Self-Publish Roadmap (edit → design → ISBN → Meb/Ookbee/KDP)
- ✅ Full Book Package (ทุกอย่างรวมกัน)

## 💡 Use cases

- นักเขียนมือใหม่อยากเขียนหนังสือเล่มแรกแต่ไม่รู้จะเริ่มยังไง
- ผู้เชี่ยวชาญ/coach อยากเขียน thought leadership book เพิ่ม authority
- นักธุรกิจอยาก self-publish business book เพื่อสร้าง brand
- นักเขียนนิยายต้องการ structure ที่แน่นก่อนลงมือเขียน
- Creator/influencer อยากต่อยอดคอนเทนต์เป็นหนังสือ

## 📦 ไฟล์ใน skill นี้

```
book-author/
├── SKILL.md       # ไฟล์หลัก (Claude อ่าน)
└── README.md      # คุณกำลังอ่านอยู่
```

## 📊 ตัวอย่างผลลัพธ์

เมื่อ run `/book-author full book package` จะได้:
- Book concept + positioning statement
- Chapter outline พร้อม word count target ต่อบท
- Voice & style guide ตาม genre
- Manuscript milestone plan
- Draft บทที่ 1 (1,500-2,000 คำ)

## 🎓 Tips

- บอก **ประสบการณ์ตรง** ที่มี — ทำให้ positioning น่าเชื่อถือขึ้นมาก
- บอก **comps** (หนังสือที่ชอบ/ใกล้เคียง) → calibrate voice ได้ชัด
- **เขียน draft 1 ให้จบก่อน** อย่าแก้ระหว่างเขียน — ใช้ skill นี้ช่วย outline ก่อน
- ใช้คู่กับ `/screenplay-writer` → adapt หนังสือเป็นบทภาพยนตร์/ซีรี่ส์
