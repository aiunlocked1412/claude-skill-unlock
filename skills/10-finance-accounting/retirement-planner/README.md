# Retirement Planner

> AI วางแผนเกษียณ — คำนวณเงินเป้าหมาย, RMF/SSF/ประกันบำนาญ, Withdrawal Strategy, FIRE

## ⚡ ใช้ยังไง

```
/retirement-planner
```

## 🎯 ทำอะไรได้บ้าง

- ✅ คำนวณ Retirement Number จริง (รวมเงินเฟ้อ + ค่ารักษาพยาบาล)
- ✅ สร้างแผนออมเงินรายเดือน + Projection ถึงวันเกษียณ
- ✅ เปรียบเทียบ RMF / SSF / ประกันบำนาญ / PVD
- ✅ Asset Allocation ตามอายุและ Risk Tolerance
- ✅ Withdrawal Strategy หลังเกษียณ (4% Rule / Bucket / Floor-Upside)
- ✅ วางแผน FIRE (Financial Independence, Retire Early)

## 💡 Use cases

- วัยเริ่มทำงานอยากรู้ต้องออมเดือนละเท่าไหร่จึงจะเกษียณสบาย
- คนอายุ 40+ รู้สึกว่าออมไม่พอ อยากรู้ยังแก้ได้ไหม
- พนักงานอยากเปรียบเทียบ RMF กับ SSF ว่าลงทุนอะไรคุ้มกว่า
- คนเกษียณแล้วอยากรู้ว่าจะถอนเงินยังไงไม่ให้หมดก่อนตาย
- คนฝันอยากเกษียณก่อน 50 (FIRE) — เป็นไปได้แค่ไหน

## 📊 ตัวอย่างผลลัพธ์

อายุ 35 เกษียณ 60 ค่าใช้จ่าย 50,000/เดือน: เงินเป้าหมาย ~25 ล้านบาท → ต้องออม ~25,000 บาท/เดือน (ผลตอบแทน 7%/ปี) แสดง Breakdown ครบถ้วน

## 🎓 Tips

- บอก **อายุจริง + อายุที่อยากเกษียณ** เป็นตัวแปรสำคัญที่สุด
- บอก **เงินออมปัจจุบัน** ด้วย — อาจต้องออมน้อยกว่าที่คิด
- อย่าลืม **ค่ารักษาพยาบาล** — มักถูกมองข้ามแต่สำคัญมาก
