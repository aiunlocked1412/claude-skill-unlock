# Mortgage Broker

> AI โบรกเกอร์สินเชื่อบ้าน — เปรียบเทียบธนาคาร, ต่อรองดอกเบี้ย, ยื่นกู้, รีไฟแนนซ์

## ⚡ ใช้ยังไง

```
/mortgage-broker
```

## 🎯 ทำอะไรได้บ้าง

- ✅ เปรียบเทียบสินเชื่อบ้านหลายธนาคาร (อัตราจริง ไม่ใช่โฆษณา)
- ✅ คำนวณงวดผ่อน + ต้นทุนรวมตลอดสัญญา
- ✅ วิเคราะห์การรีไฟแนนซ์ — คุ้มหรือไม่?
- ✅ เปรียบเทียบ Fixed vs Floating Rate
- ✅ ค่าใช้จ่ายวันโอน + Checklist เอกสาร
- ✅ แนะนำกลยุทธ์เจรจาดอกเบี้ยกับธนาคาร

## 💡 Use cases

- ซื้อบ้านครั้งแรก ไม่รู้ว่าธนาคารไหนดีที่สุด
- อยากรีไฟแนนซ์แต่ไม่รู้ว่าคุ้มไหม
- อยากเปรียบเทียบงวดผ่อน 20 vs 30 ปี
- มีบ้าน 2 หลัง อยากกู้ซื้อหลังที่ 3 (LTV เปลี่ยน)
- Freelancer / เจ้าของกิจการอยากซื้อบ้าน

## 📊 ตัวอย่างผลลัพธ์

กู้ 3 ล้าน 30 ปี ดอกเบี้ย 3%: งวด 12,648 บาท/เดือน | เทียบ 25 ปี ดอกเบี้ย 3.5%: งวด 14,963 บาท/เดือน แต่ต้นทุนรวมต่างกัน 3 แสนบาท — เห็นภาพชัด ตัดสินใจได้ดีขึ้น

## 🎓 Tips

- บอก **วงเงินที่อยากกู้** + **ระยะเวลาที่อยากผ่อน** เพื่อดูงวดจริง
- ยื่น **Pre-approval หลายธนาคารพร้อมกัน** เพื่อเพิ่มอำนาจต่อรอง
- อย่าดูแค่ดอกเบี้ยปีแรก — ดู **ดอกเบี้ยหลังปีที่ 3** ด้วย
