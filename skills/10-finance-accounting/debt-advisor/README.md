# Debt Advisor

> AI ที่ปรึกษาจัดการหนี้ — แผนชำระหนี้, รวมหนี้, เจรจาเจ้าหนี้, หนี้เสีย NPL

## ⚡ ใช้ยังไง

```
/debt-advisor
```

## 🎯 ทำอะไรได้บ้าง

- ✅ วิเคราะห์ภาพรวมหนี้ทั้งหมด + จัดลำดับความสำคัญ
- ✅ สร้างแผนชำระหนี้ (Avalanche / Snowball) + timeline ปลอดหนี้
- ✅ แนะนำการรวมหนี้ (Debt Consolidation)
- ✅ Script เจรจาเจ้าหนี้จริงๆ
- ✅ ทางออกสำหรับหนี้เสีย NPL + คลินิกแก้หนี้
- ✅ สิทธิ์ผู้บริโภค พ.ร.บ. ทวงถามหนี้ 2558

## 💡 Use cases

- คนที่จ่ายแต่ขั้นต่ำบัตรเครดิตทุกเดือน แต่หนี้ไม่ลด
- ผู้ที่มีหนี้หลายก้อนและไม่รู้จะชำระอะไรก่อน
- คนถูกติดตามหนี้รุนแรง ไม่รู้สิทธิ์ตัวเอง
- ผู้ต้องการรวมหนี้ให้ดอกเบี้ยถูกลง
- คนที่หนี้กลายเป็น NPL และกลัวถูกฟ้อง

## 📊 ตัวอย่างผลลัพธ์

หนี้บัตร 200,000 บาท ดอกเบี้ย 20%/ปี จ่ายขั้นต่ำ 5%/เดือน → ใช้เวลา 7 ปีกว่าจะหมด จ่ายดอกเบี้ยรวม 180,000 บาท แต่ถ้าใช้ Avalanche Method จ่าย 8,000/เดือน → หมดใน 2.5 ปี ประหยัดดอกเบี้ยได้ 120,000 บาท

## 🎓 Tips

- บอก **ดอกเบี้ยทุกบัตร/สินเชื่อ** ให้ครบ — ตัวเลขนี้สำคัญมากในการวางแผน
- บอก **เงินส่วนเกิน** หลังจ่ายค่าใช้จ่ายทุกเดือนจริงๆ
- ไม่ต้องอาย — ยิ่งบอกรายละเอียดมาก ยิ่งได้แผนที่ตรงและเป็นไปได้จริง
