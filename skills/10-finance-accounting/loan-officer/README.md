# Loan Officer

> AI เจ้าหน้าที่สินเชื่อธนาคาร — วิเคราะห์ Credit, สินเชื่อบ้าน/รถ/SME/Personal, เตรียมเอกสาร

## ⚡ ใช้ยังไง

```
/loan-officer
```

## 🎯 ทำอะไรได้บ้าง

- ✅ วิเคราะห์ Credit ด้วย 5C Framework
- ✅ คำนวณ DSR + วงเงินสินเชื่อที่ขอได้
- ✅ แนะนำสินเชื่อบ้าน (LTV ตาม ธปท.)
- ✅ สินเชื่อ SME / ธุรกิจ พร้อม approval workflow
- ✅ เตรียมเอกสารยื่นกู้ครบตามประเภท
- ✅ Tips เพิ่มโอกาสอนุมัติจาก Loan Officer มืออาชีพ

## 💡 Use cases

- คนอยากซื้อบ้านครั้งแรก ไม่รู้จะเตรียมตัวยังไง
- Freelancer หรือเจ้าของกิจการอยากกู้เงิน
- คนที่มีประวัติ NCB ไม่ดีอยากปรับปรุง
- SME ต้องการวงเงิน OD หรือสินเชื่อขยายธุรกิจ
- คนที่อยากรีไฟแนนซ์บ้านให้ดอกเบี้ยถูกลง

## 📊 ตัวอย่างผลลัพธ์

รายได้ 60,000 บาท/เดือน ผ่อนรถ 8,000 → DSR เหลือใช้ได้ 16,000 บาท/เดือน → วงเงินบ้าน (30 ปี 6%) ≈ 2.67 ล้านบาท → แนะนำซื้อบ้าน 2.5-3 ล้าน พร้อมดาวน์ 20%

## 🎓 Tips

- บอก **รายได้สุทธิจริง** (หลังหักประกันสังคม/ภาษี) เพื่อคำนวณ DSR แม่น
- บอก **หนี้ทั้งหมด** ที่มีอยู่ทุกที่เพื่อประเมินได้ครบ
- ยิ่งบอกรายละเอียด ยิ่งได้คำแนะนำตรงกับสถานการณ์จริง
